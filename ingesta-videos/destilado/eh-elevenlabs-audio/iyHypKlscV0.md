# How to Use AI Sound Effects – ElevenLabs SFX v2 Walkthrough
**Fuente:** eh-elevenlabs-audio | https://youtu.be/iyHypKlscV0

## Qué enseña
Presenta las mejoras de Sound Effects V2 de Eleven Labs: nueva interfaz, modelo mejorado, duración extendida hasta 30 segundos, looping sin cortes, descarga en 48 kHz WAV (estándar profesional), y generación simultánea de múltiples efectos sin esperar.

## Lecciones accionables
- Usa el **prompt box** fijo en la parte inferior para generar efectos desde cualquier pestaña (Explorar, Historial, Favoritos).
- Genera hasta **4 variantes** por prompt; el sistema permite lanzar múltiples generaciones en paralelo sin esperar a que termine la anterior.
- Para looping: activa la opción de **looping** para crear clips de 10 segundos que se repiten sin cortes perceptibles.
- Duración máxima: **30 segundos** (8 segundos más que V1).
- Ajusta la **prompt influence slider** para controlar qué tanto se adhiere el resultado al prompt.
- Descarga en **WAV a 48 kHz** (estándar para cine, TV y videojuegos) o **MP3 a 44.1 kHz**.
- Guarda efectos favoritos con el **ícono de estrella** y crea colecciones personalizadas.
- Reutiliza prompts de generaciones anteriores con el botón **"Use prompt"** (ícono al lado del favorito).
- Prompt de ejemplo usado en el video: `car whoosh` (sin comillas).

## Reglas para agentes
- Usa **48 kHz WAV** cuando el destino sea producción profesional (cine, TV, videojuegos).
- Usa **44.1 kHz MP3** cuando el destino sea web o distribución general.
- Activa **looping** solo cuando el efecto deba repetirse sin pausas (fondos ambientales, mecánicas de juego).
- Nunca generes un solo efecto a la vez si necesitas múltiples variantes; lanza todas en paralelo.
- Nunca copies y pegues prompts manualmente; usa el botón **"Use prompt"** para reutilizar.

## Errores comunes que evita o menciona
- No esperar a que termine una generación para iniciar otra: el sistema soporta múltiples generaciones simultáneas.
- Usar duraciones menores a 30 segundos cuando se necesita un clip largo: ahora el límite es 30s.
- Descargar en MP3 44.1 kHz para proyectos profesionales: el estándar es WAV 48 kHz.
- No usar la opción de looping para fondos que requieren repetición continua sin cortes.
