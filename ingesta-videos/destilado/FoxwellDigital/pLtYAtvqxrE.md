# Creative Strategy Call (April) 
**Fuente:** FoxwellDigital | https://youtu.be/pLtYAtvqxrE
## Qué enseña (2-3 líneas)
Cómo diseñar una estrategia de testing creativo escalable según el presupuesto del cliente, evitando el desperdicio publicitario. Explica que la clave no es lanzar cientos de activos, sino testear con hipótesis claras y escalar desde lo pequeño hacia lo grande, adaptando la cantidad de creatividades al nivel de gasto mensual.
## Lecciones accionables
- Define una hipótesis clara antes de cada test creativo: no crees activos solo por crear, sino para validar un ángulo o gancho específico.
- Para marcas con presupuesto menor a 25k USD/mes, limita el número de variaciones creativas simultáneas para evitar pérdidas tempranas.
- Escala el volumen de testing solo después de haber identificado qué hooks y ángulos funcionan a pequeña escala.
- Asegura que cada activo nuevo tenga suficiente presupuesto detrás para generar señal estadística significativa antes de decidir si es ganador o perdedor.
- Implementa sistemas de seguimiento de hipótesis y datos a través de todos los canales para no perder trazabilidad.
- No copies el "playbook" de marcas grandes sin haber pasado por la fase de ramp-up: el camino es empezar pequeño, aprender y luego escalar.
## Reglas para agentes
- Usa un número de tests proporcional al presupuesto mensual: a menor gasto, menos activos simultáneos.
- Nunca lances una creatividad sin una hipótesis documentada sobre hook y ángulo.
- Nunca asumas que lo que funciona para una marca funcionará para otra: adapta estrategia, hook y ángulo al cliente específico.
- Siempre asigna presupuesto suficiente a cada test para obtener señal válida antes de escalar.
- Nunca "blitzees" una cuenta con toneladas de creatividades nuevas al mismo tiempo: prioriza eliminar perdedores a pequeña escala.
## Errores comunes que evita o menciona
- Crear cientos de variaciones creativas sin hipótesis previa, lo que genera desperdicio de presupuesto.
- Asumir que un mayor presupuesto en testing equivale a mejores resultados; en realidad, aumenta el riesgo de pérdidas.
- Copiar estrategias de marcas grandes sin haber construido primero un sistema de aprendizaje a pequeña escala.
- No tener sistemas de seguimiento de hipótesis y datos entre canales, lo que impide aprender de los tests.
- Ignorar que cada cliente está en una etapa diferente y necesita una estrategia creativa a medida.
