# Small Spenders Cohort
**Fuente:** FoxwellDigital | https://youtu.be/QPP_JBysoHw

## Qué enseña (2-3 líneas)
Estrategia para lanzar anuncios en una cuenta publicitaria nueva sin datos previos de píxel, usando un producto innovador (filtro purificador para ventilador de techo). Explica cómo estructurar campañas iniciales con presupuesto limitado, combinando creatividades UGC y gráficas, y cómo manejar expectativas del cliente cuando no hay historial de conversiones.

## Lecciones accionables
- **Estructura de campaña inicial para cuenta nueva:** Lanzar 2 campañas simultáneas: una de alcance amplio (Broad) y otra con intereses específicos (ej. "eco-friendly", "air quality", "environment", "clean", marcas como "Dyson"). Sin retargeting al inicio.
- **Creatividades mínimas para el lanzamiento:** Usar al menos 4 piezas de contenido:
  - 2 videos UGC propios (uno con enfoque científico/datos, otro orgánico tipo "mira lo que compré").
  - 2 gráficos tipo "handbook" mostrando funcionamiento y datos (ej. antes/después de calidad del aire con monitor comprado específicamente).
- **Optimización de conversión desde el inicio:** Configurar campañas con objetivo de "Purchase" (compra) desde el día uno, incluso sin datos de píxel. Explicar al cliente que el aprendizaje ocurre a nivel de cuenta publicitaria y conjunto de anuncios, no solo del píxel.
- **Plan de contingencia si no hay compras:** Si después de un tiempo no se obtienen conversiones, agregar capas de optimización hacia objetivos más arriba del embudo (outbound clicks, landing page views, view content, add to cart) para alimentar datos a la cuenta.
- **Presupuesto y expectativas:** Cliente dispuesto a gastar hasta $15K/mes. Establecer expectativa clara de que pueden no obtener compras en los primeros 90 días y que se necesita paciencia para construir datos desde cero.
- **Colaboración con creadores:** Enviar producto a un creador externo para contenido UGC adicional, pero no depender de su cronograma para el lanzamiento inicial.

## Reglas para agentes
- Usa "Purchase" como objetivo de optimización en campañas nuevas, incluso sin datos de píxel.
- Nunca actives retargeting en las primeras campañas de una cuenta nueva.
- Siempre comunica al cliente que una cuenta nueva sin datos puede tardar hasta 90 días en generar compras.
- Si no hay conversiones después de un período, agrega objetivos de embudo superior (outbound clicks, landing page views, view content, add to cart) para acumular datos.
- Usa al menos 4 piezas de creative por lanzamiento: 2 videos UGC + 2 gráficos informativos.
- No aceptes gestionar orgánico si el cliente lo pide; deriva a otro servicio si no lo ofreces.

## Errores comunes que evita o menciona
- **Confiar en datos de píxel previos:** Tener el píxel instalado en el sitio por meses sin anuncios no aporta datos útiles para publicidad; el aprendizaje ocurre con las campañas activas.
- **Empezar con demasiados intereses o audiencias:** No usar lookalikes ni segmentación por persona al inicio; comenzar con una campaña Broad y otra con intereses amplios (eco-friendly, marcas de purificadores).
- **Depender de un solo tipo de creative:** No lanzar solo videos o solo imágenes; combinar formatos (UGC + gráficos) para maximizar aprendizaje.
- **No establecer expectativas de tiempo:** No prometer resultados inmediatos; explicar que una cuenta nueva necesita inversión y paciencia para que el algoritmo aprenda.
