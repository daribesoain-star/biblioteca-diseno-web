# ventas_listas_precios
**Fuente:** kame | https://help.kameone.cl/es/articles/12113886-manual-listas-de-precios-en-ventas
## Qué enseña (2-3 líneas)
Guía paso a paso para crear, asignar productos y utilizar listas de precios en el módulo de ventas de KAME ERP. Explica cómo definir precios diferenciados por tipo de cliente (mayorista, e-commerce, etc.) y aplicarlos automáticamente en cotizaciones y ventas.

## Lecciones accionables
- **Ruta de acceso:** Ir a **Maestros → Listas de Precios Venta**.
- **Crear lista:** En la esquina superior izquierda, seleccionar **"nueva lista de precio de ventas"**. Asignar un nombre único (ej. "Clientes Mayoristas", "Precios E-commerce").
- **Asignar productos:** Hacer clic en la cantidad de artículos que contiene la lista.
- **Métodos de carga:**
  - **Opción 1 (Individual):** Seleccionar producto y asignar nuevo precio manualmente.
  - **Opción 2 (Masiva):** Usar el importador para cargar cientos de productos a la vez.
- **Uso en ventas:** Al crear una cotización o venta, seleccionar la lista de precios correspondiente. El sistema aplica los precios automáticamente.

## Reglas para agentes
- Usa **Maestros → Listas de Precios Venta** para acceder a la funcionalidad.
- Asigna un **nombre único** a cada lista de precios.
- Selecciona la lista de precios **al momento de crear una cotización o venta**, no después.
- Para cargas masivas, usa siempre el **importador** en lugar de ingreso manual.

## Errores comunes que evita o menciona
- **Errores manuales en precios:** El sistema aplica los precios automáticamente al seleccionar la lista, eliminando descuentos incorrectos.
- **No tener listas diferenciadas:** Se recomienda crear listas separadas para distintos tipos de cliente (mayorista, e-commerce, etc.).
