# elec_compras
**Fuente:** kame | https://help.kameone.cl/es/articles/15668793-manual-electronico-de-compras

## Qué enseña
El módulo Electrónico de Compras centraliza la gestión de documentos tributarios electrónicos recibidos por la empresa. Permite revisar documentos que llegan a la casilla de intercambio, administrar su recepción mediante acuses de recibo, y sincronizar automáticamente el Registro de Compras con el Servicio de Impuestos Internos (SII).

## Lecciones accionables
- Accede a **Docs. Elect. Recibidos** para visualizar todos los documentos que ingresan a la casilla de intercambio (formato `[email protected]`).
- Desde Docs. Elect. Recibidos puedes:
  - **Visualizar el documento**: acceder directamente a la vista en formato PDF.
  - **Dar acuse de recibo**: gestionar la aceptación del documento comercial de forma rápida.
- Usa el módulo **Registro de Compras en el SII** como acceso directo a la integración de compras con el SII. Te redirigirá a la plataforma de conexión para sincronizar e importar automáticamente tus documentos de compra.
- La sincronización automática con el SII actualiza el registro de compras de forma simple y centralizada, optimizando la administración de documentos tributarios.

## Reglas para agentes
- Usa **Docs. Elect. Recibidos** cuando el usuario necesite revisar, visualizar en PDF o aceptar documentos electrónicos entrantes.
- Usa **Registro de Compras en el SII** cuando el usuario requiera sincronizar o importar automáticamente documentos de compra desde el SII.
- Nunca asumas que los documentos llegan automáticamente a todos los proveedores; consulta el artículo relacionado "¿Por qué no llegan todas las facturas electrónicas de mis proveedores?" si hay problemas de recepción.
- Nunca ignores el acuse de recibo como paso obligatorio para la aceptación formal del documento comercial.

## Errores comunes que evita o menciona
- **Documentos que no llegan**: si no todas las facturas electrónicas de proveedores aparecen, revisa el artículo relacionado "¿Por qué no llegan todas las facturas electrónicas de mis proveedores?" para diagnosticar causas.
- **No gestionar acuses de recibo**: omitir el acuse puede retrasar la aceptación formal y el registro contable de los documentos recibidos.
- **No sincronizar con el SII**: no usar el módulo de Registro de Compras en el SII puede resultar en un registro de compras desactualizado o incompleto.
