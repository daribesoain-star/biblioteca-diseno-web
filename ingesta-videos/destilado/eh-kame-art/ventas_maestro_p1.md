# ventas_maestro
**Fuente:** kame | https://help.kameone.cl/es/articles/9767333-manual-maestro-de-ventas

## Qué enseña
El Maestro de Ventas permite configurar y administrar los parámetros clave del proceso de ventas: vendedores, listas de precios de venta y sucursales. Explica paso a paso cómo crear y gestionar estos elementos para mantener actualizada la información del ciclo de ventas.

## Lecciones accionables
- **Crear un vendedor:** Ir a Maestros → Vendedores → Clic en botón verde → Ingresar nombre del vendedor y comisión → Registrar. El vendedor queda disponible en el listado y se puede seleccionar al registrar un documento de venta.
- **Editar, desactivar o eliminar vendedor:** Usar el menú de opciones de cada vendedor en el listado.
- **Crear lista de precios de venta:** Ir a Maestros → Lista de Precio de Ventas → Clic en botón verde "Nueva Lista de Precio" → Asignar nombre → Registrar. Aparecerá en el listado con un "0" que indica la cantidad de artículos asignados.
- **Asignar artículos a una lista de precios:** Hacer clic en el "0" de la lista deseada → Se abre ventana de asignación. Dos métodos:
  - **Individual:** Botón verde → Seleccionar artículo y asignar precio.
  - **Masivo:** Botón celeste → Importar precios de forma masiva.
- **Crear sucursal:** Ir a Maestros → Sucursal → Clic en botón verde → Completar datos requeridos en ventana emergente → Registrar. La sucursal estará disponible para seleccionar en la venta.

## Reglas para agentes
- Usa el botón verde para crear nuevos registros (vendedores, listas de precios, sucursales).
- Usa el botón celeste solo para importar precios de forma masiva en listas de precios.
- Al crear un vendedor, siempre debes ingresar nombre y comisión antes de registrar.
- Para asignar artículos a una lista de precios, haz clic en el "0" del listado, no en otro elemento.
- Selecciona el vendedor, lista de precios y sucursal correspondientes al momento de registrar un documento de venta.

## Errores comunes que evita o menciona
- No registrar un vendedor sin haber ingresado nombre y comisión.
- No olvidar asignar artículos a la lista de precios después de crearla (el "0" indica que está vacía).
- No confundir el botón de creación individual (verde) con el de importación masiva (celeste) al asignar precios.
- No omitir la configuración de sucursales, ya que son necesarias para seleccionar la ubicación correcta en la venta.
