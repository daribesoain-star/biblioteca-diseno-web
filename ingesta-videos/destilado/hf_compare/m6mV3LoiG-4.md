## Que ensena sobre Higgsfield (2-3 lineas)
Higgsfield es una plataforma unificada que centraliza Sora 2 Pro, Veo 3.1, Grok Imagine, Kling 3.0 y herramientas propias (Cinema Studio, Seedance) en una sola cuenta, eliminando la necesidad de múltiples suscripciones. Permite text-to-video, image-to-video, edición de fotos/video (Relight), presets para conversión rápida de foto a video, y reference-based motion para mapear movimientos reales a personajes AI. Incluye potenciadores Max/Max Pro para eliminar flickering y upscaling en Sora.

## Tips y flujos accionables

- **Acceso a modelos:** En el dashboard, selecciona el modelo (Sora 2 Pro, Veo 3.1, Grok, Kling 3.0) desde un menú desplegable. Escribe tu prompt y elige duración, resolución y aspect ratio antes de generar.
- **Mejora de prompts:** Si el prompt es simple, usa la función "enhance" integrada en Higgsfield para obtener mejores resultados automáticamente.
- **Presets para foto a video:** Sube una foto, selecciona un preset predefinido (ej. "Cinematic", "Social Media", "Ad") y genera un video de alta calidad al instante. Ideal para ads y contenido social donde necesitas enganchar rápido.
- **Reference-based motion:** Sube un video de referencia + una imagen de personaje AI. Higgsfield mapea los movimientos del video real al personaje. Flujo: 1) graba un video corto de ti moviéndote, 2) sube la imagen del personaje AI, 3) selecciona "Reference Motion", 4) genera.
- **Max y Max Pro para Sora:** Después de generar con Sora 2 Pro, aplica "Max" o "Max Pro" (botón dedicado) para eliminar flickering y upscalar a mayor calidad. Usar solo si buscas calidad máxima, no para comparaciones justas.
- **Edición de iluminación:** Usa "Relight" en fotos o videos para cambiar la iluminación sin regenerar todo el contenido.
- **Herramientas de imagen adicionales:** Accede a Nano Banana Pro, GPT image y Higgsfield Soul desde el mismo dashboard para crear o retocar imágenes antes de animarlas.
- **Flujo para ads:** 1) Genera imagen con GPT image o sube foto, 2) aplica preset "Ad" para video, 3) ajusta duración (3-5 segundos para redes), 4) descarga. Sin cambiar de plataforma.

## Reglas para el erudito de Higgsfield

- Para **text-to-video cinematográfico** usa **Sora 2 Pro** con prompt detallado (incluye "cinematic lighting, multiple cuts, background music, voiceover"). Si hay flickering, aplica **Max Pro** post-generación.
- Para **image-to-video de personas reales** usa **Kling 3.0** o **Grok Imagine** (Sora 2 Pro no acepta fotos de personas). Para acción rápida (explosiones, pánico), prefiere **Kling 3.0** con prompt "realistic reaction time, characters run away from danger".
- Para **animación de personajes icónicos** (ej. Simpsons) usa **Grok Imagine** con prompt detallado del estilo visual y diálogos diferenciados por personaje.
- Para **video musical con ritmo** usa **Kling 3.0** con prompt "cuts synced to rhythm, realistic character, canvas transition effect".
- Para **conversión rápida foto→video** usa **presets de Higgsfield** (ej. "Cinema Studio" o "Seedance") en lugar de escribir prompts largos. Sube foto, selecciona preset, genera en segundos.
- Para **control de movimiento** usa **reference-based motion**: sube un video de 5-10 segundos de un humano moviéndose + imagen del personaje AI. Higgsfield replica el movimiento exacto.
- Para **editar iluminación de un video existente** usa **Relight** (botón en dashboard) sin necesidad de regenerar el video completo.

## Errores comunes / que evitar

- **Usar Sora 2 Pro para image-to-video con fotos de personas:** Sora rechaza o alucina. Siempre usa Kling o Grok para ese caso.
- **No especificar duración/resolución:** Por defecto genera valores genéricos. Siempre ajusta aspect ratio (16:9 para cine, 9:16 para redes) y duración (5-15 segundos según uso).
- **Prompts demasiado simples sin "enhance":** Si el prompt es básico, activa la función de mejora automática antes de generar para evitar resultados incoherentes.
- **Ignorar Max/Max Pro en Sora:** Si usas Sora 2 Pro y ves flickering, no regeneres; aplica Max Pro post-producción para corregir.
- **No usar presets para contenido social:** Perderás tiempo escribiendo prompts. Para ads o redes, sube foto y aplica preset "Social Media" o "Ad".
- **Esperar que un modelo domine todo:** Sora gana en text-to-video cinematográfico, pero Kling y Grok dominan image-to-video. Alterna según la tarea.
- **Subir videos de referencia muy largos para motion mapping:** Máximo 10 segundos para mejor precisión. Videos más largos causan desincronización.
