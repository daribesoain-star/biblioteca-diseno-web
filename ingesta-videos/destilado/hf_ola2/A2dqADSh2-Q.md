## Que ensena sobre Higgsfield (2-3 lineas)
Higgsfield Popcorn permite crear storyboards completos con estilo visual consistente (mismo personaje, ropa, iluminación y color) a partir de una sola imagen de referencia. La herramienta genera múltiples escenas manteniendo coherencia en vestimenta, tono de piel y ambiente, ideal para presentaciones a clientes o jefes sin necesidad de dibujar manualmente.

## Tips y flujos accionables
- **Flujo básico:** Sube una foto de referencia (ej: tu rostro) → Describe la historia en texto (ej: "waking up from bed, walking to car, getting ready to drive to town, go on stage to sing") → Presiona "Generate" → Espera a que Popcorn genere todas las escenas con estilo unificado.
- **Para mantener consistencia:** Usa la misma imagen de referencia para todas las escenas; el sistema replica automáticamente ropa, color de ojos, iluminación y encuadre.
- **Edición posterior:** Después de generar, puedes hacer clic en cada escena individual para añadir más detalles o cambiar la historia (ej: "next left down you can stand this show").
- **Prompt exacto usado en el video:** "waking up from bed, walking to car, getting ready to drive to town, stand by to go on stage to sing" (sin comillas, en inglés).
- **Truco para look cinematográfico:** Describe acciones secuenciales con transiciones claras (waking → walking → driving → stage) para que el modelo entienda el flujo narrativo.
- **Exportación:** Una vez generado, puedes descargar todo el storyboard como una secuencia de imágenes o video.

## Reglas para el erudito de Higgsfield
- Para **consistencia de personaje**: usa Popcorn con una sola foto de referencia (cara completa, buena iluminación frontal).
- Para **coherencia de vestimenta**: no cambies la ropa entre escenas; el modelo la mantiene automáticamente si usas la misma imagen base.
- Para **control de iluminación**: describe condiciones de luz en el prompt (ej: "morning light", "stage spotlight") para evitar cambios bruscos.
- Para **evitar alucinaciones**: mantén prompts cortos y lineales (máximo 5-6 acciones), sin saltos temporales o cambios de ubicación drásticos.
- Para **look cinematográfico**: usa descripciones de cámara implícitas (ej: "close-up waking up", "wide shot walking to car") en lugar de términos técnicos.

## Errores comunes / que evitar
- **No uses múltiples fotos de referencia** para diferentes escenas; el sistema pierde consistencia visual.
- **Evita prompts muy largos** (>10 acciones); el modelo puede mezclar elementos o ignorar detalles.
- **No cambies la ropa o peinado** entre escenas en el texto; el modelo no lo actualizará correctamente.
- **No esperes resultados perfectos** en la primera generación; usa la edición por escena para ajustar detalles.
- **No subas fotos con poca luz o borrosas** como referencia; el modelo replicará esos defectos en todas las escenas.
