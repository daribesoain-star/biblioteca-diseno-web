# This is Higgsfield Teams: How AI Video Teams Should Work
**Fuente:** hf_ola2 | https://youtu.be/IuDQZk-ndk4

## Que ensena sobre Higgsfield (2-3 lineas)
Higgsfield Teams unifica video, foto, edición y lip-sync en un solo sistema con carpetas compartidas, recreación con un clic y dashboard de créditos/analíticas. Permite ver el prompt exacto, modelo y configuración de cada asset, y arrastrar múltiples assets para organizarlos por cliente o proyecto. La colaboración en tiempo real elimina capturas de pantalla y mensajes de "qué settings usaste".

## Tips y flujos accionables

- **Recreación exacta de assets:** Haz clic en cualquier asset → verás el prompt, modelo y cada setting → botón "Recreate" con settings idénticos en un clic.
- **Organización por carpetas:** Crea carpetas por cliente o proyecto. Selecciona assets individuales, arrastra para seleccionar varios, o selecciona todos los de un día específico. Mueve assets a una o varias carpetas a la vez.
- **Compartir carpetas:** Crea una carpeta → compártela con tu equipo → los teammates ven los assets como si ellos los hubieran generado → pueden recrear, editar, animar o upscalear con los mismos settings.
- **Vista de asset:** Cada asset muestra prompt exacto, modelo usado (Cinema Studio, Seedance, Kling, etc.) y cada parámetro (motion control, cámara, etc.). Sin necesidad de preguntar.
- **Descarga y upscale:** Desde cualquier asset, botón "Download" o "Upscale" directo.
- **Créditos compartidos:** Dashboard unificado para analytics, seats y billing del equipo.
- **Flujo de trabajo recomendado:** Genera en Higgsfield → revisa en vista cuadrada o ratio original → organiza en carpetas por cliente → comparte carpeta con equipo → ellos recrean/editan sin re-preguntar settings.

## Reglas para el erudito de Higgsfield

- **Para compartir assets con equipo:** usa carpetas compartidas → teammates ven prompts y settings exactos → recrean con un clic sin re-preguntar.
- **Para organizar proyectos grandes:** usa carpetas por cliente o proyecto → selecciona múltiples assets arrastrando o por día → mueve a varias carpetas a la vez.
- **Para ver historial de generación:** haz clic en cualquier asset → prompt, modelo (Cinema Studio/Seedance/Kling), motion control, cámara, todos los parámetros visibles.
- **Para colaboración en tiempo real:** usa carpetas compartidas → teammates editan, animan, upscalean con settings idénticos.
- **Para gestión de equipo:** usa dashboard unificado de analytics, seats y billing → créditos compartidos entre el equipo.
- **Para evitar re-trabajo:** recrea assets con "Recreate" → settings exactos del original.

## Errores comunes / que evitar

- **No usar carpetas compartidas:** genera mensajes de "qué prompt usaste" y capturas de pantalla de settings.
- **No revisar el asset completo:** perder el contexto de prompt, modelo y parámetros que ya están visibles en cada asset.
- **No organizar por cliente/proyecto:** assets mezclados dificultan la colaboración y recreación posterior.
- **No usar el botón "Recreate":** re-escribir prompts manualmente introduce errores y pierde settings exactos.
- **No compartir créditos:** cada miembro con créditos separados complica la gestión y facturación.
- **No usar vista de asset para depuración:** si un resultado no es óptimo, revisa prompt exacto y modelo usado antes de regenerar.
