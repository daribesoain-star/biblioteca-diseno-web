# entrega-formatos-brandkit
**Fuente:** impl-logotipos | https://looka.com/blog/logo-file-formats/

## Qué enseña (2-3 líneas)
Explica la diferencia fundamental entre formatos vectoriales (escalables sin pérdida) y raster (basados en píxeles, pierden calidad al agrandarse). Detalla qué formatos de logo entregar según el uso: vector para maestro/impresión (EPS, SVG, PDF) y raster para web/digital (PNG con transparencia, JPG solo con fondo sólido). Además, describe cómo armar un brand kit completo con versiones para redes sociales.

## Lecciones accionables
- **Entrega siempre una carpeta, no un archivo suelto:** incluye todos los formatos, colores y versiones de layout correctos.
- **Para edición por otro diseñador:** envía **SVG** o **EPS** (formato vector nativo de Adobe Illustrator, llamado "working file" o "master file").
- **Para web:** usa **SVG** (escalable, calidad nítida, fondo transparente).
- **Para compartir con imprentas o personas sin software de diseño:** usa **PDF** (puede contener vector y raster en un solo paquete).
- **Para uso digital cotidiano (documentos, redes):** usa **PNG** con fondo transparente (permite superponer el logo sobre fotos, fondos de color o redes sin caja blanca).
- **JPG solo cuando se requiere fondo sólido y peso liviano:** nunca para logos con transparencia.
- **Brand kit completo debe incluir:**
  - PNG dimensionados para perfil y portada de: Facebook, Instagram, Etsy, LinkedIn, Pinterest, YouTube y Twitter.
  - Versión wallpaper y watermark.
  - Variantes sobre fondos negro y blanco.
- **En la práctica, el dueño usa 2-3 formatos:**
  - PNG transparente → documentos digitales.
  - SVG → sitio web.
  - PDF → imprenta.

## Reglas para agentes
- Usa **SVG o EPS** cuando el destino sea otro diseñador que editará el logo.
- Usa **PNG con transparencia** cuando el logo deba colocarse sobre fondos de color, fotos o en redes sociales.
- Usa **JPG solo si el logo tiene fondo sólido y se necesita peso de archivo reducido**; nunca para logos con área transparente.
- Nunca entregues un solo archivo: la entrega correcta es una **carpeta** con formatos, colores y versiones de layout.
- Incluye siempre en el brand kit versiones del logo sobre fondo negro y blanco.

## Errores comunes que evita o menciona
- **Entregar solo un formato:** el handoff correcto no es un archivo, es una carpeta completa.
- **Usar JPG para logos con transparencia:** el JPG no soporta fondo transparente, generará un recuadro blanco alrededor.
- **Enviar raster (PNG/JPG) a un diseñador para editar:** los raster pierden calidad al escalar y no son editables como vectores; siempre enviar SVG o EPS.
- **No incluir versiones sobre fondos oscuros y claros:** el logo puede volverse invisible si solo se entrega sobre fondo blanco.
