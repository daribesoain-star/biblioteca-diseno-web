# How to Create AI Music - Eleven Music tutorial [elevenlabs]
**Fuente:** ola4-herramientas | https://youtu.be/MCrkItqdRRI

## Qué enseña (2-3 líneas)
Este tutorial enseña a generar música de grado profesional con Eleven Music, describiendo canciones mediante prompts de texto. Muestra cómo personalizar la duración, editar secciones en la línea de tiempo, incluir/excluir estilos específicos, añadir voces o letras, y regenerar el resultado manteniendo las modificaciones.

## Lecciones accionables
- **Prompt inicial:** Escribe una descripción textual como `"An intense orchestra build for a thriller trailer"` para generar la música base.
- **Parámetros de generación:** Selecciona el número de variaciones (ej. 2) y la duración (elige "auto" o un valor exacto como `30 seconds`).
- **Edición de secciones:** En la línea de tiempo, ajusta la longitud de secciones individuales arrastrando sus bordes, o elimina secciones completas.
- **Estilos por sección:** Usa los style tags en la parte superior para incluir o excluir elementos globalmente (ej. `drums`). Para aplicar un estilo solo a una sección específica, selecciona esa sección y añade el tag directamente en ella.
- **Regeneración con cambios:** Después de modificar secciones y estilos, haz clic en "generate" nuevamente dentro del mismo proyecto para recrear la canción respetando el prompt original + todas las modificaciones.
- **Edición por prompt de texto:** En el cuadro de texto inferior, escribe instrucciones adicionales como `"Add a trailer voiceover"` o `"Make this trailer French"` para añadir voces o cambiar idiomas sin tocar la línea de tiempo.
- **Edición de voz generada:** Si se añadió un voiceover, puedes hacer clic en las palabras individuales para editarlas o reemplazar completamente el texto.
- **Ajuste del prompt original:** Si te gusta la base pero quieres variarla, edita el prompt original directamente y vuelve a generar dentro del mismo proyecto.
- **Exportación:** Haz clic en "download" para guardar como archivo `.MP3`. Usa el botón "share" para generar un video con visualizador de audio personalizable (elige color y haz clic en "generate").
- **Historial de cambios:** En la esquina superior derecha, revisa el historial de todas las generaciones y modificaciones realizadas.

## Reglas para agentes
- Al generar música, especifica siempre el género, el estado de ánimo y el contexto de uso (ej. "cinematic orchestras", "electro pop anthems", "background music for videos", "jingle for a podcast").
- Si el usuario necesita una duración exacta, proporciona el valor en segundos en lugar de usar "auto".
- Cuando el usuario quiera modificar solo una parte de la canción, usa la edición por secciones en la línea de tiempo, no regeneres desde cero.
- Para añadir voces o letras, usa el cuadro de prompt inferior con instrucciones claras como `"Add a trailer voiceover"` o escribe la letra exacta deseada.
- Si el usuario quiere cambiar el idioma de una voz generada, usa el prompt `"Make this [idioma]"` (ej. "Make this French").
- Siempre verifica el historial de generaciones (esquina superior derecha) antes de descartar versiones anteriores.
- Para compartir en redes sociales, genera el video con visualizador de audio personalizando el color antes de descargar.

## Errores comunes que evita o menciona
- **No regenerar desde cero cuando solo necesitas ajustar una sección:** El tutorial muestra que puedes editar la longitud o eliminar secciones específicas en la línea de tiempo y luego regenerar, en lugar de empezar un proyecto nuevo.
- **No aplicar estilos globalmente cuando solo se necesitan en una sección:** Explica que puedes incluir/excluir style tags en secciones individuales, no solo de forma global.
- **No ignorar el historial de cambios:** Muestra que en la esquina superior derecha puedes ver y volver a generaciones anteriores, evitando perder trabajo.
- **No olvidar que puedes editar el prompt original:** Si te gusta la base pero quieres variarla, puedes modificar el prompt original y regenerar dentro del mismo proyecto, sin empezar de cero.
- **No asumir que la voz generada es definitiva:** Permite editar palabras específicas del voiceover o cambiar el idioma completo con un simple prompt.
