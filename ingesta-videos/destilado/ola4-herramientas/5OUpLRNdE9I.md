# Suno Ai Tutorial 2025 [suno]
**Fuente:** ola4-herramientas | https://youtu.be/5OUpLRNdE9I

## Qué enseña (2-3 líneas)
Cómo usar Suno AI para generar música e instrumentales desde cero, navegando la plataforma, eligiendo planes, y aplicando prompting específico para obtener beats de calidad profesional. Cubre desde la interfaz básica hasta la creación de pistas instrumentales con control de género, instrumentos y ambiente.

## Lecciones accionables
- Ir a **suno.com** para acceder a la plataforma; la página principal muestra pestañas: **Home**, **Create**, **Studio**, **Library**, **Search**, **Hook**, **Explore**, **Radio**.
- Para crear una canción, hacer clic en **Create**; allí elegir entre modo **Simple** (texto libre) o **Custom** (personalizado).
- En el modo Simple, escribir el prompt directamente en el campo de texto; ejemplo usado: *"Create a hard hip hop 90s old school beat that has piano and trombone"*.
- Activar el toggle **Instrumental** si se desea solo música sin voz; desactivado genera canción completa con letra.
- Usar el **dado (dice)** para obtener una generación aleatoria sin prompt específico.
- Seleccionar la versión del modelo: **V5** es la mejor recomendada; **V3** funciona bien para trap duro.
- Verificar los **créditos** disponibles en la esquina superior; cada generación consume créditos según el plan.
- Usar **Workspace** para organizar proyectos: hacer clic en el ícono de workspace, crear uno nuevo (ej. "AI project"), y todas las generaciones se guardan allí.
- Para crear un beat: escribir prompt detallado con género, instrumentos y ambiente; ejemplo: *"Create a hard hip hop southern trap beat with organs and piano"*.
- Suno genera **dos versiones** por cada creación, con título y miniatura automáticos.
- Usar la opción **Remix** o **Remaster** en una pista generada para modificarla o mejorarla.
- Extraer **stems** (pistas separadas) desde el menú de cada canción generada.

## Reglas para agentes
- Usa **V5** como modelo predeterminado para mejor calidad general; usa **V3** solo cuando busques sonidos de trap agresivo.
- Activa **Instrumental** siempre que el objetivo sea solo música de fondo, beats para rap o samples.
- Sé **específico en el prompt**: incluye género, década, instrumentos exactos (piano, trombone, órganos, cuerdas, flautas) y estado de ánimo (emocional, chill, hard).
- Nunca asumas que un prompt genérico dará buen resultado; ajusta y repite hasta obtener el sonido deseado.
- Siempre verifica los **créditos restantes** antes de iniciar una sesión de generación múltiple.
- Usa **Workspace** para agrupar canciones de un mismo proyecto o álbum; no mezcles proyectos en el workspace por defecto.

## Errores comunes que evita o menciona
- No ser específico en el prompt lleva a resultados genéricos o "elevator music" en lugar del estilo deseado.
- Ignorar la selección de versión del modelo (V5 vs V3) puede dar resultados no óptimos para ciertos géneros.
- Olvidar activar el toggle **Instrumental** cuando solo se necesita beat sin voz.
- No usar **Workspace** para organizar proyectos grandes, lo que dificulta encontrar pistas después.
- Subestimar la necesidad de probar múltiples generaciones y prompts para afinar el resultado.
