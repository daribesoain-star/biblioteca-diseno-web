# Flux AI Tutorial - 2026 [fal]
**Fuente:** ola4-herramientas | https://youtu.be/D1tcNzot2HI

## Qué enseña
El tutorial explica cómo usar Flux 1.1 Pro para generar imágenes hiperrealistas, cubriendo dos métodos principales: la plataforma oficial flux.ai y el servicio alternativo Together AI. También muestra cómo mejorar prompts usando ChatGPT con módulos especializados para Flux.

## Lecciones accionables
- **Registro en flux.ai:** Ve a flux.ai, selecciona "Create images", luego inicia sesión. Usa el "prompt generator" en la barra superior para crear prompts optimizados.
- **Generación en flux.ai:** Establece dimensiones (ej. 16:9) y lanza la generación. Cada imagen cuesta 8 créditos (2 centavos por crédito). Plan: $15.90/mes.
- **Registro en Together AI:** Ve a together.ai, haz clic en "Get started", regístrate con Google/LinkedIn/GitHub. Completa datos: username (<20 caracteres, sin símbolos), company name, responde "Yes" a "Do you have a Gen AI API production?", selecciona "Enthusiast". Recibes $1 en créditos gratis.
- **Configurar Together AI:** Ingresa tarjeta de crédito para desbloquear el modelo Flux 1.1 Pro. Selecciona "Build tariff plan" → "Get started". El $1 se carga temporalmente pero lo reembolsan.
- **Generar en Together AI:** Ve a la sección "Generation", asegúrate de que el modelo seleccionado sea "flux 1.1 pro". Pega el prompt y genera. Las imágenes se guardan automáticamente.
- **Mejorar prompts con ChatGPT:** Busca módulos GPT como "Flux Prompt Pro", "Flux Prompt", "Mid Journey Image Prompt Maker" o "Flux Prompt Generator". Pide: *"Help me create a flux prompt"*. Copia las sugerencias y pégalas en Together AI.
- **Prompts simples efectivos:** Usa un nombre aleatorio + 1-2 palabras descriptivas. Ejemplo: *"selfie"* con un nombre genérico. No necesitas prompts largos.
- **Ajuste de estilo:** Cambia "portrait" por "selfie" para variar el resultado manteniendo realismo.

## Reglas para agentes
- Usa Together AI como alternativa gratuita cuando se acaben los créditos en flux.ai.
- Nunca generes sin especificar el modelo: siempre selecciona "flux 1.1 pro" para máximo realismo.
- Usa ChatGPT con módulos especializados en Flux cuando el prompt inicial dé resultados insatisfactorios.
- Verifica el saldo de créditos en Together AI después de cada tanda de generaciones.
- No uses prompts demasiado detallados para retratos simples; un nombre + 1-2 palabras basta.

## Errores comunes que evita o menciona
- **Falta de créditos en flux.ai:** Se agotaron rápido (8 créditos por imagen). Solución: usar Together AI con $1 gratis.
- **Prompt mal optimizado:** El prompt inicial de "mujer con libro" dio resultado insatisfactorio en Together AI. Solución: usar ChatGPT para generar un prompt mejor estructurado.
- **Olvidar ingresar tarjeta de crédito en Together AI:** Sin tarjeta, el modelo Flux 1.1 Pro no se desbloquea, aunque prometen reembolsar el cargo temporal.
- **Artefactos en dedos:** La imagen generada mostraba dedos demasiado largos. Es un error común en IA que Flux 1.1 Pro reduce pero no elimina por completo.
- **Fondo borroso por defecto:** Es una limitación típica de generación de imágenes; no especifica cómo evitarlo.
