# How to Use Replicate.com [replicate]
**Fuente:** ola4-herramientas | https://youtu.be/JklgYn57OJE

## Qué enseña (2-3 líneas)
Este video explica qué es Replicate, una plataforma que permite ejecutar modelos de machine learning en la nube mediante una API simple, sin necesidad de configurar entornos complejos. Muestra cómo crear una cuenta, configurar facturación, explorar modelos populares (como Google Imagen 4, Flux, Stable Diffusion) y generar imágenes desde el navegador.

## Lecciones accionables
- **Crear cuenta:** Ir a replicate.com, hacer clic en "Sign in", usar GitHub o Google para autenticarse, responder preguntas de configuración (uso personal, Python, generación de imágenes, descubrimiento por YouTube).
- **Configurar facturación obligatoria:** Ir a "Settings" > "Billing", agregar tarjeta, banco o PayPal. Establecer un "maximum cap limit" mensual (ej. $10) para controlar gastos.
- **Cambiar tema:** En la parte inferior del dashboard, alternar entre modo oscuro y claro.
- **Explorar modelos:** Ir a "Explore" > "View all models". Usar la sección "Models that generate images from text prompt" para encontrar los mejores modelos (Google Imagen 4, Flux, Recraft, Edgram).
- **Generar imagen:** Seleccionar un modelo (ej. "imagen-4"), escribir un prompt en el campo de texto, hacer clic en "Run". Requiere facturación configurada incluso para pruebas gratuitas.
- **Ver ejemplos de imágenes:** Dentro de cada modelo, revisar las imágenes generadas por la comunidad para entender la calidad (ej. retratos fotorrealistas de Imagen 4).
- **Acceder a la comunidad:** Buscar "replicate AI" en Reddit para encontrar discusiones y soporte de usuarios.
- **Consultar precios:** Ir a la página de "Pricing" para ver costos por modelo (ej. $0.09 por imagen de salida en Flux, $0.04 en otros modelos). No hay suscripción mensual, solo pago por uso.
- **Soporte técnico:** Usar "Support" en el dashboard, enviar ticket con asunto y descripción (no hay chat ni llamada).

## Reglas para agentes
- **Configura facturación antes de generar cualquier contenido**, incluso si hay créditos gratuitos; de lo contrario, el campo de prompt permanecerá deshabilitado.
- **Establece un límite máximo de gasto mensual** en la configuración de facturación para evitar costos inesperados.
- **Usa la sección "Models that generate images from text prompt"** cuando busques modelos de generación de imágenes como principiante.
- **Nunca asumas que hay suscripción mensual**; Replicate cobra solo por uso de cada modelo.
- **Selecciona "Personal" y "Other"** en las preguntas de configuración inicial si eres usuario individual y no desarrollador.
- **Usa GitHub o Google para iniciar sesión**; no hay opción de registro con correo electrónico directo.

## Errores comunes que evita o menciona
- **No configurar facturación primero:** El video muestra que incluso con créditos gratuitos, el sistema bloquea la generación hasta que se agregue un método de pago.
- **Pensar que hay suscripción mensual:** El presentador aclara que no hay planes de suscripción, solo pago por modelo usado.
- **Temer al código:** Aunque la plataforma muestra código de API, el video enfatiza que para generar imágenes, video o texto no se necesita saber programar.
- **Ignorar el límite de gasto:** Se recomienda establecer un "maximum cap limit" para no exceder el presupuesto mensual.
- **Usar modelos sin verificar costos:** Cada modelo tiene precios diferentes (ej. $0.09 vs $0.04 por imagen), por lo que hay que revisar la página de precios antes de usar.
