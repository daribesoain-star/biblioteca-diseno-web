# Create VIRAL T-Shirt Designs Using Ideogram AI Without Photoshop [ideogram]
**Fuente:** ola4-herramientas | https://youtu.be/aLQB6l6KgXI

## Qué enseña (2-3 líneas)
Cómo generar diseños virales de camisetas usando Ideogram AI sin Photoshop, aplicando una fórmula de prompts estructurada para obtener resultados profesionales. Incluye el proceso completo desde la escritura del prompt hasta la preparación del diseño para print on demand, con consejos para evitar errores comunes de copyright y legibilidad.

## Lecciones accionables
- **Fórmula de prompt estructurada:** Usa esta estructura exacta: `[Nicho o sujeto] + [Gancho divertido o emocional] + [Estilo visual o elemento de diseño] + [Texto opcional]`. Ejemplo: "Grumpy old cat sitting on a throne. Text: 'Pause before peasants'. Bold medieval font, fantasy cartoon style."
- **Prompt VERBATIM para diseño viral:** "Imagine a chubby raccoon holding a pizza, nineties cartoon style, bold retro text that says 'Snack bandit'." — usar modelo Ideogram 3.0, aspect ratio 4.5, escala 3.0.
- **Prompt VERBATIM para diseño de ganso:** "Angry goose holding a baseball bat, standing in front of a barn. Text: 'Don't honk with me'. Character style, vintage t-shirt design, clear readable text."
- **Prompt VERBATIM para diseño de gallina:** "Humorous cartoon style illustration of a white chicken with a red comb and orange beak, standing with a surprise expression. Above it the text 'Chicken game' is written in bold, slightly distressed lettering, followed by 'Don't look at the chicken' in a smaller font. The phrase 'Game over' is written near the chicken's feet."
- **Prompt VERBATIM para diseño de texto con símbolo:** "White background with the phrase 'Mama of the sweet one' in pastel colors, incorporating a chocolate chip cookie illustration replacing the letter O in 'one'."
- **Keywords de estilo para usar en prompts:** "vintage graphic", "sticker style", "worn print look", "cartoon outline", "bold text only", "t-shirt design", "line art", "coloring book outline", "image outline", "clear readable text", "vintage t-shirt design layout".
- **Parámetros exactos en Ideogram:** Modelo 3.0, aspect ratio 4.5, escala 3.0, visibilidad pública.
- **Cómo refinar un diseño existente:** Usa la función de edición con instrucciones como "Make the text larger and clearer. Keep the cartoon style. Add an angry face on the goose."
- **Checklist de preparación para print on demand:** (1) Verificar texto: zoom para confirmar que no haya errores de ortografía o letras deformadas. (2) Recortar y centrar: asegurar que el gráfico principal esté balanceado y ocupe la mayor parte del lienzo. (3) Tamaño correcto: exportar en 4500x5400 píxeles a 300 DPI; usar upscale dentro de Ideogram o herramientas como Let's Enhance. (4) Exportar en PNG transparente para camisetas. (5) Verificar marcas registradas con TM Hunt.
- **Herramientas gratuitas para corrección:** Canva o PhotoP para arreglar texto si Ideogram genera errores.

## Reglas para agentes
- Usa la fórmula `[Nicho o sujeto] + [Gancho divertido/emocional] + [Estilo visual] + [Texto opcional]` para cada prompt de diseño de camiseta.
- Usa el modelo Ideogram 3.0 con aspect ratio 4.5 y escala 3.0 para todos los diseños de camisetas.
- Incluye siempre "clear readable text" o "bold text" en el prompt cuando el diseño contenga texto.
- Nunca generes prompts de más de una oración o párrafo; limítate a una idea visual fuerte con estilo claro.
- Nunca incluyas referencias a Disney, Nike, celebridades o cualquier marca registrada sin verificar derechos.
- Siempre verifica el texto generado con zoom al 100% antes de considerar el diseño como final.
- Siempre exporta en PNG transparente y a 4500x5400 píxeles a 300 DPI para plataformas de print on demand.
- Usa TM Hunt para verificar que frases o conceptos no estén registrados como marca antes de publicar.

## Errores comunes que evita o menciona
- **Escribir prompts demasiado largos:** "No estás escribiendo una novela" — limita el prompt a una idea visual fuerte, no un párrafo.
- **Confiar ciegamente en el texto generado por IA:** El texto puede salir "jumbled or weird" — siempre haz zoom para verificar ortografía y legibilidad.
- **Diseños descentrados o con mucho espacio en blanco:** Los diseños de IA pueden salir "off center or too much white space" — recorta y centra manualmente.
- **Baja resolución para impresión:** Ideogram no exporta en tamaños de impresión por defecto — siempre escala a 4500x5400 píxeles a 300 DPI.
- **Usar frases o imágenes con derechos de autor:** "Stay away from Disney, Nike, or any celebrity faces" — verifica siempre con TM Hunt antes de publicar.
- **Sobredimensionar el prompt con detalles innecesarios:** "Don't overcomplicate it" — una idea visual + estilo claro es suficiente.
- **No regenerar cuando el animal o elemento no sale correcto:** Si el ganso no se ve bien, "regenerate it" — es IA, a veces necesita reintentos.
