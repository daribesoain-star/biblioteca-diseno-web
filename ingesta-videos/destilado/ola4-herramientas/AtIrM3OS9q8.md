# EXACTLY How to use Higgsfield AI in 2026 [higgsfield]
**Fuente:** ola4-herramientas | https://youtu.be/AtIrM3OS9q8

## Qué enseña
El video revela siete funciones ocultas de Higgsfield AI para generar imágenes y videos de calidad profesional con consistencia de personaje, transiciones cinematográficas y control total de cámara. Enseña a seleccionar modelos de IA según el estilo deseado, crear personajes consistentes en múltiples escenas, y usar herramientas como AI Influencer y Cinema Studio para contenido publicitario de alto nivel.

## Lecciones accionables
- **Seleccionar modelo de imagen según objetivo:** Para ultra-realismo usar "Nano Banana 2". Para estilos artísticos usar "Sea Dream 5.0". Para generar imágenes de ti mismo con características de personaje usar "Higgsfield Soul".
- **Configurar resolución:** Al generar imagen, seleccionar relación de aspecto 16:9 y resolución 4K para máxima calidad.
- **Usar Outfit Swap:** Ir a "Apps" → "Outfit Swap". Subir dos imágenes: la del personaje y la de la vestimenta deseada. Genera cambio de ropa manteniendo pose, fondo e iluminación.
- **Generar múltiples ángulos con Shots:** Ir a "Apps" → "Shots". Subir una imagen y hacer clic en "Generate". Obtienes 9 tomas profesionales (close-ups, upward framings, overhead shots) en 30 segundos sin escribir prompts. Seleccionar todas, upscalear y descargar.
- **Controlar ángulo de cámara con Angles:** Usar el círculo en la pestaña derecha para colocar la cámara exacta. Posición alta = drone shot; posición baja = lower angle.
- **Crear transiciones suaves:** Ir a "Apps" → "Transitions". Subir imagen de inicio y de fin. Elegir entre más de 20 transiciones (recomendada: "Raven"). Seleccionar duración y generar.
- **Entrenar personaje consistente:** Ir a "Character" en la parte superior. Subir entre 20 y 30 fotos de referencia (más ángulos = mejores resultados). Usar previamente "Shots" para generar 9 ángulos, upscalear y descargar. Repetir hasta tener 20+ imágenes. Subirlas todas, asignar nombre (ej: "Kim") y esperar que la IA entrene el modelo.
- **Usar personaje entrenado:** Ir a "Image creation", seleccionar el personaje, escribir prompt. El personaje se mantiene consistente en cualquier entorno, vestimenta o situación.
- **Crear AI Influencer:** Ir a "AI Influencer". Configurar: Character type (human/elf/crocodile/octopus), gender, ethnicity, skin color, eye color, skin condition (freckles/birthmarks/scars), age, body type, style (hair/tattoos), render style (hyperrealistic). Generar.
- **Usar Cinema Studio:** Seleccionar preset "Premium Large Format Digital" con lente "Classic Anamorphic" a 35mm y F1.4 para look cinematográfico con blur de fondo y profundidad natural. Configurar hasta 4 imágenes por generación, relación de aspecto 16:9, resolución de 1K a 4K.
- **Usar grid preview:** Seleccionar layout 2x2 para previsualizar múltiples generaciones en diferentes grids sin usar créditos extra.
- **Usar "What's Next":** Después de generar imagen, activar esta función para obtener múltiples variaciones de lo que podría suceder después, sin escribir prompts.
- **Animar con Director Panel:** Ir a "Video section". Elegir movimiento de cámara entre docenas de opciones. Ajustar "Speed ramping" manualmente (speed up / slow motion) o dejar en "auto". Seleccionar duración (ej: 5 segundos). Elegir "Genre" para transmitir emociones (action, horror, suspense, intimate). Pegar start frame y end frame. Generar.

## Reglas para agentes
- Usa "Nano Banana 2" cuando necesites imágenes ultra-realistas; usa "Sea Dream 5.0" cuando busques estilos artísticos.
- Nunca modifiques el prompt y regeneres esperando que solo cambie un elemento; usa "Outfit Swap" para cambios específicos de vestimenta.
- Sube siempre entre 20 y 30 fotos de referencia al entrenar un personaje en "Character"; nunca menos de 20.
- Usa la función "Shots" antes de entrenar un personaje para generar los 9 ángulos necesarios como imágenes de referencia.
- Selecciona "Hyperrealistic" como render style en AI Influencer cuando quieras que parezca un humano real.
- Usa el preset "Premium Large Format Digital" con lente "Classic Anamorphic" a 35mm y F1.4 en Cinema Studio para obtener calidad cinematográfica profesional.
- Activa "What's Next" después de generar una imagen cuando necesites ideas de continuación de historia sin escribir prompts.
- Selecciona el género ("Genre") en el Director Panel según la emoción que quieras transmitir (intimate para escenas emotivas, action para dinámicas).

## Errores comunes que evita o menciona
- **Elegir el modelo de IA incorrecto:** Los principiantes no saben qué modelo seleccionar y obtienen resultados promedio. Usar la regla: Nano Banana 2 para realismo, Sea Dream 5.0 para artístico, Higgsfield Soul para selfies.
- **Modificar el prompt y regenerar esperando cambios mínimos:** La IA cambia también elementos que te gustaban. Usar "Outfit Swap" para cambios precisos.
- **Escribir prompts complejos para cada ángulo:** Antes se necesitaban 9 prompts y docenas de iteraciones para obtener diferentes ángulos de un personaje. Usar "Shots" para obtener 9 ángulos en un clic sin escribir prompts.
- **No tener suficientes ángulos de referencia:** Para consistencia de personaje se necesitan 20-30 fotos desde diferentes ángulos. Usar "Shots" repetidamente para generarlas.
- **Ignorar las transiciones entre escenas:** Transiciones mal hechas hacen que el video se vea poco profesional. Usar "Transitions" con opciones como "Raven" para resultados suaves.
