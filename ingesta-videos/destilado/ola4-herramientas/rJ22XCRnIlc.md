# How To Make Instant Movie Poster With Ideogram AI (Ideogram Beginner Tutorial) [ideogram]
**Fuente:** ola4-herramientas | https://youtu.be/rJ22XCRnIlc

## Qué enseña
Cómo generar pósters de películas realistas usando Ideogram AI, incluyendo la selección de prompts, el uso del "magic prompt", la relación de aspecto correcta y la función de remix para refinar resultados. Es un tutorial para principiantes que muestra el flujo completo desde la idea hasta la generación final.

## Lecciones accionables
- Usa la relación de aspecto **9:16** para pósters de películas verticales.
- Activa **"Magic Prompt" en Auto** para que Ideogram extienda automáticamente tu prompt básico a uno más detallado y efectivo.
- Estructura el prompt como: `[nombre de la película] movie poster [título o texto]`. Ejemplo: `Sonic movie poster Green Hill Zone`.
- Genera al menos **2 tandas** (clics en "generate") para obtener más opciones.
- Para personajes no icónicos (ej. celebridades), genera primero el fondo del póster y luego añade al personaje en un paso separado.
- Usa la función **"Remix"** (crear variación) para refinar una imagen que te guste: haz clic en "create a variation", luego ajusta el **image weight** (peso de imagen) de 0 a 2 para controlar qué tan similar se mantiene al original.
- Pon las generaciones en **"Private"** para mantenerlas ocultas.
- Edita los resultados en Photoshop si necesitas ajustes finos.

## Reglas para agentes
- Usa siempre **9:16** cuando generes pósters de películas.
- Activa **Magic Prompt en Auto** en cada generación de póster.
- Nunca uses prompts demasiado largos o complejos; el sistema ya sabe lo que necesitas con frases simples.
- Cuando remixes, ajusta el **image weight** según el grado de cambio deseado: valor bajo (cercano a 0) para cambios grandes, valor alto (cercano a 2) para mantener similitud.
- Si el personaje no es ampliamente conocido, genera primero el fondo y luego añade el personaje por separado.

## Errores comunes que evita o menciona
- No uses prompts excesivamente detallados; el Magic Prompt se encarga de expandirlos automáticamente.
- No esperes que Ideogram genere personajes específicos no icónicos (ej. John Wick) con precisión; es mejor usarlo para brainstorming o fondos.
- No uses Midjourney, Adobe Firefly o ChatGPT para texto en pósters porque no manejan bien la tipografía; Ideogram sí lo hace correctamente.
