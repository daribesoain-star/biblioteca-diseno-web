## Qué enseña
El video explica cómo seleccionar tipografías y formas para logotipos basándose en la psicología de marca, el público objetivo y la intención comunicativa. También cubre el uso de cuadrículas, herramientas de Illustrator (Shape Builder, Offset Path, Puppet Warp) y un proceso estratégico de diseño de logotipos desde la investigación hasta la entrega final.

## Lecciones accionables
- **Tipografía para Nike:** Usar **Futura Condensed Extra Black** en cursiva para comunicar fuerza, estabilidad, modernidad y sensación de movimiento en marcas deportivas.
- **Tipografía para Aston Martin:** Usar **Optima Roman** para transmitir profesionalismo, elegancia, refinamiento y formalidad en marcas de lujo.
- **Tipografía para Armani:** Usar una **tipografía serif** para evocar lujo, elegancia, estilo y prestigio en moda de alta gama.
- **Proceso de selección tipográfica:** Preguntar: ¿Qué vende la marca? ¿A quién se lo vende? ¿Qué emociones quiere comunicar? Luego elegir la tipografía que coincida con esas respuestas.
- **Tipos de logotipos:**
  - **Letter Mark / Monograma:** Usar cuando el nombre de la marca es demasiado largo (ej. NASA). Requiere hacer icónicas unas pocas letras.
  - **Word Mark:** Ideal para startups o negocios nuevos. Mantener el nombre corto. Ejemplos: Disney, Google, Coca-Cola, FedEx.
  - **Brand Mark / Pictorial Mark:** Solo símbolo, sin texto. Ventaja: reconocimiento global sin barreras de idioma. Ejemplos: Twitter, Target.
  - **Combination Mark:** Logotipo + símbolo. El más común y fácil de trabajar. Asegurar que la tipografía coincida con el estilo del símbolo.
  - **Emblem Logo:** Forma circular que sugiere autoridad, autenticidad y unión. Usar para organizaciones, gobiernos o equipos. Ejemplo: Starbucks.
  - **Mascot Logo:** Da una cara o personaje a la marca. Usar para restaurantes o marcas de comida. Difícil de que perdure en el tiempo.
- **Uso de cuadrículas en logotipos:**
  - Las cuadrículas se usan en las **etapas finales** del diseño, no al principio.
  - **Cuadrículas circulares/ovaladas:** Ideales para logotipos orgánicos y naturales.
  - **Cuadrículas con líneas rectas (45°):** Ideales para diseños rígidos y hechos por el hombre.
  - Usar círculos para medir y mantener el mismo ancho en diferentes partes del diseño.
  - Medir ángulos con precisión (ej. 72°) y replicarlos en múltiples ubicaciones.
  - Usar la cuadrícula para mantener consistente el **micro-espacio blanco** entre el símbolo y el texto.
- **Configuración de cuadrícula en Illustrator:**
  - Ir a **View > Show Grid** y activar **Snap to Grid**.
  - Usar la herramienta **Elipse** (L) con Alt+Shift para círculos perfectos.
  - Usar **Ctrl/Cmd + Y** para entrar en **Outline Mode** y verificar precisión.
  - Usar la **Shape Builder Tool** para unir/eliminar formas.
- **Offset Path en Illustrator:**
  - Seleccionar la forma, ir a **Object > Path > Offset Path**.
  - Marcar **Preview** para ver cambios en tiempo real.
  - Usar flechas arriba/abajo para ajustar; **Shift** incrementa de 10 en 10; **Ctrl/Cmd** ajusta en decimales.
  - Después de aplicar Offset Path, usar **Shape Builder Tool** con Alt para eliminar áreas blancas sobrantes.
- **Técnica de logotipo con Offset Path y Scissors Tool:**
  - Crear un círculo con trazo grueso.
  - Usar **Scissors Tool (C)** para cortar en los puntos de anclaje inferior e izquierdo, eliminando 3/4 del círculo.
  - Con **Pen Tool (P)**, trazar líneas rectas (con Shift) para cerrar la forma.
  - Crear un cuadrado perfecto con **Rectangle Tool (M)** + Shift.
  - Aplicar Offset Path, cambiar el offset a blanco, y usar Shape Builder para limpiar.
- **Scale Test con Actions Panel:**
  - Ir a **Window > Actions**, crear nueva acción (ej. "Scale Test"), presionar **Record**.
  - Duplicar el logotipo (Alt+arrastrar), usar **Scale Tool** para reducirlo al 75%, presionar Enter.
  - Repetir varias veces, luego presionar **Stop**.
  - Para probar cualquier diseño, seleccionarlo y presionar **Play** en la acción guardada.
- **Artboard Hack:**
  - Seleccionar el logotipo, tomar la **Artboard Tool**, mantener **Shift** y hacer clic en el diseño para crear un artboard perfecto al tamaño del logo.
  - Eliminar el artboard original con **Backspace**.
- **Puppet Warp para tipografía:**
  - Seleccionar el texto, ir a **Free Transform Tool** > **Puppet Warp**.
  - Hacer clic para añadir nodos/ancas; los nodos fijan esa área mientras se editan otras.
  - Para suavizar distorsiones no deseadas, usar **Direct Selection Tool (A)** y luego **Smooth Tool** (ajustar configuración con doble clic en la herramienta).
- **Uso de formas predefinidas:** Para logotipos de productos hechos por el hombre, usar formas geométricas de bordes rectos. Para logotipos naturales/orgánicos, usar curvas y líneas suaves.
- **Mirror function en Illustrator:** Usar para diseños simétricos, ahorra tiempo y garantiza precisión.
- **Global Colors y Color Groups:**
  - Seleccionar el diseño, crear un **nuevo grupo de color** desde el panel Swatches, marcar ambas casillas inferiores.
  - Desde el panel **Color**, ajustar el **Tint** (porcentaje) para crear variaciones del mismo color.
  - Usar el icono del **círculo de color** en el grupo para editar Hue, Saturation y Brightness.
  - Guardar cambios como un nuevo grupo de color para probar variantes.
- **Creative Cloud Libraries para branding:**
  - Crear una nueva librería desde el panel **Libraries**.
  - Arrastrar el símbolo del logotipo, el texto, los swatches de color y el **Character Style** a la librería.
  - Esto crea un catálogo de marca compartible entre dispositivos y con clientes.
- **Proceso completo de diseño de logotipo (caso Iron Grip Jiu-Jitsu):**
  1. **Reunión inicial con el cliente:** Conocer el negocio, metas, puntos débiles y público objetivo.
  2. **Extraer keywords:** Ej. Jiu-Jitsu, enseñanza, equipo, fuerte, comunidad.
  3. **Crear mood boards** para definir la dirección visual.
  4. **Investigar la industria:** Aprender terminología y cultura del sector.
  5. **Generar pares de palabras** a partir de las keywords (ej. "grip" + "fuerte").
  6. **Pensar visualmente:** Cómo representar esos pares en un concepto gráfico.
  7. **Aplicar psicología de formas:** Bordes rectos para fuerza/estabilidad; círculos para comunidad/unión.
  8. **Elegir tipografía:** Sans-serif bold para deportes; evitar serif por ser demasiado formal.
  9. **Iterar y refinar:** Resolver problemas de espacio negativo, probar múltiples soluciones.
  10. **Presentar al cliente** y aceptar su decisión final.

## Reglas para agentes
- Usa **Futura Condensed Extra Black** en cursiva cuando el logotipo deba comunicar fuerza, velocidad y modernidad para una marca deportiva.
- Usa **Optima Roman** cuando el logotipo deba transmitir elegancia, formalidad y refinamiento para una marca de lujo.
- Usa tipografía **serif** cuando el logotipo deba evocar lujo, prestigio y tradición en moda de alta gama.
- Nunca uses efectos de software como gradientes excesivos, sombras paralelas, biseles o filtros en un logotipo; la forma y composición deben hablar por sí mismas.
- Nunca entregues un logotipo en formato raster (pixelado o borroso); siempre usa vectores.
- Nunca ignores la alineación, el espaciado uniforme y el grosor de línea consistente en el logotipo.
- Nunca prometas más de lo que puedas entregar; siempre sobreentrega (ej. si prometes 14 días, entrega en 10; si prometes 3 conceptos, entrega 4).
- Nunca empieces a diseñar en la computadora sin antes hacer investigación, análisis de audiencia y desarrollo de conceptos (strategic design thinking).
- Nunca te conformes con la primera iteración de un diseño; itera al menos una vez, ya que la segunda versión casi siempre es mejor.
- Nunca entregues archivos de logotipo sin formato adecuado: incluye archivos vectoriales, modos de color correctos y nombres de archivo claros.
- Usa cuadrículas solo en las etapas finales del diseño para perfeccionar y limpiar el logotipo, no al inicio.
- Usa círculos para medir y mantener anchos consistentes en diferentes partes del diseño.
- Usa **Snap to Grid** y **Outline Mode (Ctrl/Cmd+Y)** en Illustrator para garantizar precisión.
- Usa la **Shape Builder Tool** con Alt para eliminar áreas no deseadas después de aplicar Offset Path.
- Usa la **Actions Panel** para crear un "Scale Test" y verificar que el logotipo funcione a diferentes tamaños.
- Usa **Global Colors** y **Color Groups** para crear y gestionar variantes de color del logotipo.
- Usa **Creative Cloud Libraries** para almacenar y compartir todos los activos de marca (símbolo, tipografía, colores, estilos de carácter).
- Usa **pares de palabras clave** para generar conceptos visuales de logotipo.
- Aplica psicología de formas: formas rectas para productos hechos por el hombre; formas curvas para orgánicos/naturales; círculos para comunidad; triángulos para fuerza/estabilidad.

## Errores comunes que evita o menciona
- **Sobreuso de efectos de software:** Gradientes excesivos, tipografía compleja, sombras, biseles y filtros hacen que el logotipo se vea gimmicky, poco profesional y anticuado (como clip art de los 90).
- **Mala ejecución técnica:** Pixelación, desenfoque (por usar raster en lugar de vector), espaciado desigual, alineación pobre, grosor de línea inconsistente y composición extraña.
- **Prometer de más y entregar de menos:** Mala comunicación, plazos incumplidos, logotipos poco originales (parecidos a plantillas de Canva) y revisiones limitadas.
- **Falta de pensamiento estratégico:** Ignorar el análisis del público objetivo, diseñar sin alinear con la misión/identidad de la marca, y crear logotipos que no resisten el paso del tiempo.
- **Falta de iteración:** Conformarse con el primer diseño sin revisarlo ni mejorarlo; la segunda iteración casi siempre es mejor.
- **Mala entrega de archivos:** Archivos sin formato vectorial, modos de color incorrectos, nombres de archivo confusos que dificultan el uso por parte del cliente.
- **Empezar a diseñar directamente con el Shape Builder Tool sin antes investigar y desarrollar conceptos.**
- **No verificar gaps en las formas** al usar Shape Builder Tool; activar "Check for Gaps" en la configuración de la herramienta.
- **Tener configuraciones de snapping incorrectas:** No tener "Align to Pixel Grid" desactivado y tener tolerancias de snapping muy altas.
- **No probar el logotipo en aplicaciones reales** (tarjetas de presentación, cartas, banners) antes de enviarlo al cliente.
