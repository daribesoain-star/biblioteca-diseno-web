## Qué enseña
Proceso completo de diseño de identidad de marca en 5 pasos: desde brief y bocetaje hasta mockups finales. Cubre trazado vectorial en Illustrator, selección cromática, tipografía, creación de patrones y stickers, y presentación de mockups en Photoshop.

## Lecciones accionables
- **Brief con ChatGPT:** Prompt usado: *"give a brief on a dog treats"* → obtiene nombre "Paw Bites" y brief con personaje mascota.
- **Trazado rápido en Illustrator:** Usar **Pen Tool** solo para líneas rectas (click sin arrastrar), luego seleccionar trazado y aplicar **Curvature Tool** (atajo: Shift + `~`) haciendo click y arrastrando en el centro de cada segmento para curvar.
- **Suavizado de curvas:** Seleccionar trazado → **Shaper Tool** → elegir **Smooth Tool** → click y arrastrar sobre bordes irregulares para alisar automáticamente.
- **Creación de grid de espaciado:** Tomar forma de referencia (ej. nariz del logo) → copiar ancho en alto en panel **Transform** para hacer cuadrado perfecto → convertir a stroke → duplicar y dividir ancho escribiendo `/ 2` en el campo Width → repetir para obtener X, X/2, X/4 → usar X/4 como espaciador entre logo y texto, X como margen exterior en las 4 esquinas.
- **Selección de color:** Investigar competidores → elegir color opuesto al que usan (ej. si usan azul/naranja/blanco, elegir rojo) → copiar código HEX del color primario → pegarlo en ChatGPT con prompt: *"give secondary color suggestions"*.
- **Tipografía:** Elegir **display font** (la usada en el logo) + **secondary body font** sans-serif legible (ej. Metropolis).
- **Patrón en Illustrator:** Crear elemento base (ej. hueso) → colocar forma principal centrada → dibujar cuadrado alrededor dejando espacio interior → duplicar elementos en diferentes tamaños rellenando el cuadrado sin huecos → eliminar cuadrado → seleccionar todo → **Object > Pattern > Make** → ajustar espaciado en panel → aplicar a cualquier forma desde el panel de colores (aparece como nuevo swatch).
- **Stickers:** Usar ilustraciones del personaje en diferentes poses → añadir bocadillo de diálogo o efecto de sello.
- **Mockup en Photoshop:** Abrir archivo PSD → hacer doble click en la capa del mockup → diseñar dentro (fondo con color primario, nombre, ilustraciones, stickers) → guardar → el cambio se refleja automáticamente → exportar como PNG.
- **Enmascarado en Photoshop:** Crear layout básico → colocar ilustración → click derecho → **Create Clipping Mask** para que encaje dentro de la forma.

## Reglas para agentes
- Usa **Pen Tool** solo para líneas rectas (click sin arrastrar) cuando traces bocetos; nunca curves manualmente con Pen Tool.
- Aplica **Curvature Tool** después de tener líneas rectas para generar curvas suaves con un solo click y arrastre.
- Usa **Smooth Tool** (dentro de Shaper Tool) para eliminar bordes irregulares en trazados vectoriales.
- Siempre crea un **grid de espaciado** usando una forma del logo como unidad base (X, X/2, X/4) antes de finalizar la composición.
- Cuando selecciones colores, investiga primero los competidores y elige un tono opuesto al dominante en el mercado.
- Para tipografía, selecciona siempre dos fuentes: una display (del logo) y una sans-serif legible para cuerpo de texto.
- Para patrones, rellena completamente el cuadrado base sin espacios vacíos antes de aplicar **Object > Pattern > Make**.
- En mockups de Photoshop, usa siempre **Clipping Mask** para ajustar ilustraciones dentro de formas.

## Errores comunes que evita o menciona
- **No curvar manualmente con Pen Tool:** es más lento y menos preciso que usar líneas rectas + Curvature Tool.
- **No dejar bordes irregulares:** usar Smooth Tool para alisar automáticamente en lugar de ajustar punto por punto.
- **No espaciar el logo sin grid:** usar medidas basadas en una forma del logo (X, X/2, X/4) para consistencia.
- **Elegir colores sin investigar competidores:** se recomienda hacer análisis de marcas rivales para destacar.
- **Usar una sola tipografía:** siempre separar display font (para títulos) de body font (para textos largos).
- **Dejar huecos en patrones:** rellenar todo el cuadrado base antes de crear el pattern para evitar espacios vacíos.
