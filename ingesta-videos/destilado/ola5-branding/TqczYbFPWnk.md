## Qué enseña
Enseña cómo aplicar una conversación estratégica para diagnosticar el problema real de un cliente potencial, usando preguntas estructuradas para descubrir el objetivo cuantificable, evaluar el rendimiento pasado y determinar el presupuesto de inversión adecuado. También demuestra el uso del embudo de marketing (funnel) para calcular cuántos leads se necesitan para obtener clientes pagados.

## Lecciones accionables
- Usa las "siete preguntas esenciales" de Daniel Montgomery (del libro *The Coaching Habit*) para guiar la conversación: empieza con "¿Qué tienes en mente?", "¿Dónde deberíamos empezar?", "¿En qué puedo ayudarte?", luego profundiza con "¿Y qué más?" y "¿Cuál es el verdadero desafío aquí para ti?".
- Pregunta "¿Cuál es el resultado más deseable para ti?" y luego hazlo cuantificable: "Si hacemos esto juntos, ¿cómo se vería el éxito? ¿Cuántas inscripciones te gustaría tener?".
- Calcula el valor de un cliente: pregunta "¿Cuánto vale un cliente para ti?" y luego multiplica por el número de clientes deseados para obtener el ingreso total.
- Determina el presupuesto de marketing preguntando "¿Cuánto dinero tiene sentido gastar para obtener esos ingresos?" y verifica la disposición del cliente a invertir (ejemplo: "¿Estarías dispuesto a gastar $5,000 para obtener $5,000 en clientes?").
- Aplica la lógica del embudo: para obtener 5 clientes pagados, necesitas ~200 inscripciones al lead magnet (auditoría gratuita), de las cuales ~8-10 responderán correos, y la mitad se convertirán en clientes.
- Si el cliente dice que su campaña no funciona, revisa el tamaño de la muestra: "El problema es que la muestra es demasiado pequeña". Si 14 inscripciones generaron 2 respuestas, ese es un buen porcentaje (~14%), solo necesitas más volumen.
- Para presupuestar, usa la fórmula: costo de adquisición de cliente (CAC) debe ser menor o igual al valor del cliente; si el cliente vale $1,000, puedes gastar hasta $1,000 para adquirirlo.

## Reglas para agentes
- Siempre empieza la conversación con preguntas abiertas: "¿Qué tienes en mente?", "¿En qué puedo ayudarte?", "¿Cuál es el verdadero desafío aquí?".
- Nunca asumas que el cliente sabe lo que quiere; profundiza preguntando "¿Y qué más?" hasta que el problema real salga a la luz.
- Cuando el cliente presente un problema (ej. "no consigo clientes"), pide un objetivo cuantificable: "¿Cómo se vería el éxito? ¿Cuántos clientes pagados quieres?".
- Calcula el valor del cliente antes de proponer cualquier inversión: pregunta "¿Cuánto vale un cliente para ti?" y luego "¿Cuánto dinero tiene sentido gastar para obtener ese ingreso?".
- No aceptes un presupuesto de marketing sin verificar que el cliente está dispuesto a invertir lo necesario para llenar el embudo: "Necesitas 200 inscripciones, no 50".
- Si el cliente dice que una campaña no funcionó, revisa el porcentaje de conversión y el tamaño de la muestra antes de descartarla: "14 inscripciones y 2 respuestas es un buen porcentaje, solo necesitas más volumen".
- Cuando propongas un proyecto, asigna un precio basado en el valor para el cliente (no en tiempo) y sé claro sobre lo que garantizas: "Puedo garantizar resultados si el precio es lo suficientemente alto para trabajar hasta que funcione".

## Errores comunes que evita o menciona
- **Muestra demasiado pequeña**: el cliente pensó que 50 inscripciones eran suficientes para obtener 5 clientes, pero el experto indica que se necesitan ~200. No juzgues el rendimiento de una campaña con pocos datos.
- **No cuantificar el objetivo**: el cliente dijo "quiero clientes", pero no especificó cuántos. El experto lo obligó a poner un número: "5 clientes pagados".
- **Subestimar el presupuesto necesario**: el cliente gastó solo $100 y esperaba resultados; el experto le mostró que debería estar dispuesto a gastar hasta $5,000 si el valor del cliente lo justifica.
- **Confundir "no funciona" con "bajo volumen"**: el cliente pensó que 2 respuestas de 14 era malo, pero el experto le explicó que es un buen porcentaje (~14%) y que solo necesita más leads al inicio del embudo.
- **No asignar un valor monetario al problema**: el experto evitó proponer una solución sin saber cuánto vale un cliente para el cliente, lo que permite fijar un precio adecuado y alinear expectativas.
