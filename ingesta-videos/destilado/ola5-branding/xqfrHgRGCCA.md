## Qué enseña (2-3 líneas)
Enseña los fundamentos de la dirección creativa: cómo construir briefs creativos efectivos, aplicar estrategias visuales (escala, estética, transporte, choque, tensión) para hacer productos y anuncios más interesantes, y gestionar un flujo de trabajo entre diseñadores, fotógrafos y equipos de marketing usando herramientas como Notion, Pinterest y Figma.

## Lecciones accionables
- **Estrategia de escala:** Haz un objeto muy grande o muy pequeño para captar atención. Ejemplo: Jacamo usa objetos gigantes en redes sociales.
- **Estrategia de estética:** Define una estética de marca (ej: Western Gothic, negro, tipografías específicas) y aplícala a cualquier referencia o idea de campaña.
- **Estrategia de belleza:** Toma una idea existente y hazla lo más estéticamente agradable posible. Ejemplo: el set de fragancias Dreze usa una caja de cartón con impresión de patrón de color para parecer lujoso sin alto costo.
- **Estrategia de transporte:** Coloca un producto o referencia en un entorno completamente ajeno (ej: un desfile en el desierto de Mojave, un auto en un glaciar).
- **Estrategia de choque:** Combina dos ideas o referencias de mood board para crear algo nuevo: toma el ángulo/composición de una foto y el look/ambientación de otra.
- **Estrategia de tensión:** Aplica emociones incómodas (vergüenza, miedo, ira, ego) a cualquier creación. Ejemplo: sistemas de seguridad usan miedo; marcas de lujo usan ego.
- **Brief creativo en Notion:** Incluir título del brief, tipo de creativo, propósito, número de assets, ratios, enlaces al contenido, referencias y cualquier info adicional. Usar carpetas "En progreso" y "Completado/Archivo".
- **Flujo de referencias:** Usar un tablero compartido de Pinterest con el equipo de diseño y fotografía. Guardar referencias de anuncios, ángulos de cámara, estilos visuales. El diseñador puede usar esas referencias para briefs de fotógrafos.
- **Kit de diseño:** Tener un archivo base (ej: en Figma) con fuentes, tamaños, botones, estilos de email, para cambios centralizados.
- **Pruebas de anuncios:** Tomar un reel o idea de otro creador (ej: Dara sobre call out ads) y usarlo como brief para generar variantes de anuncios con diferentes estilos visuales (color range, vintage ad, oversized). Probar y escalar lo que funciona.
- **Documentación de muestras:** Para generar hype antes del lanzamiento, crear carruseles y videos que documenten el proceso de muestras (ej: "Valuable Sample 2025-003 - status: en revisión") con estética de archivo.

## Reglas para agentes
- Usa Notion (o ClickUp, o Monday) para gestionar briefs creativos con campos: título, tipo de creativo, propósito, número de assets, ratios, enlaces, referencias.
- Mantén un tablero compartido de Pinterest (o Are.na) con el equipo para referencias visuales; úsalo tanto para briefs de diseño como para briefs de fotografía.
- Define un kit de diseño base (fuentes, tamaños, botones, estilos) en un archivo central (Figma) y actualízalo en un solo lugar.
- Cuando crees un brief para fotógrafo, incluye referencias de ángulos específicos que el diseñador necesita para componer.
- Siempre que veas un anuncio o reel que funcione, guárdalo como referencia y úsalo como brief para generar variantes.
- Para productos, aplica al menos una estrategia del toolkit (escala, estética, belleza, transporte, choque, tensión) para diferenciarlos.
- Nunca copies una referencia directamente; siempre combina elementos de al menos dos referencias (choque) o aplica tu estética de marca.

## Errores comunes que evita o menciona
- **Copiar referencias sin transformarlas:** No solo tomar una imagen de referencia y replicarla; hay que aplicar el toolkit (escala, estética, choque, etc.) para hacerla original.
- **No tener un sistema de briefs:** No improvisar; usar una herramienta (Notion, ClickUp) con campos estandarizados para que el equipo sepa exactamente qué se necesita.
- **No documentar el proceso de muestras:** No esperar a lanzar un producto para promocionarlo; crear contenido del proceso de muestras (carruseles, videos) para generar hype anticipado.
- **Ignorar la intención en el packaging:** No asumir que el packaging caro requiere materiales costosos; el set Dreze demuestra que una caja de cartón con impresión estratégica puede verse lujosa.
- **No tener un flujo entre diseño y fotografía:** No mantener separados los equipos; usar el mismo tablero de referencias para que el fotógrafo capture los ángulos que el diseñador necesita.
