## Qué enseña (2-3 líneas)
Proceso completo de branding para una marca vegana llamada "Nifty", desde la investigación de audiencia y tendencias hasta la creación de logo, paleta de colores, tipografía, ilustraciones y mock-ups. La creadora muestra cómo definir un público objetivo mediante datos demográficos y cómo traducir esa información en decisiones visuales concretas.

## Lecciones accionables
- Para investigar audiencia: buscar en Statista términos como "vegan demographics UK" y extraer datos sobre rangos de edad predominantes (en este caso, Gen Z y Millennials, menores de 35 años).
- Para tendencias de packaging: usar Dieline (dieline.com) con búsquedas como "vegan food trends" o "plant-based food packaging trends". Identificar categorías: monochromatic packaging, flexible/responsive logos.
- Para mood boards: usar Milanote para centralizar imágenes de Unsplash, Envato Market, Dieline blog, Pinterest y Behance. Ajustar colores de imágenes encontradas si es necesario para que encajen en la paleta.
- Para encontrar imágenes con colores específicos: en Behance usar el Color Picker (seleccionar un color del espectro) y luego buscar con términos relevantes.
- Para sketching: usar Procreate (o papel) con diferentes brushes. Probar layouts explorando ángulos, curvas y formas de llenar espacios vacíos entre letras.
- Para tipografía en Illustrator: probar all caps vs. lowercase vs. mixed case. Experimentar con layouts en arco o ángulo. Usar la herramienta de edición de tipo que permite rotar/escalar letras sin expandir el texto (manteniéndolo como tipo vivo).
- Para el logo: buscar simetría visual (ej: la "i" con punto arriba y la "y" con punto abajo como espejo). Redondear extremos de letras para acabado suave. Ajustar kerning manualmente.
- Para paleta de colores: definir al menos un color primario claro (fondo), un color brillante (acento) y un color oscuro (contraste para footers/detalles). Probar combinaciones específicas (ej: logo blanco sobre colores oscuros, logo negro sobre claros).
- Para ilustraciones: usar Procreate con textured brush. Exportar e integrar en el layout de branding.
- Para tipografía secundaria: limitarse a máximo dos typefaces que contrasten en peso/tamaño, o usar diferentes pesos de una misma familia.
- Para mock-ups: probar el diseño en business cards, web banners, packaging. Verificar contraste y legibilidad en formatos pequeños. Hacer cambios incluso en esta etapa final.

## Reglas para agentes
- Usa Statista para datos demográficos cuando investigues audiencia de un mercado específico (ej: "vegan demographics UK").
- Usa Dieline para identificar tendencias actuales de packaging cuando trabajes con marcas de alimentos.
- Usa el Color Picker de Behance cuando necesites imágenes que coincidan exactamente con una paleta de colores predefinida.
- Nunca combines más de dos typefaces diferentes en un mismo diseño de marca a menos que tengas un nivel de habilidad muy alto.
- Nunca uses combinaciones de colores sin probar primero el contraste del logo (blanco sobre fondo oscuro, negro sobre fondo claro).
- Siempre prueba el diseño en mock-ups reales (business cards, packaging, web) antes de finalizar; si no funciona en formatos pequeños, ajústalo.

## Errores comunes que evita o menciona
- No definir un público objetivo específico: la marca original solo apuntaba a "veganos" sin segmentar, lo que hacía el branding genérico.
- Usar branding genérico con hojas verdes y enfoque en salud que no diferencia el producto en el estante.
- No investigar tendencias actuales del mercado (como monochromatic packaging o responsive logos) antes de diseñar.
- Crear mood boards con demasiados estilos de tipografía o colores diferentes que no funcionan juntos.
- Usar colores que no se ven apetitosos con comida (especialmente carnes veganas cocidas o crudas).
- Forzar un layout en arco cuando las letras tienen ascendentes/descendentes que crean espaciado incómodo (ej: "nifty" en all caps sobre arco).
- Introducir una tercera tipografía que desordena el diseño (ej: añadir serif itálica además de dos typefaces ya definidas).
