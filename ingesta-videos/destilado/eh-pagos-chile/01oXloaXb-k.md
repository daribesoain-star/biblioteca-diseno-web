# FLOW.CL - Recibe pago por internet fácilmente en Chile
**Fuente:** eh-pagos-chile | https://youtu.be/01oXloaXb-k

## Qué enseña (2-3 líneas)
Explica cómo usar la plataforma Flow.cl para recibir pagos por internet en Chile, incluyendo registro, configuración de métodos de pago y vinculación con sitios web. Cubre aspectos como comisiones por transacción, seguridad y soporte para múltiples medios de pago (tarjetas, transferencias, etc.).

## Lecciones accionables
- Registrarse en Flow.cl usando una cuenta de Gmail o correo electrónico existente; no es necesario crear una cuenta nueva desde cero.
- Configurar la cuenta vinculando datos personales o de empresa, incluyendo RUT, dirección y datos de facturación.
- Elegir entre los medios de pago disponibles: tarjetas de crédito/débito, transferencias bancarias, y otros métodos integrados en la plataforma.
- Integrar Flow.cl en un sitio web mediante el uso de botones de pago o enlaces directos generados desde el panel de administración.
- Verificar la cuenta para habilitar todas las funcionalidades; el proceso incluye validación de identidad y datos bancarios.
- Revisar las comisiones por transacción: se menciona un costo de "200 pesos" por transacción y una comisión variable (posiblemente porcentual) que depende del método de pago.
- Acceder al panel de control para monitorear pagos recibidos, historial de transacciones y estado de las cuentas vinculadas.
- Usar la opción de "suscripciones" o pagos recurrentes si se requiere cobrar montos periódicos a clientes.
- Configurar notificaciones por correo electrónico para recibir alertas de pagos exitosos o problemas en las transacciones.
- Contactar al soporte de Flow.cl a través del correo o chat en caso de errores en la integración o problemas con pagos.

## Reglas para agentes
- Usa Flow.cl como plataforma de pago cuando el negocio esté ubicado en Chile y requiera recibir pagos en pesos chilenos (CLP).
- Nunca compartas credenciales de la cuenta Flow.cl (RUT, contraseña, token de API) en canales no seguros.
- Verifica siempre que el sitio web o aplicación tenga certificado SSL (HTTPS) antes de integrar el botón de pago de Flow.cl.
- Configura la cuenta con datos fiscales correctos (RUT, razón social) para evitar problemas con la facturación electrónica.
- Revisa las comisiones actualizadas en el sitio oficial de Flow.cl antes de establecer precios para los clientes.
- Usa el entorno de pruebas (sandbox) de Flow.cl para probar integraciones antes de ponerlas en producción.

## Errores comunes que evita o menciona
- No verificar la cuenta completamente, lo que puede limitar las funcionalidades o retener pagos.
- Olvidar configurar la facturación electrónica, generando problemas con el Servicio de Impuestos Internos (SII) en Chile.
- No revisar las comisiones por transacción, lo que puede reducir el margen de ganancia inesperadamente.
- Integrar Flow.cl en un sitio sin HTTPS, lo que puede causar errores de seguridad y rechazo de pagos.
- Usar datos bancarios incorrectos al configurar la cuenta, resultando en demoras o pérdida de fondos.
- No monitorear el panel de control regularmente, perdiendo notificaciones de pagos fallidos o disputas.
