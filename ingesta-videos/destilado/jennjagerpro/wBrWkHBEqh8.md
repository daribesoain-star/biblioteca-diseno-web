# Best Final Cut Shortcuts
**Fuente:** jennjagerpro | https://youtu.be/wBrWkHBEqh8

## Qué enseña
Una colección de los atajos de teclado más útiles en Final Cut Pro para acelerar el flujo de trabajo, desde navegación entre ventanas hasta edición de audio, títulos, retiming y exportación. Cubre herramientas esenciales para editores de nivel principiante e intermedio.

## Lecciones accionables
- **Shift + Z**: Ajusta todas las ventanas (timeline, canvas, browser) para mostrar el contenido completo. Úsalo cuando estés ampliado y quieras ver la línea de tiempo entera.
- **Cmd + 1**: Abre la ventana del navegador (browser).
- **Cmd + 2**: Lleva el foco a la línea de tiempo (timeline).
- **Cmd + 3**: Lleva el foco al canvas (visor de reproducción).
- **Cmd + T**: Coloca el cursor entre dos clips y presiona para crear un cross dissolve automático.
- **B**: Activa la herramienta de cuchilla (blade tool) para dividir un clip en dos.
- **N**: Activa/desactiva el snapping; permite que el playhead salte entre tomas y que los clips se adhieran a las uniones del storyline principal.
- **Cmd + G**: Agrupa clips seleccionados (ej. dos clips de audio) en una sola unidad que se puede mover junta. Funciona también con clips conectados.
- **Option (Alt) + clic y arrastre**: Duplica un clip en la línea de tiempo.
- **V**: Deshabilita (desactiva) un clip seleccionado para ver lo que hay debajo.
- **T**: Activa la herramienta slip edit; haz clic y arrastra sobre un clip para ajustar el primer y último fotograma visibles en el canvas sin cambiar la duración.
- **L**: Reproduce la línea de tiempo a velocidad normal. Presiona L nuevamente para reproducir a mayor velocidad. Usa K para pausar.
- **J**: Reproduce la línea de tiempo hacia atrás.
- **Cmd + C** en un clip con atributos modificados, luego **Cmd + Shift + V** en el clip destino: abre la ventana "Paste Attributes" donde puedes seleccionar qué atributos copiar (marcar/desmarcar casillas) y pegar.
- **Cmd + Shift + X**: Abre la ventana "Remove Attributes" para eliminar atributos específicos de un clip.
- **Control + V**: Abre el visor de keyframes (keyframe viewer) del clip seleccionado para reposicionar o ajustar keyframes.
- **Cmd + 6**: Abre la paleta de color (color board).
- **Cmd + Shift + 8**: Abre o cierra los medidores de audio (audio meters).
- **Control + +** (tecla más): Sube el volumen del clip seleccionado 1 dB por pulsación.
- **Control + -** (tecla menos): Baja el volumen del clip seleccionado 1 dB por pulsación.
- **Control + T**: Agrega un título básico (basic title) sobre el clip seleccionado.
- **Shift + N**: Con un clip seleccionado, abre la barra verde de retiming para acelerar o ralentizar el clip.
- **Shift + H**: Con el playhead posicionado donde se desea el fotograma congelado, crea un hold frame (fotograma congelado). Arrastra el borde para cambiar su duración.
- **I**: Marca el punto de inicio (in point) de un clip en el navegador.
- **O**: Marca el punto de final (out point) de un clip en el navegador.
- **Q**: Inserta el clip del navegador (con in/out marcados) como clip conectado en la línea de tiempo.
- **W**: Inserta el clip del navegador (con in/out marcados) en el storyline principal.
- **R**: Activa la herramienta de rango (range tool); selecciona una sección de la línea de tiempo para conocer su duración exacta (ej. "3 segundos y 10 fotogramas"), ajustar volumen solo en esa sección con Control + +/- , o exportar solo ese rango.
- **Cmd + E**: Abre la ventana de exportación para exportar el video terminado.

## Reglas para agentes
- Usa **Shift + Z** cuando estés ampliado en cualquier ventana y necesites ver la vista completa de inmediato.
- Usa **Cmd + 1/2/3** para cambiar rápidamente entre navegador, timeline y canvas sin usar el mouse.
- Usa **Option + arrastre** para duplicar clips en lugar de copiar y pegar manualmente.
- Usa **V** para deshabilitar un clip temporalmente y ver contenido subyacente, nunca elimines clips para inspeccionar.
- Usa **Control + +/-** para ajustar volumen en incrementos de 1 dB, nunca arrastres manualmente la línea de volumen para cambios precisos.
- Usa **R** para seleccionar rangos específicos antes de exportar parcialmente; nunca exportes la línea de tiempo completa si solo necesitas una sección.
- Usa **Cmd + Shift + V** para pegar atributos selectivamente; nunca copies atributos sin abrir la ventana de selección.
- Usa **I** y **O** para marcar in/out en el navegador antes de insertar clips con **Q** o **W**; nunca arrastres clips sin marcar.

## Errores comunes que evita o menciona
- No recordar que **Shift + Z** funciona en todas las ventanas (timeline, canvas, browser), no solo en una.
- Olvidar que **L** acelera la reproducción al presionarlo múltiples veces (no solo reproduce normal).
- No usar **N** (snapping) al arrastrar clips, lo que causa desalineaciones en el timeline.
- Ignorar que **T** (slip edit) permite cambiar el contenido visible de un clip sin alterar su duración o posición en el timeline.
- No usar **R** (range tool) para exportar solo una sección, perdiendo tiempo al exportar el proyecto completo.
- Pensar que **V** elimina clips cuando solo los deshabilita temporalmente.
- No usar **Control + V** para ver keyframes, asumiendo que no hay forma rápida de acceder a ellos.
