# FCP iPad App Quickstart
**Fuente:** jennjagerpro | https://youtu.be/-pYunDidZTc

## Qué enseña
Tutorial de inicio rápido para Final Cut Pro en iPad: desde crear un proyecto, importar medios, navegar la interfaz, editar clips en la línea de tiempo, aplicar efectos, transiciones y títulos, hasta exportar el video final.

## Lecciones accionables
- Al abrir la app, presiona el **signo +** en la esquina inferior izquierda para crear un nuevo proyecto.
- En ajustes de línea de tiempo: toca **Automático** y selecciona **Personalizado** para cambiar resolución, orientación (retrato/paisaje), espacio de color (HDR o SDR) y velocidad de fotogramas. **La velocidad de fotogramas no se puede cambiar después de crear el proyecto.**
- Para importar medios: usa el botón de importar en la interfaz, selecciona **Fotos** o **Archivos**. Conecta un lector de tarjetas o disco duro al iPad para importar desde dispositivos externos. La app copia todo el medio al iPad.
- Para marcar entrada/salida en un clip: selecciona el clip, usa los **mangos amarillos** en la vista alargada, o presiona **Play** y luego los botones **In** y **Out**.
- Para añadir clips a la línea de tiempo: mantén presionado y arrastra con el dedo, o usa el menú con opciones:
  - **Override**: reemplaza el clip en la posición del cabezal.
  - **Insert at playhead**: divide el clip existente e inserta el nuevo entre las mitades.
  - **Connect at playhead**: coloca el nuevo clip encima del existente (se mueven juntos).
  - **Append to storyline**: añade el clip después del actual sin modificar su duración.
- Para hacer zoom en la línea de tiempo: usa el gesto multitáctil de **dos dedos** (pellizcar/separar).
- Para mover clips libremente: activa el **Position tool** (icono de flecha) para crear **gap clips** (espacios negros). Para eliminar un gap clip: selecciónalo, desactiva Position tool, presiona el **tacho de basura**.
- Para activar/desactivar **Snapping** (ajuste magnético): usa el icono junto a Position tool. Actívalo para que el cabezal salte a los cortes entre clips.
- Para seleccionar múltiples clips: mantén presionado en el espacio oscuro sobre los clips y arrastra con **lazo** alrededor de ellos.
- Para usar el **Jog Wheel**: toca el icono en la esquina superior derecha, luego el círculo negro para abrirlo. Con Position tool activado, selecciona un clip y usa la **flecha doble** para recortar fotograma a fotograma.
- Para ajustar volumen: selecciona un clip, presiona el botón de **volumen** y desliza hacia abajo. Los medidores de audio están a la derecha de la pantalla.
- Para abrir el **Inspector**: presiona el botón **Inspect** (esquina inferior izquierda). Contiene pestañas:
  - **Format**: ajusta velocidad (cámara lenta/rápida), opacidad y conversión de color (HDR/SDR automática).
  - **Transform**: escala (punch in), rotación (grados o slider), recorte (crop) y distorsión (skew).
  - **Audio**: volumen y mejoras como **Voice Isolation** (IA para limpiar ruido de fondo).
  - **Video Effects**: añade efectos especiales. El más común: **Color Adjustment** (corrección de color básica con sliders).
- Para aplicar un efecto: selecciona el efecto en el navegador, selecciona el clip en la línea de tiempo, presiona **Apply**.
- Para añadir transiciones: abre el navegador de efectos, ve a **Transitions**, previsualiza con **Play**, selecciona un clip y usa el menú para añadir al **Start**, **End** o **Both**.
- Para añadir títulos: abre el navegador de **Titles**, previsualiza, arrastra y suelta en la línea de tiempo. Para modificar texto: selecciona el título, abre Inspector, pestaña **Format**, selecciona la capa de texto y cambia el texto, fuente, posición, color.
- Para añadir música: en el navegador, selecciona **Soundtracks**, previsualiza, usa el menú **Connect at playhead** para colocarlo debajo de los clips. Para recortar la música: cambia el menú a **Edge**, selecciona el clip de música, usa el mango grande al final para recortar al final del video.
- Para exportar: presiona el botón **Share** (centro superior). Opciones: **Video** (exportar completo), **Audio** (solo audio), **Frame** (fotograma actual), **Final Cut Pro for iPad project** (enviar a FCP en Mac). En video, elige códec, resolución, tipo de archivo, espacio de color. Presiona **Export** y selecciona destino (ej. AirDrop).

## Reglas para agentes
- Usa **Automático** en ajustes de línea de tiempo solo si el primer clip define las propiedades; usa **Personalizado** si necesitas control específico.
- Nunca cambies la **velocidad de fotogramas** después de crear el proyecto; es el único parámetro inmutable.
- Activa **Snapping** siempre para facilitar la alineación precisa del cabezal en los cortes entre clips.
- Usa **Position tool** solo cuando necesites mover clips libremente y crear gap clips; desactívalo para edición magnética normal.
- Para eliminar un gap clip, desactiva primero el **Position tool** antes de presionar el tacho de basura.
- Aplica **Voice Isolation** en la pestaña de audio del Inspector cuando haya diálogo con ruido de fondo.
- Usa **Color Conversion** automática en la pestaña Format del Inspector cuando mezcles clips HDR y SDR en la misma línea de tiempo.
- Para recortar música al final del video, cambia el menú a **Edge** y usa el mango grande del clip de audio.

## Errores comunes que evita o menciona
- **No poder cambiar la velocidad de fotogramas después de crear el proyecto**: se debe definir antes de continuar.
- **La app copia todo el medio al iPad**: no se puede desconectar el disco duro hasta que la importación termine, pero una vez hecho, el medio vive en el iPad.
- **No hay formato cuadrado disponible** en las opciones de resolución al crear el proyecto.
- **Los gap clips (espacios negros) no se eliminan con el tacho si el Position tool está activado**: hay que desactivarlo primero.
- **Si el Position tool está desactivado, los clips se "magnetizan" y no se pueden mover libremente**: se debe activar para crear espacios.
