# Best Apple Motion Features
**Fuente:** jennjagerpro | https://youtu.be/XtdCdzVwMEI

## Qué enseña (2-3 líneas)
Este video recorre las funciones favoritas de la creadora en Apple Motion, cubriendo efectos de texto, comportamientos, filtros, generadores, herramientas de canvas y línea de tiempo. Enseña a personalizar texto 3D, usar máscaras de imagen, replicadores, animaciones con behaviors y ajustes de cámara para dar profundidad a proyectos.

## Lecciones accionables
- **Texto metálico con color personalizado:** Cambia texto a 3D → Materiales → Metal → Desplegable "aluminum" → "Custom" → Ajusta color (ej. rosa), reduce "Shininess" para efecto rose gold.
- **Transform Glyph en texto:** Selecciona texto → Menú cursor → "Transform Glyph" → Modifica caracteres individualmente (color, fuente, etc.) sin afectar otros.
- **Texto en trayectoria circular:** Inspector → Layout → Desplegable "Type" → "Path" → Path Options → "Circle" → Ajusta "Radius" y "Path Offset"; opción de texto dentro del círculo.
- **Behavior "Type On":** Behaviors → Text Animation → "Type On" → Ajusta velocidad reduciendo barra púrpura en timeline.
- **Behavior "Drift In":** Behaviors → Text Basic → "Drift In" → Activa overlays → Usa flechas para dirección → Inspector → Format → Sube "Opacity" para eliminar fade y obtener efecto "pop on and lift up".
- **Behavior "Flip In":** Behaviors → Energetic → "Flip In" → En timeline selecciona behavior → Inspector → Control → Cambia "Direction" de "Random" a "Forwards".
- **Behavior "Arrange In":** Behaviors → Text Basic → "Arrange In" → Inspector → Control → Cambia "Animate" de "Character" a "Word" → Acelera reduciendo barra.
- **Behavior "Substitute In":** Behaviors → Text Basic → "Substitute In" → Letras aparecen aleatorizadas hasta formar palabra final.
- **Behavior "Glow Through":** Behaviors → Text Highlighter → "Glow Through" → Mueve barra en timeline → Inspector → Text tab → Appearance → Glow → Cambia color → Behaviors → Controls → Cambia "Direction" (backward, center to edges) → Ajusta "Blur", "Radius", "Scale".
- **Behavior "Scale Through":** Behaviors → Text Highlighter → "Scale Through" → Extiende barra púrpura para ralentizar.
- **Generator "Cellular":** Library → Generators → "Cellular" → Úsalo para desintegrar texto u objetos (requiere tutorial dedicado).
- **Generator "Color Solid":** Library → Generators → "Color Solid" → Base para proyectos; cambia color por defecto (azul) a blanco u otro.
- **Generator "Gradient":** Library → Generators → "Gradient" → Inspector → Desplegable para colores → Ajusta "Start" y "End Points" para rotar, expandir y suavizar transición.
- **Generator "Numbers":** Library → Generators → "Numbers" → Inspector → "Start Count" (ej. 10), "End Count" (ej. 1,000,000) → Acelera reduciendo elemento en timeline → Opciones: "Decimals", "Minimum Digits", "Format" (Number, Currency, Percent), "Spell Out".
- **Generator "Two Color Ray":** Library → Generators → "Two Color Ray" → Cambia colores → Ajusta "Waviness" y "Offset" del centro.
- **Particle Emitter "Snow Flurry":** Library → Particle Emitters → "Snow Flurry" → Inspector → Modifica cantidad, forma, color de blurs.
- **Cámara 3D:** Añade cámara (Object → New Camera) para dar profundidad y vida a proyectos 2D.
- **Replicator:** Selecciona objeto → Botón "Replicate" (top right) → Inspector → Ajusta "Shape", "Columns", "Rows", "Scale", "Offset", "Angle", "Colorize over Pattern".
- **Image Mask:** Selecciona capa (ej. gradient) → Object → "Add Image Mask" → Arrastra texto u objeto al drop zone → Activa "Invert" para recortar forma.
- **Task List (Export):** F9 o Window → "Show Task List" → Barra de progreso para exportaciones.
- **Behavior "Motion Path":** Selecciona objeto → Behaviors → Basic Motion → "Motion Path" → Arrastra puntos start/end en canvas → Opción + clic en línea roja para añadir puntos → Reduce barra en timeline para acelerar.
- **Behavior "Fade In/Fade Out":** Behaviors → Basic Motion → "Fade In/Fade Out" → Inspector → Ajusta duración (default 20 frames).
- **Behavior "Overshoot":** Behaviors → Parameter → "Overshoot" → Reduce barra → Inspector → "Apply To" → "Rotation Z" → Ajusta "Start Value" y "Ramp Duration" para bounce → Aplica también a scale, opacity, filters.
- **Behavior "Randomize":** Behaviors → Parameter → "Randomize" → Aplica a cualquier propiedad del objeto.
- **Behavior "Write On":** Behaviors → Shape → "Write On" → Dibuja formas creadas en Motion → Acelera reduciendo barra.
- **Filter "Gaussian Blur":** Filters → "Gaussian Blur" → Default 4 → Dial up a 64 → Arrastra número con mouse hacia adelante para valores mayores.
- **Filter "Flop":** Filters → Distortion → "Flop" → Invierte imagen horizontalmente.
- **Filter "Disk Warp":** Filters → Distortion → "Disk Warp" → Ajusta "Radius", "Position", "Center" para distorsión circular.
- **Filter "Colorize":** Filters → Color → "Colorize" → Inspector → Cambia "Blacks" y "Whites" a un solo color (ej. negro) para unificar logo multicolor.
- **Filter "Twirl":** Filters → Distortion → "Twirl" → Efecto de espiral.
- **Shape Tool:** Centro de pantalla → Desplegable → "Rectangle" → Inspector → "Fill" (color o gradient) → Desactiva "Outline" → Geometry → "Roundness" → Convierte a puntos para skew.
- **Bézier (BZA) Tool:** Junto a Shape Tool → Dibuja en canvas → Clic en project pane para fijar → Inspector → Geometry → "Roundness" → Opción + clic en línea para añadir puntos.
- **Anchor Point:** Menú cursor → "Anchor Point" → Icono con flechas rojas/verdes y punto azul → Mueve con flechas → Rota en Y para ver efecto de bisagra desde nuevo punto.
- **Canvas - Grid:** View → "Grid" → Alinea elementos usando cuadrícula para espaciado uniforme.
- **Canvas - Dynamic Guides:** View → "Dynamic Guides" → Líneas amarillas al alinear/centrar elementos.
- **Canvas - 3D View:** En proyecto 3D con cámara → Icono inferior izquierdo → Selecciona vista (Right, Left, etc.) → Para volver: ícono central → Desplegable "Perspective" → "Active Camera".
- **Canvas - Safe Zones:** View → "Safe Zones" → Línea azul interna: texto seguro; línea externa: gráficos seguros (para TV).
- **Timeline - Split Clip:** Selecciona clip → Edit → "Split" → Corta en dos; deselecciona y modifica escala en segunda mitad.
- **Timeline - Snapping:** Activa "Snapping" para alinear cursor entre elementos.
- **Keyframe Editor:** Botón de tres diamantes (top right timeline) → Selecciona elemento → Right-click keyframe → "Interpolation" → "Bézier" → Arrastra handles para acelerar/decelerar movimiento.

## Reglas para agentes
- Usa "Transform Glyph" cuando necesites personalizar caracteres individualmente sin afectar el resto del texto.
- Usa "Image Mask" con "Invert" cuando quieras recortar una forma (ej. texto) de un fondo o gradiente.
- Usa "Overshoot" behavior con "Rotation Z" cuando quieras añadir bounce a animaciones de objetos.
- Usa "Colorize" filter cuando necesites unificar un logo multicolor a un solo tono para compositing.
- Usa "Safe Zones" (View → Safe Zones) cuando el proyecto pueda verse en televisores; coloca texto dentro de la línea azul interna y gráficos dentro de la línea azul externa.
- Usa "Snapping" en timeline para alinear el cursor de reproducción exactamente entre dos clips o elementos.
- Usa el Keyframe Editor con interpolación Bézier cuando necesites control preciso de aceleración/desaceleración en animaciones.
- Nunca uses "Flop" filter si hay texto visible en pantalla, porque invertirá la imagen y el texto se leerá al revés.
- Nunca olvides deseleccionar otros clips después de un "Split" antes de modificar uno solo, para no afectar múltiples elementos.

## Errores comunes que evita o menciona
- No menciona errores explícitos, pero evita: dejar el "Gaussian Blur" en valor 4 (imperceptible); no ajustar la duración de behaviors (barra púrpura) para controlar velocidad; no usar "Invert" en Image Mask cuando se quiere recortar la forma en lugar de rellenarla; no activar "Snapping" para alinear cortes precisos en timeline; no usar el Keyframe Editor para animaciones complejas (es superior a Final Cut).
