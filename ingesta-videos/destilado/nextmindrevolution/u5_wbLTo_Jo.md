# I Tried the Pollo AI Agent: Is This the END of Manual Video Creation?
**Fuente:** nextmindrevolution | https://youtu.be/u5_wbLTo_Jo

## Qué enseña (2-3 líneas)
Muestra el nuevo "Agent Mode" de Pollo AI, que permite crear videos clonando el estilo de un video de referencia, generando anuncios UGC desde enlaces de productos (Amazon), o produciendo videos narrativos/documentales. El agente conversa paso a paso, pide confirmaciones y ofrece variaciones automáticas, todo en 3-4 interacciones.

## Lecciones accionables
- Para clonar un video: selecciona "Clone" en el menú desplegable del Agent Mode, pega el link del video de referencia y sube un archivo de producto. Prompt usado: *"Create a similar video to [link]. Use the attached image reference as product."*
- Para crear un anuncio desde Amazon: selecciona la opción de ad, pega el link del producto. Prompt usado: *"Use this link [Amazon link]. Extract the info, the images, and videos and create the video for us."*
- Para video narrativo: selecciona "Story", escribe el prompt. Ejemplo usado: *"Create a documentary video about the invention of the light bulb."* Luego elige settings: ratio 16:9, duración 30 segundos, idioma English, template "History".
- Durante la generación, el agente propone dos variaciones automáticas. Para elegir una, escribe el número (ej: "2") y luego "continue".
- Para modificar el guion/narración durante el proceso, puedes escribir instrucciones conversacionales como *"be more dramatic"* y el agente ajusta.
- En la creación de anuncios, puedes seleccionar múltiples variaciones (hasta 3) marcando varias opciones antes del paso final.
- Parámetros configurables: duración (hasta 30s), ratio de aspecto (16:9, etc.), formato (general, ads platform), idioma, y template temático (history, etc.).

## Reglas para agentes
- Usa "Clone" cuando quieras replicar el estilo visual y narrativo de un video existente proporcionando su URL.
- Siempre confirma cada paso que el agente propone (duración, plot, estilo, narración) antes de continuar.
- Cuando el agente pregunte "how long is the video?", responde con un número en segundos (ej: "10 seconds").
- Si no te gusta un shot o parte del plot, escribe "I don't like shot number [X]" o "I want this change" para que el agente rehaga la propuesta.
- Nunca asumas que el agente puede hacer todo: si pides algo que el modelo no puede ejecutar, te lo notificará automáticamente.
- Para obtener variaciones, selecciona hasta 3 opciones en la pantalla de preview antes del paso final.

## Errores comunes que evita o menciona
- El creador advierte que el producto está en beta pública, por lo que pueden ocurrir glitches ocasionales.
- Señala que el volumen de la música puede ser demasiado alto y no se puede ajustar manualmente (aunque están trabajando en ello).
- Menciona que actualmente no se puede subir un avatar propio, pero que será posible en una actualización próxima.
- El agente puede fallar si el prompt pide algo fuera de sus capacidades, pero lo notifica explícitamente.
- No esperes que el resultado sea tan bueno como el video de referencia original, pero sí "pretty damn good" para ser generado en 3 pasos.
