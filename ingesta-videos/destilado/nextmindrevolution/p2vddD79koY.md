# Kling AI Text-to-Image Tutorial: Step-by-Step Guide
**Fuente:** nextmindrevolution | https://youtu.be/p2vddD79koY

## Qué enseña (2-3 líneas)
Este tutorial explica cómo usar la herramienta predeterminada de texto a imagen en Kling AI (versión 2.1), cubriendo desde la generación básica hasta el uso de DeepSeek para mejorar prompts, estilos predefinidos, variaciones y opciones de edición como upscale, inpaint, expand y remove. También muestra cómo usar referencias de imagen para refinar resultados.

## Lecciones accionables
- Para generar una imagen, ve a la pestaña "Image" en la página predeterminada de Kling y selecciona "Text to Image". Escribe un prompt conversacional, por ejemplo: "a woman walking in the forest".
- Ajusta la relación de aspecto desde las opciones predeterminadas: 9:16, 16:9 u otras intermedias. Selecciona el número de outputs (ej. 1) y la resolución: "standard definition" o "high resolution". Cada cambio se refleja en el costo.
- Elige el modelo en la parte superior: usa "2.1" (el más reciente) para mejores resultados; "1.0" tiene menos opciones y menor precio.
- Para mejorar un prompt simple, usa la integración con DeepSeek: haz clic en "DeepSeek" para que genere tres sugerencias automáticas. Si el prompt es muy simple (ej. "a woman walking in the forest"), las sugerencias serán básicas; haz clic en "Regenerate" en la parte inferior para obtener nuevas opciones.
- Para personalizar las sugerencias de DeepSeek, haz clic en "Edit", pega un prompt más detallado (ej. "1980s woman walking through a stylish European street, sunny day, low angle shot, cinematic warm light") y presiona la flecha hacia arriba para procesarlo y obtener tres nuevas sugerencias.
- Sube una imagen de referencia en DeepSeek: haz clic en "Upload" o selecciona una del historial. DeepSeek usará la imagen como refuerzo, pero no captura detalles específicos del sujeto; para personajes consistentes, usa "Image Reference" o "Omni One", no "Text to Image".
- Para aplicar un estilo predefinido, haz clic en el ícono de "tiles" (junto al prompt). Se abrirá una galería de estilos; filtra por tipo y selecciona uno. El estilo se agrega al prompt como un menú desplegable; puedes cambiarlo desde ahí.
- Después de generar, pasa el cursor sobre la imagen para ver opciones: "Upscale" (crea una versión de mayor resolución), "Generate Video" (envía la imagen al generador de video), "Vary" (crea dos variaciones sutiles de la imagen, manteniendo la composición general), "Inpaint", "Expand" y "Remove" (para editar la imagen).
- Para copiar un prompt de una generación anterior, pasa el cursor sobre la imagen, haz clic en el ícono de copiar y pégalo en el campo de prompt.
- Usa "Favorites" (ícono de corazón) para guardar imágenes; luego accede a ellas desde la pestaña "Favorites" en la parte superior.
- Los "Hints" en la parte inferior son prompts aleatorios de inspiración; al seleccionarlos, reemplazan tu prompt actual.

## Reglas para agentes
- Usa el modelo "2.1" cuando generes imágenes para obtener la máxima calidad y opciones; nunca uses "1.0" a menos que el costo sea una prioridad explícita.
- Cuando uses DeepSeek para mejorar un prompt, proporciona siempre un prompt base con detalles de contexto, ángulo de cámara, estilo y hora del día (ej. "sunny day, low angle shot, cinematic warm light") para obtener sugerencias más ricas.
- Nunca uses "Text to Image" con referencia de imagen si necesitas personajes consistentes; en su lugar, usa "Image Reference" u "Omni One".
- Siempre verifica que el costo refleje los cambios en configuración (relación de aspecto, número de outputs, resolución) antes de hacer clic en "Generate".
- Para variaciones, usa "Vary" solo cuando quieras cambios sutiles en pose o fondo; nunca lo uses para transformaciones drásticas.

## Errores comunes que evita o menciona
- Evita prompts demasiado simples (ej. "a woman walking in the forest") porque DeepSeek generará sugerencias igualmente simples; siempre añade detalles como contexto, estilo y ángulo de cámara.
- No asumas que DeepSeek con referencia de imagen capturará detalles específicos del sujeto; solo lo usa como refuerzo general, no para consistencia de personajes.
- No confundas "Hints" con sugerencias de mejora; son prompts aleatorios de inspiración que reemplazan tu prompt, no lo complementan.
- No olvides que "Vary" crea variaciones muy sutiles; si buscas cambios mayores, usa "Edit" o "Inpaint".
