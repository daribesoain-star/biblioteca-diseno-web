# GPT Image 2 Character Sheets: Mastering Layouts & Consistency
**Fuente:** nextmindrevolution | https://youtu.be/d7EOD-rdDNw

## Qué enseña (2-3 líneas)
Enseña un flujo para crear character sheets consistentes con GPT Image 2 usando una imagen base del personaje, desde hojas técnicas complejas hasta layouts visuales más simples o grillas estándar. También muestra cómo reutilizar esas hojas como referencia para generar nuevas imágenes y videos manteniendo rostro, vestuario y actitud.

## Lecciones accionables
- Usa una imagen base del personaje como referencia inicial; el presentador arrastra y suelta una imagen de una mujer vikinga en Magnific/Magnifique usando el modelo GPT Image 2.
- Herramientas mencionadas: Magnific/Magnifique, ChatGPT, GPT Image 2, Chat Image 2.0, Kling 3.0 Omni, Seedance 2.0, Seedance 2.0 Fast.
- El prompt complejo original viene de Nick Sora en X; la transcripción no incluye el prompt completo verbatim, pero sí indica su estructura: perfil psicológico, guardarropa, callouts de objetos específicos, shots, patrones de comportamiento, línea emocional base y descripción del personaje.
- Completa como mínimo los campos superiores del prompt: nombre, identidad del personaje, alias, edad y altura. Ejemplo usado: nombre “Mika”, alias “Vicky”, edad “30”, altura “170” cm.
- Para una hoja técnica muy detallada, configura formato `4:3`; también se menciona `16:9` como alternativa.
- Para character sheets con mucho texto y detalle, usa `4K` y calidad `high`; el presentador remarca que el costo sube, pero mejora la lectura de texto y los detalles al hacer zoom.
- Si el costo importa, usa calidad `medium`: según el video, baja mucho el precio y sigue dando resultados superiores a opciones más bajas.
- Revisa la imagen generada haciendo zoom: valida rostro, cuerpo, accesorios, texto, descripciones y consistencia general.
- GPT Image 2 maneja bien texto dentro de la imagen, pero aun así hay que leerlo y revisar que tenga sentido.
- Si no defines manualmente detalles como comportamiento o línea emocional, GPT Image 2 los infiere desde la imagen de referencia.
- Para alejarte de la interpretación automática del modelo, añade manualmente los detalles del personaje dentro del prompt.
- Para más precisión, adjunta múltiples imágenes de referencia del mismo personaje.
- Si la primera hoja sale casi correcta, súbela como referencia y pide a ChatGPT o Chat Image 2.0 que haga correcciones específicas.
- Para reducir inconsistencias, usa una versión recortada del prompt que se enfoque solo en lo visual y elimine descripciones psicológicas o textuales excesivas.
- Para un layout clásico, usa un prompt simple de grilla con diferentes tomas del personaje; la transcripción no incluye el texto exacto del prompt.
- Después de crear la hoja, úsala como referencia para nuevas imágenes: borra la referencia original, sube la character sheet y genera poses nuevas.
- Ejemplos de imágenes posteriores: una toma dinámica y otra donde el personaje aparece sonriendo, riendo o divirtiéndose, para probar consistencia fuera de la expresión seria original.
- Para imágenes derivadas, el presentador usa `4K` con calidad `medium` para bajar costo.
- Para video con Kling 3.0 Omni: sube la character sheet como media/reference, usa `multi-shot`, duración aproximada `10-12` segundos, relación `16:9`, sonido activado.
- Shots usados en Kling: `extreme wide shot`, `low angle shot`, `medium profile shot`, `close-up shot`, `high-angle wide shot`.
- Costo citado para Kling 3.0 Omni: alrededor de `1,380` créditos en Magnific para una secuencia de unos `12` segundos en `16:9`.
- Para Seedance 2.0: sube la primera hoja como referencia, usa un prompt similar al de Kling pero mejor formateado para Seedance, configura `720`, `16:9`, sonido activado y duración `11` segundos.
- Costos citados para Seedance: `3,000` créditos en `720p`; `7,000` créditos en `1080p`.
- Seedance 2.0 da mejor consistencia de rostro, facciones y vestuario que Kling 3.0 Omni, pero es mucho más caro.
- Kling puede verse más detallado si se genera a mayor resolución que Seedance, aunque Seedance conserva mejor la identidad del personaje.
- Seedance puede rechazar ciertas referencias, especialmente imágenes de personas reales.

## Reglas para agentes
- Usa una imagen base clara cuando necesites construir una character sheet consistente.
- Usa `4:3` cuando el objetivo sea una hoja técnica o grilla de personaje.
- Usa `16:9` cuando el objetivo sea video o una composición horizontal.
- Usa `4K high` cuando la hoja incluya mucho texto, accesorios o detalles pequeños que deban leerse al hacer zoom.
- Usa `medium` cuando el costo sea importante y la salida no requiera máxima nitidez.
- Rellena siempre los campos básicos del personaje antes de generar: nombre, identidad, alias, edad y altura.
- Define manualmente psicología, comportamiento, emociones, vestuario y objetos si no quieres que el modelo los invente desde la referencia.
- Adjunta varias referencias cuando la consistencia facial o de vestuario sea crítica.
- Inspecciona cada generación con zoom antes de reutilizarla como referencia.
- Reutiliza la mejor character sheet como referencia para poses, expresiones, escenas y videos posteriores.
- Simplifica el prompt cuando la hoja salga demasiado saturada o con demasiadas inconsistencias.
- Usa grillas simples cuando solo necesites vistas visuales del personaje y no una ficha técnica completa.
- Usa `multi-shot` en Kling cuando quieras una secuencia con varios encuadres.
- Usa Seedance 2.0 cuando la prioridad sea máxima consistencia del personaje en video.
- Usa Kling 3.0 Omni cuando necesites una alternativa más barata que Seedance.
- Nunca aceptes una character sheet sin revisar texto, rostro, accesorios y coherencia de cuerpo.
- Nunca confíes en detalles psicológicos generados automáticamente si el personaje ya tiene una ficha definida.
- Nunca uses Seedance como primera opción si el presupuesto de créditos es limitado.

## Errores comunes que evita o menciona
- Crear una hoja demasiado compleja cuando una grilla visual simple sería suficiente.
- No revisar inconsistencias pequeñas en rostro, cuerpo, accesorios o texto.
- Dejar que GPT Image 2 invente comportamiento, emociones o perfil psicológico sin guía manual.
- Usar una sola referencia cuando se necesita alta precisión.
- Generar en baja calidad una hoja con mucho texto y luego perder legibilidad.
- Elegir Seedance sin considerar que puede costar mucho más que Kling.
- Usar Seedance con referencias problemáticas de personas reales y ser rechazado por la herramienta.
- Comparar calidad visual sin considerar resolución: Seedance puede ser más consistente, pero si se genera en `720p` puede verse menos detallado que Kling en mayor resolución.
