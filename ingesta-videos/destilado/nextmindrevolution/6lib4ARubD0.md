# Stop Using Generic Prompts: Master These 5 AI Styles based on tone
**Fuente:** nextmindrevolution | https://youtu.be/6lib4ARubD0
## Qué enseña (2-3 líneas)
Enseña a controlar el tono visual de imágenes y videos generados con IA usando keywords de estilo precisas, especialmente relacionadas con color, contraste, saturación, sombras y luces. El ejemplo se hace en Google Flow con Nano Banana Pro, pero el método aplica también a Hicksfield, Freepik, Polo AI u otros generadores.

## Lecciones accionables
- Usa `duo tone` cuando quieras reemplazar la escala de grises por dos tonos específicos: uno para sombras/medios tonos y otro para luces. Prompt verbatim: `duo tone blue and orange`.
- En `duo tone`, siempre especifica los colores exactos. Ejemplo verbalizado: `duo tone photography forest green mid tones and pale cream highlights`. La clave técnica es definir qué color afecta los `mid tones` y cuál afecta los `highlights`.
- Para escenas naturales en duotono, combina: tipo de toma + sujeto + keyword + colores + atmósfera + iluminación + estilo. Ejemplo: toma cenital/drone de una carretera de montaña en un bosque de pinos + `duo tone photography` + `forest green mid tones` + `pale cream highlights`.
- En cine, la versión menos agresiva de `duo tone` se menciona como `color washing`: sirve para separar historias, estados o mundos mediante tonos dominantes. Referencias: `Traffic` usa tonos distintos por línea narrativa; `The Matrix` usa verde para el mundo de la Matrix.
- Usa `box camera` para forzar una estética fotográfica antigua y específica, más precisa que pedir solo `1900s style`. Prompt verbatim: `a family portrait shot with a box camera`.
- `box camera` funciona para retratos antiguos, estética vintage, historias de época o piezas con vibra old school. También puede mezclarse con elementos modernos, aunque el autor advierte que puede verse raro.
- Prompt verbalizado para mezcla moderno/vintage: `a model in a futuristic techwear on a glass balcony` + `box camera photography style` + `monochrome` + `low angle type of shot`.
- Usa `iridescent` para una estética surreal, etérea, con reflejos tipo rosa, azul, verde y amarillo. Prompt verbatim: `a man in a room iridescent`.
- `iridescent` destaca especialmente sobre piel, maquillaje y modelos. Prompt verbalizado: `a fashion shot of a full body in a studio of a model` + `shimmering iridescent skin makeup` + iluminación de estudio con glow.
- Referencias cinematográficas para `iridescent`: `The Neon Demon` y `Annihilation`, por su uso de brillo, color cambiante y atmósfera surreal.
- Usa `muted colors` para lograr realismo, baja saturación, contraste suave y tonos apagados. Prompt verbatim: `a man walking muted colors`.
- Para controlar colores apagados, usa también `desaturated` y `muted color grading`. Ejemplo verbalizado: `cinematic shot of a model` + vestido con color apagado + campo al amanecer + `desaturated olive and gray tones` + `muted color grading` + atmósfera.
- Referencias para `muted colors`: `The Witch` por tonos apagados de horror rural; `Inside Llewyn Davis` por una estética general de color muted.
- Usa la estética griega/Aegean cuando quieras una imagen feliz, limpia, de alto contraste y alta saturación: blanco profundo, cielo azul, mar azul intenso y `cobalt blue`.
- Para arquitectura griega, combina `minimalist`, arquitectura, paredes de piedra/blancas, `cobalt blue door`, hora del día y sombras. La puerta azul cobalto es un marcador clave del estilo.
- Para estilo turístico o comercial, usa `real candid Aegean lifestyle shot`, pareja, desayuno cerca del mar, luz natural y entorno costero. Sirve para campañas de viaje o comerciales con vibra mediterránea.
- Referencia cinematográfica para la estética griega/Aegean: `Glass Onion: Knives Out`, especialmente las escenas costeras luminosas antes del giro narrativo.

## Reglas para agentes
- Usa `duo tone` cuando el usuario pida una imagen con dos colores dominantes, y exige siempre dos colores explícitos.
- Define sombras, medios tonos y luces cuando uses `duo tone`; no dejes que el modelo elija la distribución cromática.
- Usa `color washing` como referencia cinematográfica cuando el objetivo sea mood narrativo o separación visual entre mundos, historias o estados.
- Usa `box camera` cuando el usuario pida fotografía antigua, retrato vintage o estética de cámara clásica; no uses solo `1900s` si necesitas precisión.
- Usa `iridescent` cuando el objetivo sea brillo cambiante, piel luminosa, maquillaje editorial, atmósfera surreal o colores rosa/azul/verde.
- Usa `muted colors`, `desaturated` y `muted color grading` cuando el usuario pida realismo sobrio, baja saturación, horror moderno, drama indie o contraste suave.
- Usa `cobalt blue`, blanco intenso, cielo azul y mar azul profundo cuando construyas estética griega/Aegean.
- Nunca pidas “estilo bonito”, “cinematic” o “vintage” sin keywords técnicas de color, cámara, saturación o iluminación.
- Nunca mezcles estilos modernos y vintage extremos, como techwear futurista con `box camera`, sin advertir que puede generar una imagen visualmente incoherente.
- Cuando el estilo dependa del color, especifica color exacto, zona tonal afectada, tipo de luz, atmósfera y referencia visual.

## Errores comunes que evita o menciona
- Usar prompts genéricos sin keywords de estilo precisas.
- Pedir `duo tone` sin indicar los dos colores.
- Elegir combinaciones de color extrañas en duotono, lo que produce resultados poco estéticos.
- Pedir `1900s style` en vez de `box camera`, porque `1900s` puede significar demasiadas cosas.
- Mezclar objetos ultramodernos con estética vintage realista sin una intención clara.
- Confundir blanco y negro, duotono e iridiscencia: cada uno controla el tono de forma distinta.
- No definir qué parte de la imagen debe estar apagada cuando se usan `muted colors`.
- Romper la estética griega usando colores fuera del sistema visual, por ejemplo una puerta roja o rosa en lugar de `cobalt blue`.
