## Lecciones accionables
- Usa la keyword exacta `YouTube thumbnail` para activar el estilo tﾃｭpico de miniatura: texto grande, alto contraste, fondo desenfocado y composiciﾃｳn llamativa.

- Configuraciﾃｳn usada repetidamente: `16:9` / transcrito como `69`, calidad `4K`; en un caso tambiﾃｩn usa `medium`.

- Para replicar una estﾃｩtica difﾃｭcil de describir, sube una miniatura a `Gemini` o `ChatGPT` y pide que genere un prompt textual que describa la imagen.

- Para agregar persona o avatar, sube la referencia como `image number one` o como `character`; la calidad de la imagen de referencia afecta directamente el resultado.
- Prompt de sujeto mencionado: `Create a YouTube thumbnail. A man like in [IMG 1] is pointing with a determined expression at the title GPT-2. It is awesome.`

- Prompt de ediciﾃｳn mencionado: `replace the man on IMG 1 with the man on IMG 2. He is holding a tablet instead of a laptop and remove the text and arrow from IMG 1.`

## Reglas para agentes
- Usa `YouTube thumbnail` cuando el objetivo sea una miniatura de YouTube con texto grande, contraste alto y composiciﾃｳn viral.
- Usa Gemini o ChatGPT para convertir una referencia visual en prompt cuando el usuario no pueda describir el estilo con precisiﾃｳn.
- Usa `thinking mode` en Gemini cuando necesites mejor lectura de una imagen de referencia.
- Reemplaza manualmente los textos clave del prompt generado antes de ejecutar la generaciﾃｳn.
- Usa `IMG 1 as a style reference` cuando quieras conservar la estﾃｩtica de una miniatura existente.
- Usa referencias separadas: `IMG 1` para la miniatura base y `IMG 2` para el sujeto/persona que debe reemplazar al original.
- Indica explﾃｭcitamente quﾃｩ texto cambiar, dﾃｳnde estﾃ｡ ubicado y quﾃｩ color o jerarquﾃｭa tiene: por ejemplo `top yellow text`, `bottom white text`.

- Si el texto final debe ser exacto y editable, elimina texto/flechas en la generaciﾃｳn y termina la composiciﾃｳn en Figma o Photoshop.
- Nunca dejes prompts generados por Gemini sin revisar; siempre cambia nombres, tﾃｭtulos y keywords antes de generar.
- Nunca subas una referencia de estilo en el campo incorrecto si la herramienta separa `app`, `character` y `style`.
- Usa imﾃ｡genes de referencia de alta calidad para sujetos, avatares o personas.
- Itera varias veces; el video recalca que mucho del proceso es experimentar hasta que encaje con el canal y el contenido.

## Errores comunes que evita o menciona
- Intentar describir manualmente un estilo complejo en vez de pedir a Gemini/ChatGPT que lo convierta en prompt.
- Olvidar cambiar textos heredados del prompt de referencia, como `Master Gemini`.
- Subir una referencia al campo equivocado: `character` o `style` cuando debﾃｭa ir en la app.
- Ser demasiado vago con cambios de texto, color o posiciﾃｳn cuando se quiere editar directamente la miniatura.
- Usar una imagen de sujeto de baja calidad y esperar resultados fieles.
- Forzar ediciﾃｳn tipogrﾃ｡fica dentro del generador cuando conviene quitar el texto y terminarlo en Figma o Photoshop.
- Asumir que Nano Banana replica igual de bien; el video afirma que GPT Image 2 es mejor para este caso.
