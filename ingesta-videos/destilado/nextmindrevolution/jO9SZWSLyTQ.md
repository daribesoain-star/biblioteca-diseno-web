# Google Flow HUGE Update: Whisk Merger & Nano Banana 2 Revealed
**Fuente:** nextmindrevolution | https://youtu.be/jO9SZWSLyTQ
## Qué enseña (2-3 líneas)
Google Flow recibió una actualización grande de interfaz, gestión de assets y generación/edición con Nano Banana 2, Nano Banana Pro y video integrado. También muestra que Whisk se fusionará progresivamente dentro de Google Flow, con opción futura para migrar medios.

## Lecciones accionables
- Usa el nuevo selector del prompt box para alternar directamente entre **image** y **video**; el text-to-video ya no está en una pestaña separada.
- Para hacer **text-to-video**, entra en la sección de video y no agregues imágenes en **ingredients**: si no hay imágenes como ingredientes, Flow interpreta el prompt como texto-a-video.
- Gestiona proyectos desde el panel izquierdo con **all assets** o **all media** para separar imágenes y videos generados dentro del proyecto.
- Usa **search** cuando el proyecto tenga muchos assets; busca por el nombre del archivo, por eso conviene renombrar cada generación.
- Usa filtros por **type**: `images`, `video`, `collections`.
- Usa filtros por **ratio**, por origen `generated on this project`, `uploaded`, o por `favorites`.
- Crea colecciones desde el botón **plus** superior: puedes **upload an image** o **create a collection**.
- Renombra colecciones desde el menú de la colección; ejemplo usado: `awesome`.
- Arrastra y suelta assets dentro de una colección; al pasar el cursor sobre una colección se muestra cuántas imágenes y videos contiene.
- Cambia la vista entre **grid** y **batch mode**; `batch mode` reproduce el flujo visual de la versión anterior.
- Ajusta el tamaño de grilla con `small`, `medium` o `large`.
- Configura si el hover reproduce sonido con **sound on hover**.
- Configura si los detalles se muestran siempre o solo al hacer hover con **show details**.
- Usa **clear prompt when you submit** si quieres que el prompt se borre después de enviar; desactívalo si necesitas conservar el texto del prompt.
- En assets de video, abre el menú de tres puntos y usa **add to scene** para mandar una escena específica al **scene builder**.
- En assets, usa opciones comunes como **download**, **favorite** y **reuse prompt**.
- En imágenes, usa **animate** para llevar una imagen a video.
- Usa **rename** en imágenes y videos porque el nuevo sistema de tags depende de encontrar assets por nombre.
- En el editor de imagen, revisa el panel derecho para ver la información de la generación actual.
- Usa **edit history** para ver cada cambio aplicado a una imagen; puedes ocultarlo o mostrarlo con **hide history**.
- Si editas una imagen y haces clic en **done**, la versión seleccionada será la que se muestre en el dashboard principal.
- Si quieres conservar dos versiones de una imagen editada, usa **save to project**, selecciona la primera versión y luego haz clic en **done**.
- Usa **crop** para recortar la generación actual sin crear una nueva generación.
- Usa **doodle tool** igual que antes; no cambió.
- Usa **box** para seleccionar áreas rectangulares.
- Usa **lasso tool** para seleccionar una zona específica dibujando o rodeando el área; luego escribe el prompt de edición en la parte inferior.
- Para agregar o bloquear ingredientes, haz clic en **plus** desde el editor de imagen u otras pantallas; la interfaz permite buscar assets o subir nuevos.
- También puedes agregar ingredientes desde el prompt usando **add** y seleccionando el asset/tag correspondiente.
- Usa tags en vez de describir ingredientes con texto largo: ya no hace falta describir el ingrediente manualmente si está bloqueado como tag.
- Ejemplo de keyword usada: `nanocarbon nanotube`.
- Ejemplo de tag usado: `James`.
- Prompt mencionado para imagen: `he is running in the forest`.
- Límite indicado: hasta **5 characters** y **14 objects**.
- Nano Banana 2 es descrito como más rápido y más avanzado que la versión Pro para prompts complejos.
- En generación de video también puedes usar tagging en **ingredients** y **frames**.
- Cuando bloqueas un sujeto o escena en video, Flow agrega el frame automáticamente.
- Ejemplo de sujeto: `Mark`.
- Prompt verbatim de video: `Mark is running in the forest`.
- En edición de video, usa el panel derecho para revisar el historial de cambios.
- En la parte inferior del editor de video ahora están **extend**, **insert** y **remove**.
- Para **insert**, selecciona o arrastra el tramo donde quieres insertar algo y luego escribe el prompt correspondiente.
- Para **remove**, selecciona el área o elemento y escribe el prompt para quitarlo.
- Según el video, **insert** y **remove** ahora se ven más realistas; antes los elementos insertados podían parecer falsos o superpuestos.
- Usa **camera edits** para ajustar **camera motion** y **camera position**.
- Revisa las animaciones visuales de cámara antes de aplicar movimientos como `orbit up` o `dolly in`.
- Usa **fast mode** en baja resolución para previews rápidos: tarda aproximadamente **70-80-90 seconds**.
- Usa **fast mode** cuando tengas créditos limitados: una preview rápida cuesta **20 credits**.
- Una vez validado el resultado en fast mode, genera la versión de alta resolución.
- En cuentas Pro, el **scene builder** ya no muestra ciertas funciones como **extend**, **jump to** y **save frame to asset**; según el video, pasaron al tier **Ultra**.
- El botón **expand** aparece como ausente y el creador lo considera probablemente un bug de lanzamiento.
- Whisk mostrará una notificación de migración: por ahora aún se puede usar, pero eventualmente desaparecerá.
- Si usas Whisk, empieza a trabajar en Flow porque Google dará opción para mover los medios de Whisk a Flow.

## Reglas para agentes
- Usa **Nano Banana 2** cuando el prompt sea complejo o necesite mejor interpretación de sujetos, objetos y acciones.
- Usa **fast mode** para validar composición, acción y consistencia antes de gastar créditos en alta resolución.
- Usa alta resolución solo después de aprobar una preview de baja resolución.
- Renombra cada asset importante antes de usar tags, porque la búsqueda de ingredientes depende del nombre del archivo.
- Usa tags de assets cuando necesites consistencia de personaje, objeto o escena.
- Nunca describas manualmente un personaje ya guardado si puedes invocarlo por tag.
- Usa **ingredients** para fijar sujetos u objetos de referencia.
- Usa **frames** cuando quieras controlar referencias visuales en generación de video.
- Usa el prompt sin imágenes en **ingredients** cuando quieras text-to-video.
- Usa **collections** para agrupar assets por escena, personaje, prueba o campaña.
- Usa **save to project** antes de volver al dashboard si necesitas conservar una variante editada además de la original.
- Usa **done** solo sobre la variante que debe quedar visible como versión principal en el dashboard.
- Usa **lasso tool** cuando la edición afecte una zona irregular o específica.
- Usa **box** cuando la edición afecte una zona rectangular.
- Usa **crop** cuando solo necesites reencuadrar sin generar una nueva imagen.
- Usa **insert** para agregar elementos a un video y **remove** para eliminarlos.
- Verifica el tier de la cuenta antes de depender de **extend**, **jump to** o **save frame to asset** en scene builder.
- Nunca asumas que funciones antiguas de scene builder siguen disponibles en Pro.
- Si vienes de Whisk, migra el flujo de trabajo a Google Flow antes de que Whisk deje de estar disponible.

## Errores comunes que evita o menciona
- Buscar la pestaña antigua de text-to-video: ahora se logra dejando **ingredients** sin imágenes.
- No renombrar assets y luego no poder encontrarlos fácilmente para tags.
- Describir personajes con texto largo en vez de usar tags bloqueados.
- Perder variantes editadas por hacer clic en **done** sin usar **save to project**.
- Gastar créditos en alta resolución antes de validar con **fast mode**.
- Esperar que **extend**, **jump to** y **save frame to asset** sigan disponibles en Pro dentro de scene builder.
- Confundir la ausencia de **expand** con una decisión final; el video lo menciona como posible bug.
- Seguir construyendo el flujo principal en Whisk cuando la herramienta se fusionará con Flow.
- Insertar elementos en video esperando resultados perfectos como antes; el video aclara que mejoró, pero la función históricamente podía verse falsa o superpuesta.
