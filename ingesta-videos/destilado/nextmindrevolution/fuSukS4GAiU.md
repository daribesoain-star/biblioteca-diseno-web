# How to use Midjourney in minutes, step by step guide in 2026
**Fuente:** nextmindrevolution | https://youtu.be/fuSukS4GAiU
## Qué enseña (2-3 líneas)
Guía rápida de la interfaz web de Midjourney: explorar referencias, crear imágenes, reutilizar prompts, ajustar parámetros, organizar generaciones y editar resultados.  
Explica cómo usar perfiles personalizados, mood boards y referencias (`omni`, `image prompts`, `style references`) para controlar estética, sujeto, composición y estilo.

## Lecciones accionables
- En `Explore`, revisa creaciones de otros usuarios para aprender de sus prompts: abre una generación, copia o reutiliza el prompt, y filtra por `styles`, `images`, `videos`, `for you`, `random`, `hot`, `top of the day`, `week`, `month` o `liked`.
- Usa la barra superior como barra de prompt; no es solo búsqueda. Ejemplo mostrado: `a 1980s woman walking on a New York City street, daytime`.
- Al enviar un prompt, Midjourney genera un batch de 4 iteraciones. Cada batch muestra prompt, `params` y acciones adicionales.
- Para reutilizar una generación:
  - `use text`: reutiliza solo el texto del prompt.
  - `use`: reutiliza prompt más `params`.
  - `rerun`: ejecuta de nuevo el mismo prompt con la misma configuración.
  - `more`: permite copiar `ID`, `seed` y otros datos.
- Para descargar:
  - `download` en el batch descarga una grilla.
  - Clic derecho sobre una imagen individual y `save image` guarda una sola imagen.
  - Al abrir una iteración individual, también puedes descargar, dar like, fijar o usar más opciones.
- Usa el pin/`spotlight` para añadir una generación al perfil público principal.
- Revisa `Settings` antes de generar: define ratio, modelo y sliders de estética.
- Slider `stylization`: controla cuánto “toque Midjourney” aplica el modelo. Ejemplo: `a realistic wide shot of a modern woman walking on a London street in that sunny day`. Valor bajo/0 produce una imagen más estándar o fotográfica; valor alto/1000 produce una imagen más estilizada/profesional.
- Slider `weird`: aumenta rareza visual. Ejemplo: `a realistic close shot of a black cat in the street`. Bajo = resultado estándar; alto = imagen más extraña.
- Slider `variety`: controla variación entre las 4 salidas. El parámetro interno se llama `chaos`. Ejemplo: `a medium close shot of a man reading a book`. Bajo = resultados similares; alto = resultados muy distintos.
- Si el prompt es poco detallado, `stylization`, `weird` y `variety/chaos` toman más control sobre el resultado.
- Usa `Organize` para buscar generaciones anteriores, filtrar resultados, cambiar layout, filtrar por perfiles y crear búsquedas guardadas.
- En `Organize`, `search prompts` busca dentro de tus propias creaciones, no en `Explore`. Ejemplo de keyword: `woman`.
- En `Organize`, mantén `Shift` y haz clic en varias generaciones para seleccionarlas en lote; luego puedes descargar todas, enviar a trash, dar like o añadir a carpetas.
- Crea carpetas para mantener limpio el workspace.
- Crea `saved searches` como etiquetas: `add search`, define título, añade términos y reutiliza el filtro desde la barra lateral.
- Antes de crear en una cuenta nueva, configura un perfil en `Personalize`; sirve para enseñar al modelo tu estética preferida.
- Activa o desactiva el perfil con la letra `P` en la parte superior. Si tienes varios perfiles, selecciónalos desde el dropdown.
- Puedes activar varios perfiles a la vez para que Midjourney mezcle sus estilos.
- Usa `Mood boards` para aplicar el estilo de un conjunto de imágenes de referencia. Puedes crear uno con `new mood board`, subir imágenes, añadir desde link o elegir imágenes de tus propias creaciones.
- Ejemplo de mood board: crear uno llamado `cat` a partir de imágenes seleccionadas.
- Para usar un mood board, selecciónalo desde el dropdown superior igual que los perfiles.
- Ejemplo mostrado: con un mood board de estilo dibujo a mano, el prompt `a man walking` produce un hombre caminando con ese estilo aunque el prompt no lo pida explícitamente.
- `Omni reference` intenta bloquear un sujeto concreto: arrastra y suelta el sujeto, ajusta el slider de fuerza y genera. Ejemplo: `a woman walking in Times Square`.
- Según el video, `omni reference` solo funciona con versión 7, mientras que la última mencionada es 8.1.
- `Image prompts` usa la imagen completa como inspiración para iluminación, color, estilo, composición y elementos; no se centra solo en el sujeto.
- `Style references` usa una imagen como referencia de estilo para crear algo nuevo. Ejemplo: `a man walking in the street`.
- En `Explore > styles`, busca estilos ya creados. Al pasar el cursor sobre un estilo, copia su `ID` para usarlo como parámetro `S ref ID`.
- En una generación, usa las opciones `image`, `style` y `prompt`; `style` coloca esa imagen en `style references`.
- Usa `try style` para ejecutar el prompt actual con ese estilo.
- Guarda estilos con el corazón y recupéralos desde `likes`.
- Usa la búsqueda sobre un style ID para encontrar variaciones similares del mismo estilo.
- Para editar imágenes, usa `edit` desde una generación:
  - `simplified edit`: brush, cambio de ratio, outpaint e inpaint.
  - `open in edit tab`: abre el editor avanzado.
  - Editor avanzado: cambio de ratio, outpaint, inpaint, brush, `smart select`, historial de ediciones, `retexture` y trabajo con `layers`.
- `Retexture` permite cambiar la textura de una imagen mediante prompt.
- Las `layers` permiten traer distintas creaciones, pintar sobre ellas y crear una composición nueva, aunque no funcionan como Canva, Photoshop o Figma.
- Flujo general recomendado: encuentra inspiración en `Explore`, crea en `Create`, edita en `Edit`, gestiona en `Organize`, y usa `Personalize`/`Mood boards` para sincronizar Midjourney con tu estilo.
- Planes:
  - `Basic`: suficiente para probar Midjourney o crear pocas imágenes.
  - `Standard`: recomendado para crear muchas imágenes.
  - `Pro` y `Mega`: para uso intensivo, profesional o si estás generando ingresos.
- `Relaxed image generations` no consume cuota mensual, pero tarda más. No está disponible en `Basic`; sí en `Standard`, `Pro` y `Mega`.

## Reglas para agentes
- Usa `Explore` cuando necesites inspiración, prompts reutilizables, estilos o referencias visuales existentes.
- Copia `use text` cuando solo necesites el prompt; usa `use` cuando también necesites preservar `params`.
- Usa `rerun` cuando quieras una nueva tanda con exactamente el mismo prompt y configuración.
- Nunca asumas que `download` del batch guarda una imagen individual; verifica si descarga grilla o imagen única.
- Usa clic derecho y `save image` cuando necesites guardar una sola iteración.
- Revisa `Settings` antes de generar cualquier imagen: ratio, modelo, `stylization`, `weird` y `variety`.
- Usa `stylization` bajo si el objetivo es realismo estándar; súbelo si el objetivo es una imagen más pulida o con estética Midjourney.
- Usa `weird` solo cuando quieras resultados visualmente extraños o menos convencionales.
- Usa `variety/chaos` alto cuando necesites explorar direcciones muy distintas; úsalo bajo cuando necesites consistencia entre salidas.
- Detalla más el prompt cuando no quieras que los sliders decidan demasiado por ti.
- Usa `Organize` y `search prompts` para recuperar generaciones propias; nunca confundas esa búsqueda con la de `Explore`.
- Usa carpetas y `saved searches` cuando el volumen de generaciones crezca.
- Activa `Personalize` con `P` cuando quieras que el resultado obedezca el gusto entrenado del perfil.
- Usa varios perfiles juntos solo cuando quieras una mezcla de estilos.
- Usa `Mood boards` cuando quieras transferir el estilo de un conjunto de imágenes.
- Usa `omni reference` cuando el objetivo sea mantener un sujeto; verifica que el modelo compatible sea versión 7.
- Usa `image prompts` cuando quieras transferir composición, iluminación, color, elementos o ambiente general.
- Usa `style references` cuando quieras transferir solo el estilo visual.
- Guarda estilos importantes con like y reutiliza su `S ref ID`.
- Usa `Edit` para inpaint, outpaint y cambios de ratio; usa `open in edit tab` cuando necesites historial, retexture, smart select o layers.
- Elige `Standard` si vas a generar muchas imágenes y necesitas `relaxed image generations`.

## Errores comunes que evita o menciona
- Pensar que la barra superior es solo búsqueda, cuando en realidad es la barra de prompt.
- Descargar el batch completo esperando recibir una imagen individual.
- Ignorar `params` al reutilizar una generación y no obtener el mismo comportamiento.
- No revisar `Settings` antes de generar y dejar que sliders/modelo/ratio afecten el resultado sin control.
- Usar prompts demasiado vagos y permitir que `stylization`, `weird` o `variety` dominen la generación.
- Buscar manualmente entre demasiadas imágenes sin usar `Organize`, filtros, carpetas o `search prompts`.
- Confundir `search prompts` con la búsqueda pública de `Explore`.
- Esperar consistencia perfecta de personajes en Midjourney; el video dice que no es el mejor lugar para personajes consistentes.
- Usar `omni reference` sin considerar que, al momento del video, solo funciona con versión 7.
- Confundir `omni reference`, `image prompts` y `style references`: sujeto, imagen completa y estilo cumplen funciones distintas.
- Comprar `Basic` esperando tener `relaxed image generations`; esa opción empieza en `Standard`.
