# WhatsApp Flows docs oficiales (Meta)
**Fuente:** https://developers.facebook.com/docs/whatsapp/flows/ (consultada 2026-07-09)

## Qué es / qué regula
WhatsApp Flows permite crear interacciones estructuradas para mensajes comerciales. Las empresas pueden definir, configurar y personalizar mensajes con interacciones eficaces para que los clientes se comuniquen de forma más estructurada. Se usa para generar clientes potenciales, recomendar productos, captar nuevos clientes potenciales para ventas o cualquier fin donde una comunicación estructurada resulte más natural.

## Reglas y datos duros para agentes
- **Casos de uso populares documentados:** Consigue clientes potenciales para préstamos preaprobados, Obtener cotización de seguro, Recopilar intención de compra, Oferta personalizada.
- **Flujo de préstamos preaprobados:** Presenta monto de préstamo, permite personalizar condiciones (período de amortización), seleccionar método de pago y desembolso, e incluye autenticación del usuario para confirmar identidad.
- **Flujo de cotización de seguro:** Recopila preferencias y detalles personales (número de miembros cubiertos, riesgos de salud, primas), permite ver, personalizar y elegir planes, y seleccionar frecuencia de pago.
- **Flujo de intención de compra:** Recopila datos personales (nombre, dirección de correo electrónico) y permite seleccionar productos o servicios de interés.
- **Flujo de oferta personalizada:** Permite seleccionar productos, presupuesto, y recibe recomendación específica; luego se envía mensaje con instrucciones para completar la compra.
- **Cumplimiento obligatorio:** Deberás cumplir con las regulaciones locales aplicables en relación con el tratamiento de datos de usuarios.
- **Política aplicable:** El caso de uso debe cumplir con la Política de mensajes de WhatsApp Business.
- **Secciones de documentación disponibles:** Información general, Primeros pasos, Guías (prácticas recomendadas, ejemplos con JSON de Flow), Referencia (JSON de Flow, REST API de Flows, códigos de error), Sección de pruebas, Obtener asistencia, Registro de cambios (versiones actuales, cambios en la API).
- **Fecha de actualización del documento:** 21 may 2026.

## Errores que evita / prohibiciones
- No usar Flows para fines que no cumplan con la Política de mensajes de WhatsApp Business.
- No tratar datos de usuarios sin cumplir las regulaciones locales aplicables.
- No usar casos de uso no documentados sin verificar cumplimiento normativo y de políticas.
