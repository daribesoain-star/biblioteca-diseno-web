# 10+ Actual Use Cases of Nano Banana Pro (With Prompts Included)
**Fuente:** imskaigenerated | https://youtu.be/6IjWtDo6G0w

## Qué enseña
Muestra 12 casos de uso prácticos de Nano Banana Pro en Open Art, desde transformación anime a realista hasta generación de infografías faciales y objetos 3D esponjosos. Incluye prompts exactos para replicar cada resultado y demuestra cómo el modelo mantiene la identidad del sujeto mientras aplica estilos complejos.

## Lecciones accionables
- **Facial Aesthetic Report:** Subir foto (ej. Dwayne Johnson) a Nano Banana Pro. Prompt: "keep the person's face exactly as it is, not change anything about them, and just overlay a clean infographic on top with labels pointing to each facial feature with percentage scores". Resultado: infografía clínica con líneas blancas y puntuaciones (88% overall).
- **Anime a Realista:** Subir screenshot de anime (ej. Toji, Itachi y Sasuke). Prompt exacto: "Create a realistic photo of this character". El modelo preserva criaturas, poses, iluminación y textura de piel.
- **Upscaling a 4K:** Subir imagen borrosa (ej. camello). Prompt exacto: "enlarge this image to 4K resolution". Resultado: detalle en pelaje y textura de arena nítidos.
- **Movie Poster Swap:** Subir póster (ej. Joker con Joaquin Phoenix). Prompt: "Replace the character with Elon Musk". Mantiene título, tagline, maquillaje y composición. Otro ejemplo: póster de Matrix con prompt "Replace all the characters with some of the top powerful AI leaders in the world".
- **3D Fluffy Object:** Subir imagen (ej. escultura de cabeza colorida). Prompt: "Transform this image into a 3D fluffy object. Use the exact colors. The shape is fully covered in fur with hyperrealistic hair texture and soft shadows. The object on a clean, light gray background and floats gently in space. The style is surreal, tactile, and modern, evoking a sense of comfort and playfulness. Studio lighting, high-resolution render". Funciona en paloma, perro de dibujos animados, etc.
- **Décadas Fashion Grid:** Subir foto de persona. Prompt: "Make a 4x4 grid starting with the 1880s. In each section, I should appear styled according to that decade, clothing, hairstyle, facial hair, accessories. Use colors, background, and film style accordingly". Resultado: 1880s victoriano, 1920s flapper, 1950s, 1970s disco hasta 2020s.
- **Apple Genmoji Avatars:** Subir foto grupal (ej. grupo de personas o Avengers). Prompt: "Show me one unique Apple Genmoji for each person in the image with consistent proportions, head and shoulders only, on a white background". Captura trajes y rasgos faciales.
- **ID Card Realista (bonus):** Subir foto (ej. Cristiano Ronaldo). Prompt: "turn the photo into a realistic national identification card with a made-up country called Portava, complete with a flag, fake details like name, date of birth, ID number, expiration date". Resultado: holographic sheen, gold chip, firma.

## Reglas para agentes
- Usa "Create a realistic photo of this character" cuando transformes anime a realista, sin modificar la prompt.
- Usa "enlarge this image to 4K resolution" exactamente para upscaling, sin añadir descripciones extra.
- Usa "Replace the character with [nombre]" para swaps en pósters, manteniendo el resto del prompt mínimo.
- Usa el prompt completo de "3D fluffy object" (con "exact colors", "hyperrealistic hair texture", "light gray background", "floats gently in space", "studio lighting") para cualquier objeto a esponjoso.
- Usa "Make a 4x4 grid starting with the 1880s" para grids de moda por décadas, especificando "clothing, hairstyle, facial hair, accessories" y "colors, background, and film style accordingly".
- Usa "Show me one unique Apple Genmoji for each person" con "consistent proportions, head and shoulders only, on a white background" para avatares estilo emoji.
- Nunca uses prompts que pidan modificar la identidad facial en el Facial Aesthetic Report (debe mantener la cara exacta).
- Nunca uses este modelo para crear identificaciones falsas reales (solo para demostración artística).

## Errores comunes que evita o menciona
- Evita prompts largos innecesarios en transformaciones anime a realista: con solo 6 palabras ("Create a realistic photo of this character") funciona perfectamente.
- Evita modificar la composición original en movie poster swaps: el modelo mantiene título, tagline y fondo automáticamente.
- Evita usar imágenes de baja resolución sin upscaling previo si necesitas calidad de impresión.
- Evita pedir cambios de color en objetos 3D fluffy: el prompt especifica "use the exact colors" para preservar la paleta original.
- Evita prompts vagos en grids de décadas: especificar "clothing, hairstyle, facial hair, accessories" y "film style" es clave para resultados coherentes.
- No intentes usar AI para crear IDs falsos reales (advertencia explícita del creador).
