# 11 Nano Banana 2 Prompts You NEED to Try!
**Fuente:** imskaigenerated | https://youtu.be/oZsbQ-eY-XA

## Qué enseña (2-3 líneas)
El video revela 11 casos de uso avanzados de Nano Banana 2, más allá de ser una versión rápida y barata de Nano Banana Pro. Muestra cómo aprovechar conocimiento del mundo en tiempo real, renderizado de texto en imágenes, generación de celebridades y personajes sin referencias, edición precisa de imágenes, y creación de infografías, productos conceptuales y gráficos a partir de datos.

## Lecciones accionables
- **Prompt para cómic de noticias en tiempo real:** Usar Higgs Field → hover sobre Image → seleccionar Nano Banana 2. Configurar: aspect ratio 16:9, resolución 4K, batch size 4. Prompt: *"a comic of today's most interesting news story"*
- **Prompt para póster de ventana con clima real:** Template: *"a photorealistic window view poster of [location] showing the current weather outside the [specific view]"* — reemplazar [location] por "New York", [specific view] por "window". Nano Banana 2 infiere clima actual (nublado) aunque temperatura puede ser imprecisa.
- **Renderizado de texto en imagen (ej. Coca-Cola a chino):** Subir imagen existente → prompt: *"convert the text to Chinese, keep everything else exactly the same"* → aspect ratio en "auto" para que coincida con input → Generate. Traduce incluso texto pequeño en tapas.
- **Aspect ratios disponibles:** 1:1, 9:16, 16:9, 3:4, 4:3, 21:9, 8:1, 1:8 y más. Usar 21:9 para panorámicas: *"a vast panoramic world that shifts from snowy mountains on the far left, quiet green valley in the center, and a golden desert on the far right beneath one continuous sunrise sky"*
- **Generación de celebridades sin referencia:** Prompt: *"a 16:9 group photo of the current most famous celebrities in the world"* — sin nombres ni imágenes de referencia. Nano Banana 2 usa búsqueda automática de imágenes.
- **Generación de personajes de caricatura sin referencia:** Mismo prompt pero reemplazar "celebrities" por "cartoon characters"
- **Mezcla celebridades reales + caricaturas:** Prompt: *"a photo that seamlessly mixes famous real-life celebrities with famous fictional cartoon characters"*
- **Grid de emociones:** Prompt: *"a 4x4 grid of a face across 16 labeled emotions"*
- **Infografía de especies en peligro:** Prompt: *"a grid of the six most endangered snakes, each with their own scientific names and descriptions"*
- **Infografía de montañas:** Prompt: *"a grid of the five tallest mountains in the world, each with their own scientific names and descriptions"*
- **Edición de imagen existente (cambio de peinado):** Subir imagen (ej. creada con Higgs Field Soul) → prompt: *"change her hairstyle to a classic ponytail, keep everything else exactly the same"*
- **Concepto de producto de lujo:** Template prompt: *"[brand name] product concept, high-end, luxury, premium, minimalist, elegant"* — reemplazar [brand name] por "Gucci"
- **Conversión de tabla de datos a gráfico:** Pegar tabla de datos en prompt → *"convert this data table into a simple visual chart"*
- **Consistencia de hasta 5 personajes con imágenes de referencia:** Subir hasta 5 imágenes de referencia → prompt: *"a photo of all of these characters about to start a foot race against each other"*
- **Estilo de cámara específica:** Añadir al inicio del prompt: *"Ilford HP5+ 400 camera style"* — Nano Banana 2 automáticamente genera en blanco y negro sin indicarlo explícitamente.

## Reglas para agentes
- Usa aspect ratio "auto" cuando edites imágenes existentes para que coincida con el input original
- Siempre selecciona Nano Banana 2 en el menú desplegable del prompt box antes de generar
- Configura batch size en 4 para tener múltiples opciones de resultado
- Para personajes famosos, no subas imágenes de referencia; confía en la búsqueda automática de Nano Banana 2
- Cuando especifiques un modelo de cámara (ej. Ilford HP5+ 400), no añadas instrucciones de color; el modelo inferirá las características automáticamente
- Para infografías, incluye explícitamente "scientific names and descriptions" en el prompt para obtener texto preciso
- Nunca uses nombres de celebridades en el prompt si quieres probar la capacidad de búsqueda automática; usa frases como "current most famous celebrities"

## Errores comunes que evita o menciona
- **No asumir que Nano Banana 2 es solo una versión más rápida y barata de Nano Banana Pro** — tiene capacidades únicas que nadie está usando
- **No esperar consistencia perfecta en el clima en tiempo real** — la temperatura puede ser ligeramente incorrecta aunque las condiciones climáticas generales (nublado/soleado) suelen ser precisas
- **No subir imágenes de referencia para personajes famosos** — el modelo puede identificarlos automáticamente sin necesidad de referencias
- **No olvidar que el renderizado de texto en otros idiomas funciona** — incluso traduce texto pequeño en objetos como tapas de botellas
- **No subestimar la capacidad de generar emociones precisas en grids** — aunque otros modelos fallan, Nano Banana 2 lo logra
- **No ignorar la edición de imágenes existentes** — funciona cambiando un elemento específico (ej. peinado) sin alterar el resto de la imagen
- **No pensar que los gráficos a partir de datos son complicados** — el proceso es directo: pegar tabla + prompt simple
