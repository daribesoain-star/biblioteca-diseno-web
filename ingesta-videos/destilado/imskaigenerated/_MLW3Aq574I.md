# GPT Image 2 Can Fake ANYTHING! (11 real use cases)
**Fuente:** imskaigenerated | https://youtu.be/_MLW3Aq574I

## Qué enseña (2-3 líneas)
GPT Image 2, usado desde OpenArt AI, destaca por `thinking mode`, generación de sets coherentes multiimagen y mejor renderizado de texto no latino.  
El video muestra 11 casos prácticos donde el modelo produce imágenes con texto legible, layouts realistas, consistencia visual y aspecto de captura, póster, empaque o anuncio terminado.

## Lecciones accionables
- Acceso: entra a `OpenArt`, haz clic en `image`, cambia el modelo a `GPT image 2` y genera desde ahí.
- Usa `thinking mode` para prompts complejos: el modelo razona antes de generar, mejora precisión, estructura y manejo de instrucciones largas.
- Para sets coherentes, sube una imagen de referencia cercana del personaje y describe un storyboard; el modelo puede generar hasta `8 images in one go` manteniendo personaje y objetos consistentes.
- Caso 1, screenshots falsos: prompt descrito como `full desktop YouTube page in dark mode`, con `real video title`, `view counts`, `comment section`, `proper UI layout`, `flat and pixel perfect like an actual screenshot`; configurar `aspect ratio` en `16 by 9`, `quality` en `high`, luego `generate`.
- Variante de screenshot: describe una vista desktop de `Twitter X timeline` en dark mode con `main tweet`, `engagement numbers`, `reply` y `trend sidebar`; especifica `verified badge`, `handle` y `timestamp`.
- Caso 2, menús de restaurante: pide un `full casual dining menu` con `Starters`, `mains`, `desserts`, `beverages`, nombres de platos, precios y layout de PDF impreso; exige precios alineados a la derecha y texto legible.
- Caso 3, escritura a mano: usa detalles físicos concretos como `blue ballpoint pen`, `casual notebook page`, `natural, slightly imperfect human handwriting`, `uneven spacing`, `slight slant`, `wavy baseline`, `ink pressure variation`.
- Variante manuscrita: pide un ensayo en lápiz cursivo titulado `The History of Money in the World`, en página envejecida, líneas tenues, marcas de agua sutiles y varios párrafos con estilo cursivo consistente.
- Caso 4, cómics y manga: prompt con `full-color shonen manga page`, `four to six panels`, joven guerrero, espada de cristal llamada `the blade of eternal flame`, templo de montaña, `speed lines`, `energy bursts`, primeros planos emocionales y globos de diálogo legibles.
- Variante de cómic: secuencia de `six-panel superhero versus villain fight`, rooftop nocturno, dos personajes con looks definidos y consistentes, diálogo y efectos de sonido dentro de los frames.
- Caso 5, anuncios: prompt de beauty ad premium con `glow lip balm`, frasco `glass dropper bottle`, modelo interactuando con producto, tipografía limpia, marca, tagline y detalles del producto dentro de la imagen; textos usados: `glow lip balm`, `hydrating shine`, `lightweight`, `nourishing`, `radiant`.
- Caso 6, infografías: pide una infografía textil limpia sobre cómo funcionan los `large language models`, layout de izquierda a derecha, `five steps`, cajas ordenadas, etiquetas cortas, flechas conectando pasos y título claro arriba.
- Caso 7, thumbnails de YouTube: prompt de `split screen before and after transformation thumbnail`, texto central `30 days`, tonos fríos desaturados a la izquierda, tonos cálidos dorados a la derecha y borde luminoso dividiendo el frame.
- Caso 8, mockups y packaging: describe marca premium de café con pouch mate resellable y caja a juego, nombre `Ember Roast`, detalles completos, `tasting notes`, `brewing guide`, `barcode`, `QR code`, íconos de certificación, textura mate, zipper y válvula desgasificadora.
- Caso 9, UI y apps: prompt de dashboard de productividad estilo `Notion` o `Linear`, con `sidebar navigation`, `task table`, `metric cards`, `weekly progress chart`, `activity feed`, plano y frontal como screenshot real de producto.
- Caso 10, localización: usa el prompt `replace all text with natural, fluent Chinese newspaper style writing preserving layout, column structure, and typographic realism`; repetir para japonés e hindi preservando estructura de columnas.
- También funciona con texto no latino en contexto: `Shibuya street at night with a Japanese billboard`, `Seoul subway train with a Korean route display`, `Chinese calligraphy studio with a hanging scroll`, `Dhaka tea stall with a hand-painted Bangla sign`.
- Caso 11, pósters de película: prompt de póster sci-fi premium con astronauta flotando cerca de nave dañada, señal misteriosa, tagline `we were never alone`, título `Last Signal` abajo y créditos pequeños en formato de póster.

## Reglas para agentes
- Usa `GPT image 2` cuando el output requiera texto legible dentro de la imagen.
- Usa `thinking mode` cuando el prompt tenga múltiples restricciones, layouts complejos o escenas secuenciales.
- Configura `aspect ratio: 16 by 9` y `quality: high` para screenshots desktop o thumbnails horizontales.
- Describe explícitamente estructura, contenido textual, jerarquía visual y estilo físico del soporte.
- Para UI falsa, exige `flat`, `front-facing`, `pixel perfect` y layout de screenshot real.
- Para personajes recurrentes, aporta imagen de referencia y define rasgos constantes antes de pedir múltiples paneles.
- Para packaging, lista todos los elementos imprimibles: marca, detalles, notas, guía, barcode, QR code e íconos.
- Para localización, ordena preservar layout, columnas y realismo tipográfico al reemplazar texto.
- Nunca pidas solo “un anuncio bonito”; especifica producto, material, modelo, textos exactos, tagline y placement.
- Nunca asumas que el texto saldrá bien si no indicas idioma, ubicación, longitud y función visual del texto.

## Errores comunes que evita o menciona
- Distorsión de UI en screenshots generados.
- Texto ilegible en comentarios, tablas, globos de diálogo, empaques o anuncios.
- Pérdida de consistencia de personaje entre paneles o imágenes.
- Mal renderizado de caracteres no latinos en chino, japonés, coreano, hindi o bengalí.
- Layouts rotos al traducir o reemplazar texto en piezas editoriales.
- Prompts demasiado vagos que no especifican columnas, flechas, precios, badges, timestamps o detalles de producto.
- Tener que reconstruir manualmente en Photoshop piezas como screenshots, menús, pósters, thumbnails o mockups.
