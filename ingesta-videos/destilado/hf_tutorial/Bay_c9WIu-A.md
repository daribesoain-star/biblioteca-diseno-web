# Higgsfield AI Tutorial for Beginners (Step-by-Step 2026)
**Fuente:** hf_tutorial | https://youtu.be/Bay_c9WIu-A

## Que ensena sobre Higgsfield (2-3 lineas)
Higgsfield AI permite generar imágenes y videos hiperrealistas con modelos como **Soul** (moda/editorial), **Cinema Studio**, **Seedance** y **Kling 2.1**. Incluye control de cámara, animación facial (face swap), creación de avatares consistentes (Soul ID) y herramientas de edición como instruct, edit prompt y draw-to-video. La plataforma tiene planes gratuitos (cola lenta) y de pago (10x más rápido, 40% descuento anual).

## Tips y flujos accionables

### Flujo de generacion de imagen (paso a paso)
1. **Ir a:** hickfield.ai → Login/Sign up (usar Google)
2. **Seleccionar:** pestaña "Image" → "Create Image"
3. **Prompt:** escribir en barra inferior (ejemplo: *"create an action scene of a cast taking a photo where an explosion occurs in the background"*)
4. **Referencia:** clic en "+" para subir imagen de referencia (JPG/PNG desde PC)
5. **Modelos disponibles:** elegir entre **Soul**, **Cinema Studio**, **Seedance**, **Kling 2.1**, **PhotoDump Studios**
6. **Parametros:**
   - **Aspect Ratio:** seleccionar (16:9, 1:1, 9:16, etc.)
   - **Quality:** slider (bajo/medio/alto)
   - **Batch Size:** numero de variantes (1-4)
7. **Generar:** clic "Generate" (gratis = cola lenta; pago = 10x rapido)
8. **Post-procesamiento:**
   - **Edit:** clic en imagen → "Instruct" (ej: *"use only a single person in the shot"*) o "Edit Prompt" para modificar prompt original
   - **Animate:** opciones "Video", "Speak", "UGC", "Draw to Video"
   - **Publicar:** icono de publicacion

### Flujo de video con face swap
1. **Ir a:** pestaña "Video" → "Create Video"
2. **Seleccionar:** "Character" → subir foto propia como "Source Image"
3. **Target Image:** subir imagen donde se insertara el rostro
4. **Modelo:** elegir en barra inferior (Soul recomendado para realismo)
5. **Generar face swap:** clic "Generate" (esperar cola)
6. **Resultado:** rostro reemplazado con alta precision
7. **Animar:** seleccionar "Video" o "Speak" desde las opciones inferiores
8. **Prompt de video:** en panel izquierdo escribir (ej: *"explosion in the background"*)
9. **Generar video:** clic "Generate" (requiere plan pago)

### Flujo de exploracion de estilos
1. **Secciones en pantalla principal:** "Hicksfield Soul Apps", "Kling 2.1", "Master", "Camera Controls", "Viral", "Action Movements", "UGC", "Commercial"
2. **Copiar estilo:** clic "Generate" en cualquier tarjeta para usar ese preset
3. **Popcorn:** "Create visual stories" → elegir entre "Auto" o "Manual" + "InstaDump" packs
4. **Community:** explorar contenido de otros usuarios para inspiracion

### Atajos y trucos
- **Soul ID:** para avatar digital consistente, subir varias fotos propias antes de generar
- **PhotoDump Studios:** preset para look cinematografico con grano y color grading
- **Camera Controls:** en seccion "Master" para motion control (pan, tilt, zoom, dolly)
- **Instruct vs Edit Prompt:** "Instruct" es mejor para cambios especificos sin reescribir todo; "Edit Prompt" para cambiar completamente la escena
- **Board:** crear boards para organizar proyectos (solo plan pago)

## Reglas para el erudito de Higgsfield

- **Para imagen hiperrealista de moda/editorial:** usa modelo **Soul** con **PhotoDump Studios** preset, aspect ratio 3:4, quality alto, batch size 2
- **Para video cinematografico con motion control:** usa **Cinema Studio** o **Seedance** con **Camera Controls** (pan 15°, tilt -5°, zoom 1.2x)
- **Para face swap preciso:** usa modelo **Soul** con source image bien iluminada y target image con angulo facial similar
- **Para animacion de personaje hablando:** genera imagen con **Soul**, luego en "Animate" selecciona **Speak** con prompt de dialogo
- **Para explosion/accion en video:** prompt exacto *"explosion in the background"* con modelo **Kling 2.1** para efectos realistas
- **Para ahorrar creditos:** usa **Batch Size 1** y **Quality bajo** en pruebas; solo sube a Quality alto en exportacion final
- **Para velocidad maxima:** suscribete a plan anual (40% descuento) para 10x generacion

## Errores comunes / que evitar

- **Usar free plan para proyectos grandes:** la cola es extremadamente lenta; solo sirve para pruebas de 1-2 imagenes
- **No subir referencia de rostro bien iluminada:** el face swap alucina si la source image tiene sombras duras o angulo extremo
- **Ignorar aspect ratio:** si no se selecciona, por defecto usa 1:1; para video usar 16:9 o 9:16
- **Usar "Edit Prompt" cuando se necesita "Instruct":** "Edit Prompt" reescribe todo el prompt; "Instruct" solo modifica lo indicado
- **No explorar secciones de estilos:** perder acceso a presets como "Viral", "Action Movements" y "UGC" que optimizan prompts
- **Confundir modelos:** Soul es para realismo extremo; Cinema Studio para cinematic look; Kling 2.1 para efectos dinamicos; Seedance para animacion estilizada
- **Olvidar que "Board" y "Assist" requieren plan pago:** no intentar usarlos en free plan
- **No verificar creditos disponibles:** en perfil (top right) antes de generar lotes grandes
