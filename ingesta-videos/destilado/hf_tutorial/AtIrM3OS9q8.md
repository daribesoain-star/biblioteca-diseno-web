# EXACTLY How to use Higgsfield AI in 2026
**Fuente:** hf_tutorial | https://youtu.be/AtIrM3OS9q8

## Que ensena sobre Higgsfield (2-3 lineas)

Higgsfield permite generar imágenes y videos con calidad cinematográfica usando presets específicos, control de cámara profesional y consistencia de personajes. Las 7 funciones clave son: selección de modelo según estilo, apps de edición (outfit swap, shots, transitions), ángulos de cámara, creación de personajes consistentes, AI influencers, Cinema Studio con lentes reales, y el panel de director para video.

## Tips y flujos accionables

### Selección de modelo para imágenes (crítico)
- **Ultra-realismo:** usa **Nano Banana 2** (preset exacto)
- **Estilo artístico/stylized:** usa **Sea Dream 5.0** (preset exacto)
- **Personajes propios (character feature):** usa **Higgsfield Soul**
- **Resolución:** siempre selecciona **16:9** y **4K** para máxima calidad

### Apps de edición (dentro del panel "Apps")

**Outfit Swap:**
1. Sube imagen del personaje
2. Sube imagen de la ropa que quieres ponerle
3. Genera → el fondo, pose e iluminación se mantienen intactos

**Shots (9 ángulos automáticos):**
1. Sube una imagen de personaje
2. Haz clic en "Generate"
3. Obtienes: close-ups, upward framings, overhead shots
4. Selecciona todas, upscáelas y descárgalas
5. Repite hasta tener **20-30 imágenes de referencia** para personaje consistente

**Transitions (transiciones cinematográficas):**
1. Sube imagen de inicio y fin
2. Elige entre **20+ transiciones**: la recomendada es **Raven**
3. Selecciona duración
4. Genera → transición suave entre escenas

**Angles (control de cámara):**
- Usa el **círculo en el panel derecho** para posicionar la cámara exacta
- Arriba = drone shot
- Abajo = low angle

### Character Consistency (personaje consistente)
1. Crea personaje con imagen inicial
2. Ve a **Shots** → genera 9 ángulos → upscala → descarga
3. Repite hasta tener **20-30 fotos** de diferentes ángulos
4. Ve a **Character** → sube todas las fotos → nombra el personaje (ej: "Kim")
5. Espera que el AI entrene el modelo
6. Ahora puedes ponerlo en cualquier entorno/vestimenta sin perder identidad

### AI Influencer (sin prompts)
1. Ve a **AI Influencer**
2. Configura: **Character type** (human/elf/crocodile/octopus), **Gender**, **Ethnicity**, **Skin color**, **Eye color**
3. Detalles extra: **Skin condition** (freckles, birthmarks, scars), **Age**, **Body type** (athletic), **Style** (hair, tattoos)
4. **Render style:** selecciona **Hyperrealistic**
5. Genera → personaje listo para redes sociales

### Cinema Studio (configuración profesional)
- **Preset recomendado:** **Premium Large Format Digital**
- **Lente:** **Classic Anamorphic** a **35mm**
- **Apertura:** **F1.4** (fondo borroso, profundidad natural)
- **Resolución:** de **1K a 4K**
- **Grid:** usa **2x2 grid** para previsualizar múltiples generaciones sin gastar créditos
- **What's Next:** después de generar, usa esta función para obtener variaciones de historia sin escribir prompts

### Panel de Director (video)
- **Camera movements:** docenas de opciones disponibles
- **Speed ramping:** ajuste manual o **Auto** si no sabes qué elegir
- **Duration:** recomienda **5 segundos**
- **Genre:** elige emoción: **Action**, **Horror**, **Suspense**, **Intimate** (recomendado para escenas suaves)
- Sube **start frame** y **end frame** → genera

## Reglas para el erudito de Higgsfield

- **Para ultra-realismo** usa el preset **Nano Banana 2** con resolución **16:9 y 4K**
- **Para estilo artístico** usa el preset **Sea Dream 5.0**
- **Para personajes propios** usa **Higgsfield Soul**
- **Para cambiar ropa sin alterar fondo/pose** usa **Apps > Outfit Swap**
- **Para obtener 9 ángulos de un personaje en 30 segundos** usa **Apps > Shots**
- **Para transiciones cinematográficas** usa **Apps > Transitions** con preset **Raven**
- **Para control exacto de cámara** usa el **círculo en Angles** (arriba = drone, abajo = low angle)
- **Para personaje consistente en múltiples escenas** sube **20-30 fotos de referencia** en **Character**
- **Para AI influencer sin prompts** usa **AI Influencer** con **Render style: Hyperrealistic**
- **Para look cinematográfico profesional** usa **Cinema Studio** con preset **Premium Large Format Digital**, lente **Classic Anamorphic 35mm**, apertura **F1.4**
- **Para previsualizar sin gastar créditos** usa **Grid 2x2**
- **Para generar variaciones de historia sin prompts** usa **What's Next**
- **Para video con control emocional** usa **Panel de Director** con **Genre** según la escena y **Duration 5 segundos**
- **Para speed ramping** si no sabes qué valor usar, selecciona **Auto**

## Errores comunes / que evitar

- **No usar el modelo incorrecto para el estilo deseado:** principiantes eligen al azar → resultados mediocres. Usa Nano Banana 2 para realismo, Sea Dream 5.0 para artístico.
- **Modificar prompt esperando que solo cambie una cosa:** el AI cambia todo lo que te gustaba. Usa **Outfit Swap** en lugar de re-promptear.
- **No generar suficientes ángulos de referencia:** menos de 20 fotos para personaje consistente → el personaje varía entre escenas. Mínimo 20-30 fotos.
- **Usar prompts para cada ángulo nuevo:** quema créditos y da resultados inconsistentes. Usa **Shots** para 9 ángulos en un clic.
- **Ignorar el control de cámara:** sin usar **Angles** las tomas se ven planas. Posiciona la cámara con el círculo.
- **No usar Cinema Studio para calidad cinematográfica:** las imágenes básicas no tienen profundidad ni blur profesional. Usa el preset con lente anamórfico.
- **Olvidar el Grid 2x2:** genera imágenes de una en una gastando créditos innecesarios. Previsualiza con grid primero.
- **No usar What's Next para narrativa:** escribir prompts para cada variación de historia es lento. Usa What's Next para ideas automáticas.
- **Elegir Genre incorrecto en video:** el género define la emoción del video. No lo dejes en default si buscas un tono específico.
- **No upscalar las imágenes de Shots antes de usarlas para Character:** baja calidad en referencias → personaje inconsistente. Siempre upscala antes de descargar.
