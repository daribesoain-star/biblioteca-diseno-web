# Higgsfield AI tutorial how to use guide
**Fuente:** hf_tutorial | https://youtu.be/Higgsfield AI tutorial how to use guide

## Que ensena sobre Higgsfield (2-3 lineas)
Enseña a usar 7 funciones ocultas para generar videos e imágenes cinematográficas con IA: selección de modelos de imagen (Nano Banana 2, Sea Dream 5.0, Higgsfield Soul), control de cámara profesional (Cinema Studio con lentes anamórficos, apertura F1.4), consistencia de personaje mediante entrenamiento con 20-30 fotos de referencia, y transiciones suaves entre escenas usando la app Transitions.

## Tips y flujos accionables

### Selección de modelo de imagen (imagen base)
- **Para ultra-realismo:** usa **Nano Banana 2**
- **Para look artístico/estilizado:** usa **Sea Dream 5.0**
- **Para personajes propios (character features):** usa **Higgsfield Soul**
- **Resolución recomendada:** 16:9, calidad **4K**
- **Prompt usado en tutorial:** *"young woman exploring an erupting volcano"*

### Apps de control de imagen
1. **Outfit Swap** (Apps > Outfit Swap)
   - Sube 2 imágenes: personaje + prenda deseada
   - El fondo e iluminación se mantienen intactos
   - Ideal para AI influencers o UGC content

2. **Shots** (Apps > Shots)
   - Sube 1 imagen → genera 9 ángulos profesionales automáticamente (close-ups, upward framing, overhead shots)
   - Sin necesidad de prompts
   - Tiempo: ~30 segundos
   - Flujo: genera → upscale → descarga todas → úsalas como referencia para consistencia

3. **Angles** (panel derecho, círculo interactivo)
   - Arrastra el punto para elegir ángulo exacto de cámara
   - Drone shot: cámara arriba
   - Low angle: cámara abajo

### Transiciones cinematográficas
- **Apps > Transitions**
- Sube 2 imágenes: start frame + end frame
- Elige entre **20+ transiciones** (recomendada: **Raven**)
- Ajusta duración
- Resultado: transición suave tipo flying camera o morphing

### Consistencia de personaje (Character)
1. Genera imagen base del personaje
2. Ve a **Shots** → genera 9 ángulos → upscale → descarga
3. Repite hasta tener **20-30 fotos de referencia** (más ángulos = mejor resultado)
4. Ve a **Character** (menú superior) → sube todas las fotos
5. Nombra el personaje (ej: "Kim")
6. Espera que la IA entrene el modelo
7. Ahora puedes ponerlo en cualquier entorno/vestimenta sin perder el look

### AI Influencer (sin prompts, sin workflow de 30 fotos)
- **AI Influencer** (menú superior)
- Opciones configurables:
  - **Character type:** human / elf / crocodile / octopus
  - **Gender:** male / female
  - **Ethnicity:** Asian / etc.
  - **Skin color:** selector exacto
  - **Eye color:** amber / etc.
  - **Skin condition:** freckles / birthmarks / scars
  - **Age:** adult
  - **Body type:** athletic
  - **Style:** hair / tattoos
  - **Render style:** hyperrealistic
- Genera un personaje listo para redes sociales

### Cinema Studio (control profesional de cámara)
- **Preset recomendado:** Premium Large Format Digital
- **Lente:** Classic Anamorphic a **35mm**
- **Apertura:** **F1.4** (fondo borroso, profundidad natural, look cinematográfico)
- **Aspect ratio:** 16:9
- **Resolución:** 1K a 4K
- **Grid (vista previa):** 2x2 para comparar variaciones sin gastar créditos
- **What's Next:** genera múltiples variaciones de lo que podría pasar después (sin prompt)

### Video (Director Panel)
- **Camera movements:** docenas de opciones
- **Speed ramping:** ajuste manual (slow motion / speed up) o **auto**
- **Duration:** 5 segundos (recomendado)
- **Genre (emoción):** action / horror / suspense / **intimate** (usado en tutorial)
- Sube start frame + end frame → genera

## Reglas para el erudito de Higgsfield

1. **Para ultra-realismo** usa el preset **Nano Banana 2** con resolución **4K** y aspect ratio **16:9**
2. **Para look artístico** usa **Sea Dream 5.0**
3. **Para consistencia de personaje** entrena con **20-30 fotos de referencia** usando la función **Shots** primero
4. **Para look cinematográfico** usa **Cinema Studio** con preset **Premium Large Format Digital**, lente **Classic Anamorphic 35mm**, apertura **F1.4**
5. **Para transiciones suaves** usa **Apps > Transitions** con preset **Raven**
6. **Para control de cámara exacto** usa el **círculo interactivo** en el panel derecho (Angles)
7. **Para AI influencers sin prompts** usa **AI Influencer** con render style **hyperrealistic**
8. **Para variaciones de historia** usa **What's Next** después de generar la imagen base
9. **Para comparar sin gastar créditos** usa **Grid** en **2x2**
10. **Para video con emoción** selecciona **Genre** en el Director Panel (intimate, action, horror, suspense)

## Errores comunes / que evitar

- **No elegir el modelo correcto de imagen:** principiantes usan cualquier modelo y obtienen resultados promedio. Usa Nano Banana 2 para realismo, Sea Dream 5.0 para artístico.
- **Modificar prompt y regenerar esperando cambios mínimos:** la IA cambia cosas que te gustaban. Usa **Outfit Swap** o **Shots** en su lugar.
- **Usar menos de 20 fotos de referencia para personaje:** el resultado será inconsistente. Mínimo 20-30 fotos con diferentes ángulos.
- **No usar Cinema Studio para look profesional:** sin preset de cámara y lente, el resultado se ve genérico.
- **Ignorar el género/emoción en video:** sin seleccionar Genre, el video no transmite la atmósfera deseada.
- **No usar Grid antes de generar:** desperdicias créditos probando variaciones que podrías previsualizar gratis.
- **Usar transiciones sin elegir preset:** la transición por defecto puede no ser la adecuada. Prueba Raven para look cinematográfico.
- **No upscalar las imágenes de Shots antes de usarlas para entrenar personaje:** baja calidad afecta la consistencia final.
