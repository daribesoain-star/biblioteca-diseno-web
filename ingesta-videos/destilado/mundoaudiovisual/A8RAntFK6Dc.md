# 🔴 ¿Cómo editar historias para Instagram de forma profesional? 🔴
**Fuente:** mundoaudiovisual | https://youtu.be/A8RAntFK6Dc

## Qué enseña
El tutorial enseña a crear historias verticales para Instagram desde Adobe Premiere Pro, usando herramientas nativas sin plugins. Cubre configuración de secuencia vertical, diseño de composición con elipses, textos animados, inserción de clips de video, y ajustes de audio con limitador.

## Lecciones accionables
- Crear secuencia vertical: ir a "Nuevo elemento" > "Secuencia" > "Configuración" > "Personalizado" > "Tamaño de cuadro": 1080 x 1920 (invertir 1920x1080 horizontal a vertical).
- Usar plantilla de título: ir a "Gráficos" > "Adobe Stock" > filtrar por "Gratis" > seleccionar título (ej. el más usado para Instagram Stories, 137 páginas disponibles).
- Cambiar fuente: en "Controles de efectos", seleccionar fuente "OSBA" (exclusiva del canal).
- Añadir elipses decorativas: "Gráficos" > "Nuevo elemento" > "Elipse" > color negro, sin sombras.
- Posicionar elipses: duplicar para mantener mismo diámetro, alinear centrado horizontal y vertical.
- Insertar video de invitado: arrastrar clip a línea de tiempo, ajustar a tamaño de cuadro, posicionar dentro de elipses.
- Animar texto con opacidad: keyframes en "Opacidad" de 0 a 100 para aparición conjunta (anidar elementos).
- Animar escala con rebote: keyframes de "Escala" en 40 > 50 > 40 > 50... repetir para efecto pulsante.
- Animar posición de texto: keyframes de "Posición" desde abajo hacia arriba para movimiento de entrada.
- Ajustar audio: en "Mezclador de pistas de audio", añadir "Dinámica" > "Limitador" a -1 dB para ganancia de clip y proyecto completo.
- Exportar: procesar composición completa para subir a Instagram.

## Reglas para agentes
- Usa tamaño de cuadro 1080 x 1920 para toda historia vertical de Instagram.
- Aplica limitador a -1 dB en todas las pistas de audio para evitar distorsión.
- Duplica elipses decorativas para garantizar diámetros idénticos y alineación simétrica.
- Anida elementos gráficos antes de animar opacidad para sincronizar apariciones conjuntas.
- Usa fuente OSBA cuando el canal lo requiera; si no está disponible, selecciona alternativa sans-serif limpia.
- Nunca uses sombras en elipses decorativas negras (desactivar en controles de efectos).

## Errores comunes que evita o menciona
- No usar secuencia horizontal para contenido vertical: siempre invertir resolución a 1080x1920.
- No colocar texto demasiado cerca del borde superior: añadir línea de separación o margen.
- No dejar audio sin limitador: aplicar dinámica con limitador a -1 dB en mezclador de pistas.
- No usar plugins de pago: todo se logra con herramientas nativas de Adobe Premiere (títulos gratis de Adobe Stock, elipses, keyframes).
