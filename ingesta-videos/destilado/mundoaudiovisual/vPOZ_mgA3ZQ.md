# DJI MINI 3 PRO 🔴 VUELOS INTELIGENTES 🔴 ¿Cómo grabarlos y Cómo editarlos?
**Fuente:** mundoaudiovisual | https://youtu.be/vPOZ_mgA3ZQ

## Qué enseña
El video enseña a grabar correctamente los modos de vuelo inteligente (QuickShots) del DJI Mini 3 Pro y a editarlos en DaVinci Resolve para obtener resultados cinematográficos. Cubre desde la configuración de cámara hasta técnicas de edición como speed ramps y corrección de color.

## Lecciones accionables
- **Configuración de cámara para QuickShots:**
  - Usar modo Pro, 30 fps, velocidad de obturación 1/60 (regla del doble de fps).
  - Balance de blancos manual: 5700K en día soleado, 5000-5400K en día nublado.
  - ISO en automático para compensar cambios de luz.
  - Codec H.265 en formato MP4 (Windows) o MOV (Mac).
  - Usar filtro ND para mantener obturación correcta.
- **Selección de ubicación:** Elegir estructuras alargadas (puentes, carreteras) para que los QuickShots luzcan más dinámicos, especialmente el modo espiral.
- **Grabación de QuickShots:**
  - Activar QuickShots desde el menú superior del controlador.
  - Presionar símbolo "+" o dibujar un cuadrado con dos dedos para detectar sujeto.
  - Configurar distancia máxima (120m en droning, 80m en cohete, 120m en espiral).
  - Ocultar el controlador detrás del cuerpo durante la grabación.
- **Mejora de tomas:** Mover al sujeto durante la grabación (caminar, desplazarse) para obtener resultados más cinematográficos que con sujeto estático.
- **Edición en DaVinci Resolve:**
  - Crear proyecto en 4K, 30 fps (nativo del dron).
  - Corrección de color en módulo Color: bajar altas luces, subir sombras, ajustar curvas, añadir saturación y realce de color (sin exagerar).
  - Speed ramp: presionar Control+R para mostrar triángulos de velocidad, añadir "add speed point" al inicio y final del tramo a acelerar.
  - Ajustar velocidad al 400-700% para aceleración.
  - Convertir keyframes a curva (Caps Lock + seleccionar keyframe, convertirlo en curva) para transición orgánica.
  - Abrir frame interpolation (Command+Shift+C en Mac) para ver curva de velocidad.
  - Pegar ajustes de color entre clips arrastrando desde clip corregido al destino.
  - Añadir efectos de sonido tipo "whoosh" sincronizados con los cambios de velocidad (descargar de artlist.io).

## Reglas para agentes
- Usa balance de blancos manual siempre, nunca automático, para evitar cambios bruscos de color.
- Configura la velocidad de obturación al doble de los fps (ej. 30fps → 1/60) usando filtro ND.
- Selecciona ubicaciones con estructuras alargadas cuando uses el modo espiral.
- Mueve al sujeto durante la grabación de QuickShots para mejorar el resultado visual.
- En speed ramps, convierte los keyframes a curvas para transiciones orgánicas, nunca uses cambios abruptos de velocidad.
- Usa H.265 en MP4 para Windows, H.265 en MOV para Mac.
- Genera archivos optimizados en DaVinci Resolve antes de previsualizar speed ramps.

## Errores comunes que evita o menciona
- **No usar balance de blancos automático:** Provoca cambios de color indeseados al variar la luz.
- **No grabar con sujeto estático:** Las tomas con movimiento del sujeto son mucho más cinematográficas.
- **No usar speed ramps abruptos:** Las transiciones deben ser curvas progresivas, no cambios instantáneos de velocidad.
- **No usar el modo órbita en movimiento:** El dron se desorienta y hace movimientos extraños; funciona mejor con sujeto estático.
- **No olvidar ocultar el controlador:** Debe estar detrás del cuerpo para no aparecer en el encuadre.
- **No usar la configuración nativa del proyecto:** Ajustar el proyecto a 4K 30fps para aprovechar la resolución del dron.
