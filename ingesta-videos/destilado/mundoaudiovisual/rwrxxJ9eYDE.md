# Curso de Motion 5 tutorial en español - 🎬 * Capítulo 1 - Primeros pasos
**Fuente:** mundoaudiovisual | https://youtu.be/rwrxxJ9eYDE

## Qué enseña (2-3 líneas)
Introducción a Motion 5 como editor de efectos (no editor de video) y plugin para Final Cut Pro, Premiere y otros editores. Explica la pantalla de inicio, la interfaz principal (Librería, Inspector, Canvas, Línea de tiempo), creación de grupos, importación de elementos, herramientas de transformación 2D/3D y primeros pasos con formas y capas.

## Lecciones accionables
- Para abrir Motion: hacer clic en el icono del programa (identificado por su icono 3D).
- En la pantalla de inicio, elegir "Blank project" (proyecto en blanco) para empezar desde cero.
- Configurar preset: seleccionar resolución (ej. 1080p), 30 fps, duración 10 segundos en timecode.
- Para crear formas: usar el botón de formas (pen icon) → seleccionar rectángulo, círculo o línea. Mantener Shift presionado mientras se dibuja para mantener proporciones perfectas (círculo perfecto, línea perpendicular/horizontal).
- Para renombrar un grupo: hacer doble clic sobre el nombre del grupo en el panel de proyectos.
- Para crear un nuevo grupo: clic derecho → "New Group".
- Para importar contenido: arrastrar archivo al grupo deseado en el panel de proyectos.
- Para escalar sin perder proporciones: seleccionar el elemento, mantener Shift presionado mientras se arrastra el controlador de escala.
- Para rotar en 3D: activar el botón de rotación 3D (icono de cubo) → usar los círculos de rotación en el canvas: círculo rojo para rotación en X (arriba/abajo), círculo verde para Y (izquierda/derecha), círculo azul para Z (perpendicular).
- Para rotar con precisión: colocar el cursor sobre el valor numérico de rotación en el Inspector y usar flecha arriba/abajo para movimiento lento, o Shift + flecha arriba/abajo para movimiento rápido.
- Para resetear rotaciones a cero: hacer clic en el botón de reset junto al parámetro en el Inspector.
- Para ver reglas, cuadrículas o zonas de seguridad: menú de visualización en el canvas → activar/desactivar opciones.
- Para cambiar vista del canvas: usar el menú desplegable de visualización (100%, ajustar a pantalla, etc.).
- Para mover el canvas: usar la herramienta mano (hand tool) o el botón "adjust" para centrar la vista.
- Para grabar keyframes manualmente: presionar el botón rojo de grabación de keyframes → mover elementos en el canvas → presionar play para ver la animación grabada.
- Para eliminar elementos del canvas: seleccionar y presionar Delete.
- Para eliminar grupos: clic derecho sobre el grupo → "Delete".

## Reglas para agentes
- Usa "Blank project" cuando inicies un efecto desde cero; no uses plantillas predefinidas a menos que quieras estudiar cómo fueron creadas.
- Usa Shift al dibujar círculos o líneas para mantener proporciones exactas.
- Usa Shift al escalar para mantener la relación de aspecto original del elemento.
- Usa el botón de rotación 3D cuando trabajes con rotaciones en los ejes X, Y o Z; no uses solo el Inspector 2D para rotaciones tridimensionales.
- Nunca trabajes directamente sobre el grupo por defecto "Group 1" sin renombrarlo; renómbralo para mantener organización (ej. "Shapes", "Mundo Audiovisual").
- Nunca asumas que el software es un editor de video; úsalo exclusivamente para crear efectos, títulos, transiciones y generadores.
- Verifica siempre que el grupo correcto esté seleccionado antes de importar contenido; el contenido se añade al grupo activo.

## Errores comunes que evita o menciona
- Confundir Motion con un editor de video: aclara que es un editor de efectos, no para editar películas completas.
- Olvidar que los proyectos de Motion se llaman "Compositions" (Composiciones), no proyectos.
- No usar Shift al escalar, lo que distorsiona las proporciones del elemento.
- No usar Shift al dibujar círculos, resultando en óvalos desproporcionados.
- No renombrar grupos, lo que dificulta la organización cuando hay múltiples capas.
- No seleccionar el grupo correcto antes de importar, causando que el contenido se añada al grupo equivocado.
- No usar el botón de rotación 3D para rotaciones tridimensionales, limitándose solo a rotaciones 2D.
