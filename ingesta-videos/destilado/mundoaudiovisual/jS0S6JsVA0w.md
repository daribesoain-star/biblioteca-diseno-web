# 📸 Curso Gratis de Adobe Photoshop - Módulo 2 - Capítulo 3 - Las herramientas de selección.
**Fuente:** mundoaudiovisual | https://youtu.be/jS0S6JsVA0w

## Qué enseña (2-3 líneas)
Este capítulo explica todas las herramientas de selección en Photoshop: marco rectangular, marco elíptico, lazo, lazo poligonal, lazo magnético, selección de objeto, selección rápida y varita mágica. Muestra cómo cada una delimita áreas específicas de una capa para aplicar ediciones, y enseña atajos de teclado para crear, invertir y eliminar selecciones.

## Lecciones accionables
- **Herramienta Marco Rectangular (Rectangular Marquee Tool):** Selecciona un área rectangular. Para modificar solo esa zona, usa la herramienta Borrador o cualquier ajuste. Atajo para rellenar selección con color frontal: `Command + Delete` (Mac) / `Ctrl + Delete` (Windows). Para recuperar la última selección: `Shift + Command + Delete` (Mac) / `Shift + Ctrl + Delete` (Windows).
- **Herramienta Marco Elíptico (Elliptical Marquee Tool):** Selecciona áreas elípticas. Mantén presionada la tecla `Shift` mientras arrastras para seleccionar un círculo perfecto.
- **Herramienta Lazo (Lasso Tool):** Selecciona a mano alzada cualquier área. Deshacer selección: `Control + Z` (Mac) / `Ctrl + Z` (Windows).
- **Herramienta Lazo Poligonal (Polygonal Lasso Tool):** Haz clic en puntos para crear una selección con bordes rectos. Ejemplo: seleccionar el ojo de una persona haciendo clic alrededor del contorno. Luego aplica un ajuste de color (ej: reemplazo de color) que solo afectará el área seleccionada.
- **Herramienta Selección de Objeto (Object Selection Tool):** Modo Lasso o Rectángulo. Dibuja un recuadro o lazo alrededor del objeto deseado; Photoshop lo detecta automáticamente. Funciona mejor con alto contraste. También usa el botón **"Select Subject"** en la barra de opciones para seleccionar automáticamente el sujeto principal de la imagen.
- **Herramienta Selección Rápida (Quick Selection Tool):** Pinta sobre el área que deseas seleccionar. Ajusta el tamaño del pincel en la barra de opciones. Usa los botones **"Añadir a selección"** (+) y **"Restar de selección"** (-) en la barra superior. Atajo: mantén presionada la tecla `Option` (Mac) / `Alt` (Windows) para restar. Para eliminar el fondo, invierte la selección: `Seleccionar > Invertir` y luego presiona `Suprimir`.
- **Herramienta Varita Mágica (Magic Wand Tool):** Selecciona píxeles de color similar con un solo clic. Parámetro clave: **Tolerancia** (valor por defecto 32). A mayor tolerancia, selecciona más tonos similares. Para seleccionar múltiples áreas, mantén presionada la tecla `Shift` y haz clic en cada zona. Para eliminar el fondo seleccionado, haz clic derecho sobre la capa y elige **"Rasterizar capa"** (Rasterize Layer) antes de borrar.
- **Herramienta Lazo Magnético (Magnetic Lasso Tool):** Aparece en el espacio de trabajo **Essentials** (Ventana > Espacios de trabajo > Essentials). Se adhiere automáticamente a los bordes del objeto mientras arrastras el cursor a lo largo del contorno.

## Reglas para agentes
- Usa `Command + Delete` (Mac) / `Ctrl + Delete` (Windows) para rellenar la selección con el color frontal; usa `Shift + Command + Delete` (Mac) / `Shift + Ctrl + Delete` (Windows) para recuperar la última selección.
- Mantén presionada `Shift` al usar Marco Elíptico o Varita Mágica para añadir a la selección existente.
- Mantén presionada `Option` (Mac) / `Alt` (Windows) al usar Selección Rápida para restar áreas de la selección.
- Usa el botón **"Select Subject"** en la barra de opciones de Selección de Objeto para seleccionar automáticamente el sujeto principal.
- Rasteriza la capa (clic derecho > Rasterizar capa) antes de eliminar el fondo seleccionado con Varita Mágica o Selección Rápida, si la capa no es un objeto inteligente.
- Cambia al espacio de trabajo **Essentials** (Ventana > Espacios de trabajo > Essentials) para que aparezca la herramienta Lazo Magnético.

## Errores comunes que evita o menciona
- No se puede eliminar directamente el fondo de una capa que no sea un objeto inteligente o no esté rasterizada; primero hay que rasterizarla.
- La herramienta Selección de Objeto funciona mejor con alto contraste entre el objeto y el fondo; si el contraste es bajo, la selección será imprecisa.
- La Varita Mágica con tolerancia baja (ej: 32) selecciona solo tonos muy similares; aumentar la tolerancia (ej: 100) selecciona un rango más amplio de colores.
- Al usar el Lazo Poligonal, los clics deben ser precisos para evitar selecciones con bordes irregulares.
