# DJI Mini 3 Pro - GRABA Y EDITA vídeos ÉPICOS - Parte 1
**Fuente:** mundoaudiovisual | https://youtu.be/Y8kxOTiUpPs

## Qué enseña
Cómo grabar tomas aéreas épicas con el DJI Mini 3 Pro en formato horizontal y vertical (para Instagram Reels, TikTok), y cómo preparar el material para una edición profesional en Adobe Premiere con configuración de doble monitor. Incluye técnicas de vuelo manual y uso de modos inteligentes.

## Lecciones accionables
- Esperar a tener **8 satélites** y el **punto de retorno a casa (RTH) actualizado** antes de despegar.
- Para la **toma de línea costera**: subir el dron a **100 metros**, activar **modo Cine** y grabar avanzando hacia la costa.
- Para **sobrevolar una carretera**: volar a **28-30 metros** de altura para dar sensación de mayor velocidad.
- Para **órbitas manuales**: usar **modo Normal**, mover el joystick **muy lentamente hacia la izquierda** mientras se rota el dron suavemente para mantener el objetivo centrado.
- Para **vuelo descendente con avance**: cambiar a **modo Normal**, descender y avanzar simultáneamente.
- Para **tomas de montaña o presa**: elevar el dron hasta encuadrar el paisaje, luego girar y grabar en **modo Cine**.
- Usar **modo Cine** para tomas suaves y lentas; **modo Normal** para maniobras más rápidas como órbitas.

## Reglas para agentes
- Usa **modo Cine** para tomas de paisaje amplio y movimientos lentos.
- Usa **modo Normal** para órbitas manuales y descensos con avance.
- Nunca despegues sin tener al menos **8 satélites** y el **punto RTH actualizado**.
- Nunca realices órbitas automáticas con **punto de interés** si hay obstáculos como torres o postes cerca; prefiere órbitas manuales.
- Siempre verifica que el dron esté estable antes de comenzar una toma, especialmente con viento.

## Errores comunes que evita o menciona
- No esperar suficientes satélites antes del despegue, lo que impide actualizar el punto de retorno a casa.
- Usar el modo de **punto de interés automático** cerca de torres o postes, que pueden interferir con el seguimiento.
- Volar demasiado alto al sobrevolar carreteras, perdiendo la sensación de velocidad.
- No cambiar de **modo Cine a Normal** según el tipo de maniobra, resultando en movimientos demasiado lentos para órbitas.
