# MAKING OFF - Así hacemos un TRABAJO AUDIOVISUAL CON DRONES
**Fuente:** mundoaudiovisual | https://youtu.be/0ot-mPoXKXE

## Qué enseña (2-3 líneas)
Muestra el proceso completo de producción audiovisual con drones en una propiedad privada: desde la obtención de permisos y planificación de vuelo hasta la ejecución coordinada de tomas estables (Mavic 3 / Mini 3 Pro) y tomas FPV de alto impacto. Explica la división de roles entre piloto de cámara estable y piloto FPV, y cómo se planifican las tomas in situ.

## Lecciones accionables
- Obtén autorización por escrito del propietario antes de volar en propiedad privada; no asumas que puedes grabar libremente.
- Divide roles en equipo: un piloto para tomas estables (Mavic 3 / Mini 3 Pro) y otro para tomas FPV (dron 3.5 pulgadas).
- Coordina altura y posición cuando dos drones vuelan simultáneamente para evitar colisiones.
- Usa el histograma para verificar exposición correcta: mantenlo centrado con información.
- Configura el dron en **modo cine** para órbitas lentas y suaves; cambia a **modo normal** para retrocesos y ascensos.
- Para tomas de aproximación frontal: ajusta el ángulo del gimbal a 0°, vuela recto sobre el objetivo.
- En órbitas: mantén velocidad muy baja, sin prisa; luego puedes añadir *speed ramps* en edición.
- Al retroceder: eleva el dron gradualmente mientras subes el gimbal lentamente, manteniendo el objetivo centrado.
- Usa el **teleobjetivo del Mavic 3** para efectos de paralaje.
- Para FPV: vuela entre árboles y caminos estrechos para tomas inmersivas; usa baterías más ligeras para mayor flotabilidad y estabilidad en vuelo invertido.
- Planifica las tomas FPV en los puntos más impactantes visualmente para captar la atención del espectador.
- El editor de video debe tener creatividad e imaginación para el montaje final; alterna roles de edición según el proyecto.

## Reglas para agentes
- Usa el histograma para verificar exposición antes de cada toma; nunca confíes solo en la vista previa.
- Coordina altura y posición con otros pilotos cuando vueles múltiples drones en el mismo espacio.
- Nunca vueles en propiedad privada sin autorización explícita por escrito del propietario.
- Activa el modo cine para órbitas y movimientos lentos; cambia a modo normal solo para desplazamientos más rápidos.
- Mantén el objetivo centrado en el encuadre durante retrocesos y ascensos.
- Verifica la batería antes de cada vuelo; aterriza cuando el dron indique batería baja.
- Para FPV: usa baterías más ligeras si buscas mayor flotabilidad y estabilidad en maniobras invertidas.

## Errores comunes que evita o menciona
- No asumir que se puede volar libremente en propiedad privada sin permisos.
- No apresurar las órbitas; la velocidad lenta permite añadir *speed ramps* en postproducción.
- No ignorar las alertas de batería baja; aterriza inmediatamente cuando aparezcan.
- No descuidar la detección de obstáculos: el Mini 3 Pro detecta obstáculos correctamente, pero hay que estar atento.
- No compartir públicamente tomas interiores de propiedades privadas sin consentimiento del propietario.
- No subestimar el riesgo de vuelo FPV entre árboles: pueden ocurrir choques y caídas (el dron quedó atrapado en un árbol, pero se recuperó).
