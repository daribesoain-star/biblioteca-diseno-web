# Curso GRATIS Final Cut Pro 11 - Capítulo 3: Las Herramientas - TRUCOS Y CONSEJOS
**Fuente:** mundoaudiovisual | https://youtu.be/B4S3IecL-S8

## Qué enseña
Este capítulo explica las herramientas básicas de Final Cut Pro 11 (selección, corte, posición, zoom, mano) y revela un flujo de trabajo profesional para editar clips eliminando silencios de forma ultrarrápida. El instructor enseña a personalizar atajos de teclado para cortar y acortar desde el inicio con un solo botón, logrando una edición fluida incluso con múltiples pistas sincronizadas.

## Lecciones accionables
- **Atajo para cortar:** Presiona **Command + B** para hacer un corte en la posición del cabezal reproductor.
- **Atajo para acortar desde el inicio (personalizado):** Ve a **Comandos > Personalizar**, busca la función **"Acortar desde el inicio"** y asígnale la tecla **` (acento grave, junto al 1)**. Guarda como copia del conjunto de comandos.
- **Flujo rápido para eliminar silencios:**
  1. Coloca el cabezal donde inicia el silencio.
  2. Presiona **Command + B** para cortar.
  3. Presiona la tecla **`** (acento grave) para acortar desde el inicio del clip seleccionado.
  4. Coloca el cabezal donde termina el silencio.
  5. Presiona **Command + B** para cortar.
  6. Presiona **Suprimir** para eliminar el silencio.
- **Edición con múltiples pistas (ej. cámara + captura de pantalla):**
  1. Selecciona ambos clips haciendo clic en el primero, luego **Mayús + clic** en el segundo.
  2. Coloca el cabezal donde inicia el silencio.
  3. Presiona **Command + B** (corta ambos clips simultáneamente).
  4. Presiona **`** (acorta desde el inicio en ambos).
  5. Repite: coloca cabezal, **Command + B**, **`**, **Suprimir**.
- **Herramienta Mano:** Presiona **H** para moverte por la línea de tiempo sin cambiar la selección.
- **Herramienta Seleccionar:** Presiona **A** para volver al modo selección.
- **Herramienta Corte (Trim):** Presiona **T** para ajustar el punto de corte entre dos clips arrastrando.
- **Herramienta Posición (Position):** Presiona **P** para mover un clip dejando un espacio vacío.
- **Herramienta Zoom:** Haz clic para acercar; **Opción + clic** para alejar.
- **Herramienta Selección de intervalo:** Arrastra para marcar un rango y luego presiona **Suprimir** para eliminarlo.

## Reglas para agentes
- Usa **Command + B** para cortar en lugar de la herramienta de tijera (B).
- Asigna la tecla **`** (acento grave) a "Acortar desde el inicio" para eliminar silencios en un solo paso.
- Cuando edites con múltiples pistas, selecciona **todos los clips sincronizados** antes de cortar.
- Usa **A** para seleccionar y **H** para moverte por la línea de tiempo; evita cambiar entre herramientas manualmente.
- Nunca uses la herramienta de tijera (B) para cortar: es más lenta que Command + B.
- Nunca combines clips de diferentes pistas sin seleccionarlos todos antes de cortar.

## Errores comunes que evita o menciona
- **No usar la herramienta de tijera (B):** Requiere cambiar entre herramientas (seleccionar, cortar, seleccionar de nuevo, eliminar), lo que multiplica los pasos. En su lugar, usa Command + B y la tecla personalizada `.
- **No cortar sin seleccionar todas las pistas:** Si trabajas con cámara + captura de pantalla y solo seleccionas una, el corte solo afecta a esa pista, desincronizando el material.
- **No personalizar atajos:** El flujo predeterminado de Final Cut Pro es lento; personalizar "Acortar desde el inicio" acelera drásticamente la edición.
- **Olvidar guardar como copia:** Al personalizar comandos, el sistema pide guardar como copia del conjunto de comandos; si no se hace, se pierde la personalización.
