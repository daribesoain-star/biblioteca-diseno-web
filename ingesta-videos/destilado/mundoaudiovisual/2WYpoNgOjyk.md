# 📸 Curso Gratis de Adobe Photoshop - Módulo 6 - Capítulo 1 - Fotografía de productos
**Fuente:** mundoaudiovisual | https://youtu.be/2WYpoNgOjyk

## Qué enseña
Cómo editar una fotografía de producto (perfume) tomada con luz casera (cocina) para lograr un resultado profesional. Cubre corrección de color con Camera Raw, eliminación de imperfecciones y creación de sombras realistas mediante transformación y desenfoque.

## Lecciones accionables
- **Aplicar filtro Camera Raw:** Capa → Filtro → Filtro RAW de Cámara. Ajustar Contraste, Intensidad y Saturación para recuperar el color del objeto sin sobresaturar. Reducir Negros si es necesario.
- **Eliminar fondo:** Usar herramienta de selección (no se especifica cuál) para enmascarar el objeto. Crear máscara de capa. Añadir capa nueva para fondo: Capa → Nueva → Capa (o arrastrar al icono "+").
- **Suavizar bordes de máscara:** Seleccionar máscara → Propiedades → Desvanecer (pluma) con valor bajo.
- **Eliminar imperfecciones y reflejos no deseados:** Herramienta Parche (J) → rodear el área defectuosa → arrastrar a zona limpia. Aplicar en reflejos rojos, suciedad o irregularidades.
- **Crear sombra tipo reflejo en superficie:**
    1. Duplicar capa del objeto: seleccionar capa → arrastrar al icono "+" en panel de capas.
    2. Capa inferior duplicada → Edición → Transformar → Perspectiva. Aplastar la base hacia abajo.
    3. Edición → Transformar → Distorsionar. Arrastrar puntos de esquina hacia arriba para inclinar la forma.
    4. Aplicar superposición de color negro: Capa → Estilo de capa → Superposición de color → color negro (#000000).
    5. Convertir a objeto inteligente: clic derecho en capa → Convertir en objeto inteligente.
    6. Filtro → Desenfocar → Desenfoque gaussiano. Radio suave (sin perder forma del perfume).
    7. Reducir opacidad de la capa de sombra (valor bajo).
    8. Opcional: con pincel negro suave, pintar en la base del objeto para reforzar sombra de contacto.
- **Ajustar composición:** Herramienta Recortar (C) para reencuadrar la imagen.

## Reglas para agentes
- Usa luz blanca homogénea de cocina para minimizar reflejos en objetos reflectantes (perfumes, vidrio, metal).
- Aplica Camera Raw antes de cualquier otra edición para corregir color y contraste base.
- Usa Herramienta Parche para eliminar reflejos no deseados, no para borrar sombras naturales.
- Crea sombras en capa separada debajo del objeto, nunca sobre la misma capa.
- Convierte la capa de sombra a objeto inteligente antes de aplicar desenfoque gaussiano para poder reajustar el filtro después.
- Reduce opacidad de la sombra para que se vea natural, no opaca.

## Errores comunes que evita o menciona
- Usar luz dura (bombilla directa) en objetos reflectantes genera reflejos no deseados; prefiere luz homogénea de cocina.
- Sobresaturar el color al corregir en Camera Raw; mejor ajustar Intensidad en lugar de Saturación.
- Dejar el objeto "volando" sin sombra; siempre añadir sombra de superficie para realismo.
- Hacer sombras demasiado opacas o con bordes duros; usar desenfoque gaussiano y opacidad baja.
