# 📸 Curso Gratis de Adobe Photoshop - Módulo 2 - Capítulo 11 - Sobreexponer, subexponer y esponja.
**Fuente:** mundoaudiovisual | https://youtu.be/7JkOlQbuKN8

## Qué enseña (2-3 líneas)
Explica el uso de las herramientas Sobreexponer (Dodge), Subexponer (Burn) y Esponja (Sponge) en Photoshop para ajustar exposición y saturación de forma localizada. Muestra cómo aplicar estos ajustes selectivamente en sombras, medios tonos e iluminaciones, y cómo modificar la saturación de áreas específicas.

## Lecciones accionables
- **Atajo de teclado:** Presiona la tecla `O` para acceder al conjunto de herramientas Sobreexponer, Subexponer y Esponja. Presiona `Shift + O` para alternar entre ellas.
- **Configurar Sobreexponer (Dodge Tool):** En la barra de opciones superior, selecciona el `Rango` (`Range`): `Sombras` (`Shadows`), `Medios tonos` (`Midtones`) o `Iluminaciones` (`Highlights`). Ajusta la `Exposición` (`Exposure`) al 50% para cambios sutiles. Usa un pincel con `Dureza` (`Hardness`) al 0% para transiciones suaves.
- **Aplicar Sobreexponer:** Pinta sobre áreas que quieras aclarar (ej. rostros, ojos, manos). Para iluminar zonas oscuras de ropa o fondo, selecciona el rango `Sombras` y pinta suavemente.
- **Configurar Subexponer (Burn Tool):** En la barra de opciones, elige el `Rango` (`Range`) y la `Exposición` (`Exposure`) al 50%. Usa un pincel grande con dureza mínima para oscurecer fondos o áreas que distraigan.
- **Aplicar Subexponer:** Pinta sobre áreas que quieras oscurecer (ej. fondo, suelo, edificios) para dirigir la atención hacia los sujetos iluminados. Evita sobreexponer para no generar grises no deseados.
- **Configurar Esponja (Sponge Tool):** En la barra de opciones, selecciona el `Modo` (`Mode`): `Saturar` (`Saturate`) para aumentar saturación o `Desaturar` (`Desaturate`) para reducirla. Ajusta el `Flujo` (`Flow`) alto (ej. 100%) para cambios rápidos.
- **Aplicar Desaturar:** Pinta sobre el fondo para volverlo blanco y negro, dejando los sujetos a color. Usa selecciones previas para mayor precisión.
- **Aplicar Saturar:** Pinta sobre áreas específicas (ej. rostros) para intensificar colores. Usa con moderación para evitar tonos anaranjados exagerados.

## Reglas para agentes
- Usa `Sobreexponer` con rango `Iluminaciones` cuando quieras realzar brillos en ojos o reflejos.
- Usa `Subexponer` con rango `Sombras` cuando necesites oscurecer fondos sin afectar sujetos iluminados.
- Ajusta la `Exposición` al 50% como valor predeterminado para cambios controlados; nunca superes el 50% sin verificar el resultado.
- Usa `Dureza` del pincel al 0% para bordes suaves y transiciones naturales.
- Usa `Esponja` en modo `Desaturar` para eliminar saturación de fondos y dirigir la atención al sujeto principal.
- Nunca apliques `Saturar` con flujo máximo sobre pieles, ya que genera colores anaranjados irreales.
- Siempre compara antes/después (tecla `Ctrl+Z` o panel Historia) para evaluar cambios sutiles.

## Errores comunes que evita o menciona
- **Sobreexposición excesiva:** Aplicar demasiada exposición con `Sobreexponer` o `Subexponer` genera grises planos y pérdida de detalle. Se recomienda usar 50% de exposición y capas de ajuste no destructivas.
- **Saturación exagerada en piel:** Usar `Esponja` en modo `Saturar` con flujo alto sobre rostros produce tonos anaranjados no naturales. Se debe aplicar con moderación.
- **Falta de sutileza:** Los cambios deben ser graduales; pintar con dureza alta o exposición alta crea parches evidentes. Se prefiere dureza 0% y exposición baja.
- **Uso excesivo de herramientas:** Se menciona que es mejor usar `Camera Raw` para ajustes globales y reservar estas herramientas para retoques localizados finos.
