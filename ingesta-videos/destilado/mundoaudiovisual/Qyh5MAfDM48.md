# DJI MINI 5 PRO
**Fuente:** mundoaudiovisual | https://youtu.be/Qyh5MAfDM48

## Qué enseña
Este video enseña a grabar y editar QuickShots en el DJI Mini 5 Pro (aplicable a todos los drones DJI), mostrando cómo usar cada modo de vuelo automatizado y cómo aplicar speed ramps en DaVinci Resolve para que los clips no sean aburridos y destaquen visualmente.

## Lecciones accionables
- Para activar QuickShots: ir a modo **Video** → seleccionar **QuickShot** en el menú.
- **Dronie**: seleccionar sujeto → ajustar distancia máxima a **120** → iniciar. El drone retrocede mientras asciende.
- **Rocket**: apuntar al sujeto → ajustar altura máxima → iniciar. Se recomienda **caminar mientras el drone ejecuta el QuickShot** para obtener un plano más dinámico.
- **Orbit**: seleccionar sujeto → elegir dirección (derecha/izquierda) → iniciar. Para un orbit más lento y controlado: salir de QuickShot → ir a **Video** → usar **Spotlight** o **Point of Interest** → presionar el botón derecho para orbitar a velocidad reducida.
- **Spiral**: seleccionar sujeto → ajustar altura máxima a **120** → iniciar mientras se camina. Es el favorito del creador.
- **Boomerang**: seleccionar sujeto → iniciar. El drone traza una trayectoria de ida y vuelta.
- **Asteroid**: apuntar al sujeto → iniciar. El drone se aleja, toma fotos panorámicas y genera un efecto de globo terráqueo al regresar.
- Para editar en **DaVinci Resolve**:
  - Aplicar corrección de color: ir al módulo **Color** → añadir **Saturation Matrix** → saturar áreas específicas y ajustar contraste/sombras.
  - Crear speed ramp: presionar **Shift+R** → añadir **velocity points** en el timeline → comprimir la sección entre puntos para acelerar.
  - Hacer la transición orgánica: abrir **keyframes** → en **speed processing** → convertir puntos a **curva**.
  - Usar clips grabados a **50 fps** para permitir **cámara lenta** (reducir velocidad a 50%) antes del speed ramp.
  - Sincronizar cortes y aceleraciones con el **ritmo de la música** (recomienda usar pistas de **Music Gallery** en mundoaudiovisual).

## Reglas para agentes
- Usa **QuickShots** solo si vas a editar con **speed ramps**; de lo contrario, los clips resultan aburridos.
- Siempre configura la **distancia/altura máxima** (120) antes de iniciar un QuickShot para evitar colisiones.
- Nunca uses el QuickShot **Rotación** si buscas un resultado profesional; es mejor hacerlo manualmente.
- Aplica **corrección de color** (saturación y contraste) a todos los clips antes de editar velocidad.
- Cuando uses **Dronie** o **Rocket**, camina mientras el drone ejecuta el plano para mayor dinamismo.
- Para **Orbit**, prefiere el modo **Point of Interest** en lugar del QuickShot si necesitas velocidad lenta y controlada.

## Errores comunes que evita o menciona
- No usar QuickShots sin editar: ver el clip completo sin aceleración es tedioso para el espectador.
- No olvidar que el drone regresa al punto de inicio después de cada QuickShot; hay que caminar de vuelta si te desplazaste.
- El QuickShot **Rotación** es básico y prefiere hacerse manualmente para mejor control.
- El **Asteroid** puede servir para un efecto puntual, pero no es el más versátil.
- No grabar a **50 fps** limita la posibilidad de usar cámara lenta en la edición.
- No sincronizar los speed ramps con la música hace que el video pierda impacto visual.
