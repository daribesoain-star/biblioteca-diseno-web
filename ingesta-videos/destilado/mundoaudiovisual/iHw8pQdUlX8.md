# CURSO DE FOTOGRAMETRÍA con el DJI Mini 3 Pro 🗺️ Capítulo 1
**Fuente:** mundoaudiovisual | https://youtu.be/iHw8pQdUlX8

## Qué enseña (2-3 líneas)
Introducción a la fotogrametría con el DJI Mini 3 Pro: qué es, para qué sirve (cartografía, ortofotografía, arquitectura) y qué software se necesita. Explica los conceptos de solapamiento entre fotografías, modelos 3D teselados, modelos digitales de elevación, curvas de nivel y mediciones de área/volumen. También cubre la instalación de DroneLink (versión beta para Android) y la adquisición de licencias de Metashape.

## Lecciones accionables
- **Definición de fotogrametría:** ciencia que estudia la forma, tamaño, posición y dimensiones de cualquier objeto mediante una serie de fotografías aéreas con solapamiento.
- **Solapamiento (overlap):** entre dos fotografías deben existir puntos repetidos para que el programa de edición estime la geometría 3D del terreno.
- **Geolocalización por coordenadas:** las fotos deben tener datos de coordenadas incrustados para obtener información precisa del terreno.
- **Medición de área y volumen en Metashape:**
  1. Dibujar un polígono sobre el área de interés.
  2. Hacer clic en "Aceptar" para fijar el polígono.
  3. Hacer clic en "Medir".
  4. Seleccionar "Calcular volumen".
  5. Leer resultados: perímetro 2D/3D, área en metros cuadrados y volumen en metros cúbicos.
- **Precisión de las mediciones:** aproximadamente 95% de exactitud respecto a la realidad.
- **Hardware requerido:**
  - DJI Mini 3 Pro con controlador RC1 táctil.
  - Controlador RCN1 (segunda mano desde €45-€75, buscar "DJI RP-N1" en Wallapop).
  - Teléfono Android (no iOS para la versión beta actual).
- **Instalación de DroneLink (versión beta):**
  1. Abrir Google Chrome en el teléfono Android.
  2. Escribir exactamente: "Drone League DJI 3 Pro".
  3. Seleccionar el primer resultado.
  4. Desplazarse hacia abajo y seleccionar "Drone League Download".
  5. Elegir "DJI 3 Pro drone".
  6. Hacer clic en "beta version".
  7. Desplazarse hasta "Drone Link DJI APK".
  8. Abrir con el instalador de paquetes.
  9. Si el teléfono bloquea la instalación: ir a Configuración > permitir instalación de aplicaciones.
  10. Volver atrás e instalar paquetes.
  11. Permitir acceso a ubicación.
  12. Iniciar sesión con cuenta de DroneLink.
- **Instalación de DJI Fly:**
  1. Ir al centro APK de DJI.
  2. Descargas > Descargar APK para Android.
  3. Instalar y registrarse con usuario.
- **Software necesario:**
  - **DroneLink:** plan Elite (pago único de $99.99) para mapas avanzados, misión mapping con cuadrícula y desbloqueo de ángulos de cámara. Plan Básico no permite crear mapas. Plan Premium permite mapas básicos.
  - **Metashape (Agisoft):** licencia estándar $179. Opción educativa: $54.91 (si eres estudiante universitario). Adquirir en Geovi (empresa española).
  - **Blender:** gratuito desde blender.org, para modelado 3D avanzado y creación de entornos.

## Reglas para agentes
- Usa el controlador RCN1 (con móvil Android) en lugar del RC1 táctil cuando necesites instalar aplicaciones de terceros como DroneLink.
- Usa la versión beta de DroneLink solo si tienes DJI Mini 3 Pro con RCN1; con RC Pro usa la versión oficial.
- Nunca instales DroneLink desde Google Play o Apple Store; descarga siempre el APK beta desde el sitio oficial de DroneLink.
- Usa un teléfono Android (no iOS) para la versión beta actual de DroneLink.
- Nunca uses el plan Básico de DroneLink si necesitas crear mapas; elige al menos el plan Premium o Elite.
- Usa el plan Elite de DroneLink ($99.99) para misiones profesionales con cuadrícula avanzada y control de inclinación del gimbal.
- Usa la licencia educativa de Metashape ($54.91) solo si eres estudiante universitario; de lo contrario, adquiere la licencia estándar ($166 en Geovi).

## Errores comunes que evita o menciona
- **Error:** pensar que el DJI Mini 3 Pro tiene sensor nativo de 48 megapíxeles. **Realidad:** el sensor es de 12 megapíxeles nativos; los 48 megapíxeles efectivos se logran mediante tecnología que multiplica cada píxel por 4.
- **Error:** intentar instalar aplicaciones de terceros en el controlador DJI RC1 táctil. **Solución:** usar el controlador RCN1 con un teléfono Android.
- **Error:** asumir que las mediciones fotogramétricas son 100% exactas. **Realidad:** tienen aproximadamente un 95% de precisión, lo cual es muy cercano a la realidad pero no absoluto.
- **Error:** descargar DroneLink desde tiendas oficiales de aplicaciones. **Solución:** descargar el APK beta directamente desde el sitio web oficial de DroneLink.
- **Error:** usar iPhone para la versión beta de DroneLink. **Realidad:** actualmente solo está disponible para Android; se necesita un teléfono Android para este flujo de trabajo.
