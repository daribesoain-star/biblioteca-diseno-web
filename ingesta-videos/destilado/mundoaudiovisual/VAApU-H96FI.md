# 📸 Curso Gratis de Adobe Photoshop - Módulo 2 - Capítulo 6 - Las herramientas de edición.
**Fuente:** mundoaudiovisual | https://youtu.be/VAApU-H96FI

## Qué enseña (2-3 líneas)
Este capítulo cubre las herramientas de edición de Photoshop: Pincel corrector puntual, Pincel corrector, Parche, Herramienta de movimiento con detección de contenido y Ojos rojos. También explica la Herramienta Tampón de clonar y Tampón de motivo, mostrando cómo eliminar imperfecciones, objetos no deseados y corregir texturas en retratos y paisajes.

## Lecciones accionables
- **Atajo de teclado para herramientas de edición:** Presiona la tecla `J` para acceder al grupo de herramientas (Pincel corrector puntual, Pincel corrector, Parche, Movimiento con detección de contenido, Ojos rojos). Mantén presionada la tecla `Shift` y presiona `J` repetidamente para alternar entre ellas.
- **Herramienta Parche:** Selecciona un área no deseada dibujando con el mouse. Arrastra la selección hacia una zona con la textura que deseas copiar. Suelta el mouse para que la textura de destino reemplace el área seleccionada. Usa `Ctrl+D` (Windows) o `Cmd+D` (Mac) para deseleccionar.
- **Pincel corrector puntual:** Pinta directamente sobre imperfecciones pequeñas (manchas, poros, vellos). El pincel fusiona automáticamente la textura y el color circundante. Ajusta el tamaño del pincel: en Windows, clic derecho y arrastra; en Mac, usa `Shift + ;` para agrandar y `Shift + ,` para reducir. Reduce la dureza (hardness) para transiciones más suaves.
- **Pincel corrector (Healing Brush):** Define manualmente un punto de origen manteniendo presionada la tecla `Alt` (Windows) u `Option` (Mac) y haciendo clic en la textura que deseas copiar. Luego pinta sobre el área a corregir. Útil para igualar tonos desiguales o eliminar reflejos localizados.
- **Herramienta de movimiento con detección de contenido:** Selecciona un objeto con la herramienta Lazo (o Lazo poligonal). En la barra de opciones, elige modo `Mover` (Move) para trasladar el objeto a otra ubicación, o modo `Extender` (Extend) para duplicarlo preservando el fondo original. Presiona `Enter` para aplicar.
- **Herramienta Ojos rojos:** Haz clic y arrastra un cuadrado alrededor del ojo con ojos rojos. Photoshop corrige automáticamente el tono rojizo.
- **Herramienta Tampón de clonar (Clone Stamp):** Atajo de teclado: `S`. Define un punto de origen presionando `Alt` (Windows) u `Option` (Mac) y haciendo clic. Luego pinta sobre el área a cubrir. Ajusta la dureza al mínimo para fusiones más naturales. Útil para eliminar objetos grandes (cables, botellas, pulseras) copiando texturas de áreas cercanas.
- **Herramienta Tampón de motivo (Pattern Stamp):** Selecciona un patrón predefinido desde la barra de opciones (ej. césped, agua, hojas). Pinta sobre el área para rellenarla con ese patrón repetitivo.

## Reglas para agentes
- Usa la **Herramienta Parche** cuando necesites eliminar objetos o imperfecciones grandes y quieras reemplazar la textura manualmente arrastrando la selección.
- Usa el **Pincel corrector puntual** para imperfecciones pequeñas y difusas (poros, manchas, vellos finos) sin necesidad de definir un punto de origen.
- Usa el **Pincel corrector** cuando necesites control exacto sobre la textura de origen, especialmente para igualar tonos desiguales en la piel.
- Usa la **Herramienta de movimiento con detección de contenido** en modo `Mover` para reubicar objetos y en modo `Extender` para duplicarlos manteniendo el fondo.
- Usa la **Herramienta Tampón de clonar** para eliminar objetos grandes o repetitivos (cables, botellas, accesorios) copiando texturas de áreas adyacentes con dureza mínima.
- Usa la **Herramienta Ojos rojos** exclusivamente para corregir el efecto de ojos rojos en fotografías con flash.
- Nunca uses el **Tampón de motivo** para correcciones realistas de texturas naturales; úsalo solo para efectos decorativos o rellenos de patrones repetitivos.
- Nunca uses el **Pincel corrector** sin definir primero un punto de origen con `Alt`/`Option`; de lo contrario, no funcionará correctamente.

## Errores comunes que evita o menciona
- **No definir punto de origen en el Pincel corrector:** Si no presionas `Alt`/`Option` para seleccionar la textura a copiar, la herramienta no funcionará como se espera.
- **Usar el Tampón de clonar con dureza al 100%:** Esto crea bordes duros y poco naturales. El instructor recomienda reducir la dureza al mínimo para fusiones más suaves.
- **Copiar texturas no deseadas (ej. cabello) al corregir la piel:** Si seleccionas un área con cabello como origen, el pincel corrector pegará cabello en la zona a corregir. Debes elegir cuidadosamente el punto de origen.
- **Sobreeditar la piel:** El instructor advierte que está exagerando las correcciones para el tutorial; en la práctica, no se deben eliminar todas las imperfecciones para mantener un aspecto natural.
- **No usar el modo correcto en Movimiento con detección de contenido:** Si usas `Mover` cuando deberías usar `Extender`, puedes perder el objeto original en lugar de duplicarlo.
