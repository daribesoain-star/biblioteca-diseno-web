# 📸 Curso Gratis de Adobe Photoshop - Módulo 3 - Capítulo 2 - Organizar las capas
**Fuente:** mundoaudiovisual | https://youtu.be/wXkS8FjLeBo

## Qué enseña (2-3 líneas)
Este capítulo enseña a crear, agrupar, enlazar, alinear y distribuir capas en Photoshop. También repasa conceptos básicos como el bloqueo de capas, la rasterización de texto y el uso de opacidad para efectos como sombras.

## Lecciones accionables
- **Desbloquear capa de fondo:** Haz clic en el candado (padlock) de la capa "Background" para convertirla en una capa normal. Para revertir: menú `Layer > New > Background from Layer`.
- **Crear nueva capa:** Usa `Shift + Ctrl + N` (Windows) o `Shift + Command + N` (Mac). En el cuadro de diálogo, asigna nombre, color, modo de fusión ("Normal") y opacidad ("100%").
- **Crear capa desde botón:** Haz clic en el ícono `+` (plus symbol) en la parte inferior del panel de capas.
- **Rasterizar capa de texto:** Haz clic derecho sobre la capa de texto y selecciona "Rasterize Layer" para convertir el texto editable en píxeles (ya no se podrá modificar el texto).
- **Enlazar capas:** Selecciona dos o más capas con `Shift`, haz clic derecho y elige "Link Layers". Al mover una, se mueven todas. Para desenlazar: clic derecho > "Unlink Layers".
- **Agrupar capas:** Selecciona las capas deseadas, haz clic derecho y elige "Group from Layers". Asigna un nombre al grupo (ej. "Audiovisual World Poster").
- **Mostrar/ocultar capas:** Usa el ícono del ojo (eye icon) a la izquierda de cada capa o grupo para alternar su visibilidad.
- **Ajustar opacidad:** Selecciona una capa y modifica el valor de "Opacity" en la parte superior del panel de capas (ej. reducir a 50% para sombras).
- **Eliminar capa:** Selecciona la capa y haz clic en el ícono del bote de basura (trash can) en la parte inferior del panel, o arrastra la capa al mismo.
- **Alinear capas:** Selecciona dos o más capas con `Shift`. En la barra de opciones superior, usa los botones de alineación: centrar en eje horizontal, centrar en eje vertical, alinear parte superior, inferior, izquierda o derecha.
- **Distribuir capas:** Con tres o más capas seleccionadas, usa los botones de distribución en la misma barra para espaciarlas uniformemente.

## Reglas para agentes
- Usa `Shift + Ctrl + N` (Windows) o `Shift + Command + N` (Mac) para crear una nueva capa desde el teclado.
- Usa el ícono `+` para crear una capa automáticamente; si un grupo está seleccionado, la capa se crea dentro del grupo.
- Rasteriza capas de texto solo cuando ya no necesites editar el texto; después de rasterizar, el texto se vuelve píxeles no editables.
- Enlaza capas cuando necesites moverlas o transformarlas juntas, pero manteniéndolas como capas separadas.
- Agrupa capas relacionadas (ej. texto + marco) para mantener el panel organizado y moverlas como una unidad.
- Ajusta la opacidad de una capa de sombra (ej. 50%) para que se vea natural; usa el borrador para corregir bordes.
- Alinea capas seleccionando al menos dos; distribuye capas seleccionando al menos tres.
- Nunca pintes directamente sobre una capa de texto sin rasterizarla primero, o Photoshop te pedirá rasterizar antes de continuar.

## Errores comunes que evita o menciona
- **No rasterizar texto antes de pintar sobre él:** Si intentas pintar sobre una capa de texto, Photoshop muestra un mensaje pidiendo rasterizar; hacerlo elimina la capacidad de editar el texto.
- **Confundir capa con elemento:** La capa no es solo el objeto (texto, forma), sino todo el lienzo; pintar fuera del objeto afecta el área transparente de la capa.
- **Olvidar que el orden de capas afecta la visibilidad:** Una capa superior cubre a las inferiores; mueve capas arrastrándolas en el panel para cambiar el orden.
- **No desagrupar antes de alinear:** Si alineas un grupo con otras capas, el grupo se mueve como unidad; para alinear elementos individuales, sácalos del grupo primero.
