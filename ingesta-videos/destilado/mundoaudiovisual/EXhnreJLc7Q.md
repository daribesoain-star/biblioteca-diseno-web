# Curso de edición de video - Davinci Resolve - 🎬 Capítulo 1 - Primeros Pasos
**Fuente:** mundoaudiovisual | https://youtu.be/EXhnREJLc7Q

## Qué enseña (2-3 líneas)
Familiarización con la interfaz de DaVinci Resolve, explicando la pantalla de inicio, la creación de bases de datos y proyectos, y la función de cada pestaña principal (Media, Edit, Fusion, Color, Fairlight, Delivery). Se enfoca en reducir la intimidación inicial y preparar al usuario para editar desde la pestaña Edit.

## Lecciones accionables
- Al abrir DaVinci Resolve, la primera pantalla muestra el gestor de proyectos. Para crear una base de datos, haz clic en "New Database" y especifica su ubicación (ej. en un disco duro externo o SD). Si la carpeta no está vacía, crea una nueva carpeta dentro de la ubicación deseada (ej. "Da Vinci Crucible Course Database") y selecciónala.
- Para crear un proyecto: dentro del gestor, haz clic en "Create Project", asígnale un nombre (ej. "test for courses") y presiona "Create". Se abrirá automáticamente la interfaz de edición.
- En la interfaz principal, identifica las pestañas superiores: Media (organización de clips), Edit (edición principal), Fusion (efectos visuales, equivalente a After Effects o Motion), Color (corrección de color), Fairlight (edición de audio, equivalente a Adobe Audition) y Delivery (exportación).
- Para ajustar la configuración del proyecto, haz clic en el ícono de engranaje (Project Settings) en la esquina inferior derecha. Allí configura la resolución (ej. 1920x1080) y los fotogramas por segundo (ej. 29.97 fps si grabas con cámara). Guarda los cambios con "Save".
- En la pestaña Edit, usa el panel "Media Pool" (panel izquierdo) para importar y organizar clips en carpetas. Si no ves el "Source Monitor" (visor de origen), es porque tu monitor es pequeño; puedes trabajar sin él o usar un monitor más grande.
- Usa el "Inspector" (panel derecho) para ver y modificar propiedades de video, audio, efectos y transiciones. Actívalo o desactívalo con el botón correspondiente.
- El control deslizante de volumen general del proyecto (parte inferior) permite bajar o subir el audio temporalmente sin modificar el proyecto. El botón que se vuelve amarillo reduce el volumen para escucha cómoda durante la edición.
- Para exportar, ve a la pestaña Delivery. Usa la configuración predeterminada "YouTube 1080p" para subir a YouTube en resolución 1080p.

## Reglas para agentes
- Usa la pestaña Edit para la mayoría de las tareas de edición de video, no la pestaña "Editing" (que es para edición rápida simple).
- Crea siempre una carpeta vacía nueva cuando crees una base de datos, no uses una carpeta existente con contenido.
- Configura la resolución y fps del proyecto en Project Settings (ícono de engranaje) antes de empezar a editar.
- Nunca modifiques el volumen del proyecto con el control deslizante general si necesitas cambios permanentes; usa el Inspector o la pista de audio.
- Usa la pestaña Delivery con el preset "YouTube 1080p" para exportar videos destinados a YouTube.

## Errores comunes que evita o menciona
- No confundir la pestaña "Editing" (edición rápida simple) con la pestaña "Edit" (edición completa de video). El curso se enfoca en la pestaña Edit.
- Si no ves el Source Monitor en pantallas pequeñas, no es un error; simplemente no hay espacio suficiente. No es necesario tenerlo para editar.
- Al crear una base de datos, no selecciones una carpeta que ya contenga archivos; DaVinci Resolve exigirá una carpeta vacía. Crea una nueva carpeta específica para la base de datos.
- No modifiques la configuración de proyecto (resolución, fps) después de haber empezado a editar, ya que puede causar problemas de compatibilidad con los clips.
