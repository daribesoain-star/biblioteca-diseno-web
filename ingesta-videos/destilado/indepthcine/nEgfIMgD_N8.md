# Why Camera Tests Are Important
**Fuente:** indepthcine | https://youtu.be/nEgfIMgD_N8
## Qué enseña (2-3 líneas)
Los camera tests convierten ideas visuales abstractas en decisiones técnicas verificadas antes del rodaje.  
Distingue entre prueba creativa de cámara y gear check técnico, y muestra cómo probar lentes, filtros, iluminación, formato, LUTs y flujo de postproducción para llegar al set con un look definido.

## Lecciones accionables
- Diferencia dos procesos: **camera test** = exploración creativa en preproducción para elegir equipo y lenguaje visual; **gear check** = verificación técnica uno o dos días antes del rodaje para confirmar que el equipo funciona.
- En un camera test, prueba combinaciones específicas de **cámara + lentes + filtros + iluminación + arte + vestuario + maquillaje** para ver cómo producen el look buscado.
- Evalúa lentes comparando **sharpness** de vidrio moderno contra el carácter de lentes vintage; prueba también cómo distintos filtros de difusión afectan los highlights.
- Si pruebas cámaras o film stocks, evalúa **dynamic range**, **colour reproduction**, **noise** y **grain structure**.
- Prueba ratios de iluminación variando fuerza de **key light** y **fill light** para definir cuánta sombra acepta o busca el director.
- Incluye pruebas de producción, maquillaje y vestuario para verificar cómo reaccionan colores, aplicaciones de maquillaje y telas bajo las condiciones reales de rodaje; revisa moiré o interferencias en telas con patrones sobre sensores digitales.
- Usa screen tests de actores cuando sirva para ver por primera vez cómo encarnan al personaje frente a cámara.
- En el gear check, el **focus puller** debe armar el paquete de cámara y hacer lens tests para confirmar calibración por focal length; también revisar que la cámara no tenga dead pixels excesivos.
- En el gear check, el **gaffer** debe energizar y revisar luces; el equipo de grip debe confirmar que todo esté completo y operativo.
- Si no hay presupuesto para días de prueba, intenta coordinar pruebas dentro del día de gear check o negocia con la rental house para testear equipo in-house si hay buena relación y disponibilidad.
- En proyectos largos como features o series, incluye un test day como line item de presupuesto y coordínalo con producción como parte del rental agreement.
- Antes del test, define el concepto visual y el equipo que quieres probar; no llegues a “ver todos los lentes del edificio”.
- Usa referencias de experiencia previa, gear tests online y recomendaciones de la rental house para acotar opciones.
- Consulta a la rental house: puede sugerir paquetes compatibles con presupuesto o equipos adecuados para el look buscado.
- En producciones high-end, considera tuning de lentes con técnicos especializados; ejemplo mencionado: **Dan Sasaki at Panavision**, ajustando efectos como flare colour o edge distortion.
- Al rodar cada prueba, usa slate y registra explícitamente: **lens**, **stop**, **filters** y **camera settings**.
- No dependas solo de lens charts: sirven para sharpness y distortion, pero no sustituyen una imagen realista.
- Graba pruebas con una persona en un espacio real, iluminado con film lights reales o con sol exterior, para evaluar piel, sensor y vidrio en condiciones útiles.
- Si el reto es de configuración, prueba la configuración física: para formato **9:16**, construye y prueba el vertical rigging de la cámara.
- Si necesitas zoom durante una toma, instala un **Microforce**, verifica que el zoom speed deseado sea posible y anota el speed setting.
- Lleva el test footage a una grading suite para comparar combinaciones técnicas y de color.
- Usa las pruebas para establecer el workflow de postproducción desde preproducción.
- Crea una **LUT / Look Up Table** a partir de las pruebas; puede hacerla un colorist, diseñarla el DP o ajustarse desde una LUT existente.
- Asegúrate de que la LUT esté correctamente construida para no subexponer ni sobreexponer en set porque la imagen monitorizada se vea demasiado oscura o demasiado clara.
- Monitorea en set con la LUT del proyecto para que director, productores y cliente vean una imagen cercana a la visión final.
- Aplica la misma LUT a rushes y transcoded proxies para el editor.
- Usa el final grade para ajustar y perfeccionar, no para empezar a descubrir cómo se ve la película.
- Crea un **frame leader** o **framing chart**: alinea físicamente las frame lines del aspect ratio en cámara y graba un clip de prueba que muestre esas líneas.
- Para frame lines digitales, usa herramientas como **Netflix frame line tools** o **Arri frame line tools**, ingresando cámara, shooting specs, squeeze factor y aspect ratio.
- El frame leader puede agregarse digitalmente al project file, imprimirse y filmarse como test clip, o marcarse con cinta en una tabla según las frame lines de la cámara.
- Entrega el frame leader al equipo de edición para comunicar el aspect ratio en que debe cortarse y finalizarse el proyecto.
- Pasa la LUT generada en las pruebas al colorist al final del proyecto como roadmap del intento creativo.

## Reglas para agentes
- Usa **camera test** cuando la decisión sea creativa: look, lentes, filtros, sensor, stock, color, ratios de luz, vestuario, maquillaje o lenguaje visual.
- Usa **gear check** cuando la decisión sea funcional: calibración, dead pixels, luces, grip, disponibilidad y estado operativo del equipo.
- Nunca confundas camera test con gear check: uno define el look; el otro confirma que el equipo elegido funciona.
- Antes de recomendar equipo, exige un concepto visual concreto y una lista acotada de elementos a probar.
- Nunca propongas probar “todo”; limita el test a opciones justificadas por guion, director, presupuesto o referencias previas.
- Usa slate en cada clip de prueba y registra lens, stop, filters y camera settings.
- Nunca evalúes look final solo con lens charts; incluye una persona, piel, espacio real y luz real.
- Usa pruebas de wardrobe y makeup cuando haya colores fuertes, texturas, patrones o riesgo de moiré.
- Usa pruebas de rigging cuando el formato físico cambie, especialmente 9:16 vertical.
- Usa Microforce y registra speed setting cuando el plano requiera zoom controlado.
- Crea o ajusta una LUT antes del rodaje y valida que no induzca errores de exposición.
- Monitorea con la LUT aprobada en set y aplícala a rushes/proxies para mantener coherencia visual.
- Usa frame leader o framing chart cuando el proyecto tenga aspect ratio específico, squeeze factor o frame lines críticas.
- Entrega LUT y frame leader a postproducción para que edición y color respeten el intento visual definido en prep.

## Errores comunes que evita o menciona
- Llegar al set sin un look definido y gastar tiempo caro experimentando.
- Descubrir demasiado tarde que el director quería otra cantidad de sombra, contraste, color o textura.
- Confundir una prueba creativa con una revisión técnica de funcionamiento.
- Probar demasiadas opciones sin criterio y no obtener conclusiones útiles.
- No slatear las pruebas y luego no saber qué clip corresponde a qué lente, stop, filtro o setting.
- Evaluar lentes solo con charts y no ver cómo responden con piel, espacio y luz real.
- Ignorar moiré o interferencias de telas con patrones en sensores digitales.
- No probar rigs físicos, como cámara vertical para 9:16, hasta el día de rodaje.
- No verificar si el zoom speed deseado es posible con el equipo disponible.
- Crear una LUT demasiado oscura o clara que provoque subexposición o sobreexposición en set.
- Dejar que el look se “descubra” recién en el final grade.
- No comunicar correctamente aspect ratio y frame lines al equipo de edición.
- No entregar al colorist la LUT de pruebas como guía del intento creativo.
