# The Most Common Optimization Mistakes that Advertisers Make
**Fuente:** JonLoomer | https://youtu.be/fmdhCPFOBEE

## Qué enseña (2-3 líneas)
Explica que los anunciantes confunden el objetivo de campaña con la optimización del conjunto de anuncios, y que optimizar para una acción incorrecta (como clics o vistas de página de destino) en lugar de la acción deseada (como compras) desperdicia presupuesto. Meta entrega anuncios según el "performance goal" del conjunto de anuncios, no según el objetivo de campaña.

## Lecciones accionables
- **Confusión objetivo vs. optimización:** El objetivo de campaña (ej. "Sales") solo determina las opciones disponibles en el conjunto de anuncios. Lo que realmente controla la entrega es el **performance goal** dentro del conjunto de anuncios (ubicado en "Conversion location" > "Performance goal").
- **Performance goal literal:** Si seleccionas "Landing page views", "Link clicks", "Daily unique reach" o "Impressions", Meta optimiza exclusivamente para esa métrica, ignorando compras o conversiones. No asume intención de compra aunque uses el objetivo "Sales".
- **Optimizar para la acción deseada:** Si quieres compras, debes seleccionar "Conversions" como performance goal y elegir "Purchase" como conversion event. No uses "Landing page views" ni "Link clicks" esperando que mantengan una tasa de conversión del 5%.
- **Bajo presupuesto:** Aunque sea caro o difícil salir de la fase de aprendizaje, prioriza optimizar para "Purchase" desde el inicio. No uses atajos como "Link clicks" a $0.20 esperando 5 compras por $20.
- **Verificación técnica:** En el conjunto de anuncios, ve a "Conversion location" > "Website" > "Performance goal" > "Conversions" > "Purchase" (o el evento de conversión deseado). No selecciones "Maximize number of link clicks" ni "Landing page views".
- **Objetivos intercambiables:** El performance goal "Landing page views" es idéntico bajo el objetivo "Sales" o "Traffic". No hay diferencia en la entrega.

## Reglas para agentes
- Usa "Conversions" como performance goal y "Purchase" como conversion event cuando quieras compras, incluso con bajo presupuesto.
- Nunca selecciones "Landing page views", "Link clicks", "Daily unique reach" o "Impressions" como performance goal si tu objetivo final son conversiones (compras, registros, etc.).
- Nunca asumas que el objetivo de campaña ("Sales", "Traffic") determina la optimización de entrega; verifica siempre el performance goal del conjunto de anuncios.
- Cuando configures un conjunto de anuncios, primero define el conversion event exacto (ej. "Purchase") antes de ajustar el presupuesto o la audiencia.
- Si el rendimiento es bajo, revisa el performance goal antes de cambiar el objetivo de campaña.

## Errores comunes que evita o menciona
- **Confundir objetivo con optimización:** Creer que seleccionar "Sales" como objetivo automáticamente optimiza para compras, cuando el performance goal puede ser "Landing page views" y Meta solo buscará vistas, no conversiones.
- **Optimizar para la acción incorrecta:** Usar "Landing page views" o "Link clicks" esperando que el 5% de tráfico convierta a compras, ignorando que Meta entrega tráfico de baja calidad (clics accidentales, bots, fraude) porque solo optimiza para esa métrica.
- **Asumir que el tráfico de anuncios es igual al tráfico orgánico:** Pensar que mantener una tasa de conversión del 5% con clics a $0.20 es viable, cuando el algoritmo no prioriza calidad si el performance goal no es "Conversions".
- **Usar el objetivo "Traffic" con performance goal "Conversions":** Aunque es posible, no hay diferencia en entrega vs. "Sales" con el mismo performance goal; el error es no alinear el performance goal con la acción deseada.
