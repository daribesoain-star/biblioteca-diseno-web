# Is Your Campaign Objective Irrelevant?
**Fuente:** JonLoomer | https://youtu.be/g7bthW7eJbA

## Qué enseña
El objetivo de campaña en Meta Ads es mayormente decorativo; lo que realmente dirige el algoritmo es el *performance goal* (objetivo de rendimiento). Cambiar la terminología de "campaña de ventas" a "optimizando para compra" evita confusiones y enfoca correctamente la estrategia.

## Lecciones accionables
- **Selecciona el *performance goal* correcto, no el objetivo de campaña**: el algoritmo solo optimiza para el evento que elijas en "Optimización para entrega de anuncios" (ej. *Purchase*, *Landing Page Views*, *Link Clicks*), sin importar si el objetivo de campaña es *Sales*, *Traffic* o *Engagement*.
- **Usa objetivo *Sales* para acceder a segmentos de audiencia de reporting**: aunque optimices para un evento distinto a *Purchase* (ej. *Complete Registration* para lead magnets), el objetivo *Sales* desbloquea informes de audiencia que otros objetivos no ofrecen.
- **Aprovecha objetivo *Engagement* para tráfico de calidad**: configura *Website Conversion Location* y optimiza para eventos personalizados como *Time Spent* o *Scroll Depth* en lugar de *Link Clicks* o *Landing Page Views* para obtener visitas más valiosas a tu sitio.
- **Cambia tu lenguaje**: en lugar de decir "estoy ejecutando una campaña de ventas", di "estoy optimizando para *Purchase*" o "estoy optimizando para *Landing Page Views*". Esto clarifica la estrategia y educa a otros anunciantes.
- **Verifica los *performance goals* disponibles por objetivo**: aunque ciertos objetivos de campaña restringen opciones, muchos *performance goals* (ej. *Landing Page Views*) están disponibles en múltiples objetivos (*Traffic*, *Sales*, *Leads*). El algoritmo no diferencia entre ellos.

## Reglas para agentes
- **Usa el *performance goal* como referencia principal** al analizar una campaña, no el objetivo de campaña.
- **Nunca asumas que un objetivo *Sales* implica optimización para *Purchase***; verifica siempre el evento de conversión seleccionado en el nivel de conjunto de anuncios.
- **Cuando reportes resultados, especifica el evento de optimización** (ej. "campaña optimizada para *Add to Cart*") en lugar del objetivo de campaña.
- **Si buscas tráfico de calidad, evita *Link Clicks* y *Landing Page Views***; usa eventos personalizados (*Time Spent*, *Scroll Depth*) dentro del objetivo *Engagement* con ubicación *Website*.

## Errores comunes que evita o menciona
- **Creer que el objetivo de campaña *Sales* hace que el algoritmo busque compradores**: el algoritmo solo responde al *performance goal*; si optimizas para *Landing Page Views*, ignorará intenciones de compra.
- **Llamar "campaña de ventas" a una que optimiza para *Add to Cart* o *Initiate Checkout***: esto confunde a clientes y colegas, y diluye el enfoque estratégico.
- **Usar *Engagement* solo para interacciones superficiales**: se puede usar para conversiones web de calidad si se configura correctamente (*Website Conversion Location* + evento personalizado).
- **Asumir que *Traffic* es la única opción para obtener visitas**: *Sales* y *Leads* también permiten optimizar para *Landing Page Views* o *Link Clicks*, con beneficios adicionales de reporting.
