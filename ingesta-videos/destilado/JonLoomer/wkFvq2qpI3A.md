# Why Your Performance Goal Changes Everything
**Fuente:** JonLoomer | https://youtu.be/wkFvq2qpI3A

## Qué enseña (2-3 líneas)
El performance goal en Meta Ads define literalmente qué acción quiere el anunciante y el algoritmo optimiza para conseguir esa acción exacta, actuando como el principal mecanismo de segmentación. Cambiar el performance goal transforma por completo a qué audiencias se entregan los anuncios, incluso sin modificar la segmentación explícita, y puede generar tráfico de baja calidad si se elige una meta incorrecta.

## Lecciones accionables
- **Selecciona "Maximize number of conversions" con el evento "Purchase"** siempre que sea posible para obtener tráfico de alta calidad; evita optimizar para clics baratos o vistas.
- **Para leads, define un límite de edad máximo (age maximum)** en el conjunto de anuncios cuando optimices para "Maximize conversions" con evento "Lead", porque el algoritmo tiende a sobreexplotar al grupo de 65+ años que completa formularios con alta frecuencia.
- **Elimina la colocación "Audience Network"** si usas performance goal de "Maximize link clicks", ya que genera clics accidentales, fraudulentos o de bots que inflan métricas sin valor real.
- **Cambia el conversion event dentro de "Maximize conversions"** según el objetivo real: si pasas de "Purchase" a "Lead", la audiencia cambia automáticamente (menos remarketing que con Purchase, pero más que con link clicks).
- **Usa remarketing como efecto colateral esperado**: al optimizar para "Purchase", el algoritmo destinará un porcentaje significativo del presupuesto a clientes previos, lista de emails, visitantes del sitio y personas que interactuaron con anuncios anteriores.
- **Consulta la guía completa en johnloomer.com/inflated** para un roadmap detallado sobre cómo abordar resultados inflados según cada performance goal.

## Reglas para agentes
- Usa "Maximize number of conversions" con evento "Purchase" cuando el objetivo final sea una venta; nunca uses "Maximize link clicks" como atajo para conseguir ventas baratas.
- Cuando optimices para "Lead", establece siempre un age maximum (ej. 64) para evitar que el algoritmo concentre el presupuesto en mayores de 65 años.
- Si usas "Maximize link clicks", desactiva manualmente la colocación "Audience Network" para reducir clics de baja calidad.
- Nunca selecciones performance goals de etapas intermedias (add to cart, initiate checkout, view content) esperando que funcionen como proxy para ventas; el algoritmo no considera calidad.
- Trata el performance goal como tu principal herramienta de segmentación algorítmica: cambiarlo redefine la audiencia sin tocar targeting explícito.

## Errores comunes que evita o menciona
- **Optimizar para clics baratos o vistas** pensando que llevará a ventas baratas: el algoritmo encuentra clics accidentales, bots y spam, no compradores reales.
- **Usar eventos intermedios (add to cart, initiate checkout, view content)** como sustituto de "Purchase": es extremadamente improbable que funcione y genera resultados de baja calidad.
- **Quejarse de resultados de baja calidad** cuando se usa un performance goal que no considera calidad (ej. link clicks, video views).
- **No ajustar la edad máxima al optimizar para leads**: el algoritmo explota al grupo 65+ que completa formularios, elevando el gasto en ese segmento sin control.
- **Confiar en que eliminar Audience Network resuelve todo** al optimizar para clics: el algoritmo sigue buscando clics baratos en otros lugares (personas que clickean todo, bots).
