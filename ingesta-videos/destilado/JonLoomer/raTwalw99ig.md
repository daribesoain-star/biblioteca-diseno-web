# Does Manual Bidding Actually Work?
**Fuente:** JonLoomer | https://youtu.be/raTwalw99ig

## Qué enseña (2-3 líneas)
JonLoomer explica por qué la puja manual (cost cap o bid cap) rara vez supera a la optimización automática de Meta, ya que la plataforma prioriza gastar todo el presupuesto incluso si eso eleva los costos. Señala que fijar un costo objetivo puede restringir la entrega a inventarios baratos (ej. Audience Network) y generar resultados inconsistentes, especialmente si no se optimiza para compras.

## Lecciones accionables
- Para escalar con control de costos, usa **Cost per result goal** o **Bid cap** combinado con un presupuesto alto (ej. "pago $13 por cliente, presupuesto ilimitado"), pero espera entrega errática si Meta no encuentra suficientes subastas al costo fijado.
- Si optimizas para **link clicks**, establecer un **Cost per result goal** de $0.10 puede limitar las impresiones solo a **Audience Network**, reduciendo calidad y alcance.
- Para campañas de **purchase**, la puja manual es más viable que para otros eventos, pero aun así produce resultados inconsistentes; solo pruébala como experimento controlado.
- Configura el **performance goal** en "Maximize number of conversions" (evento: purchase) para que Meta busque el mayor volumen posible, aunque gaste todo el presupuesto.

## Reglas para agentes
- Usa **Cost per result goal** o **Bid cap** solo cuando estés dispuesto a sacrificar consistencia en la entrega y calidad de resultados.
- Nunca combines puja manual con un presupuesto bajo; si el costo objetivo no se alcanza, Meta detendrá la entrega y no gastarás el presupuesto.
- No optimices para **link clicks** con puja manual si buscas conversiones de alto valor; los clics baratos de Audience Network son de baja calidad.
- Siempre que uses puja manual para **purchase**, monitorea la frecuencia y el CTR; si la entrega se detiene, vuelve a la puja automática.

## Errores comunes que evita o menciona
- Creer que fijar un **cost cap** bajo (ej. $10 por compra) garantiza ese costo; Meta no puede forzar conversiones y puede dejar de mostrar anuncios si no encuentra subastas al precio.
- Asumir que la puja manual escala automáticamente en días buenos; Meta prioriza gastar el presupuesto, no mantener el costo, por lo que en días malos el CPA puede dispararse.
- Usar puja manual para eventos que no sean **purchase** (ej. link clicks o landing page views) porque la calidad de los resultados se degrada significativamente.
- Ignorar que Meta ya optimiza para el menor costo posible por defecto; la puja manual solo añade restricciones que suelen empeorar el rendimiento.
