# Facebook Campaign Structure with Ad Sets Tutorial
**Fuente:** JonLoomer | https://youtu.be/kL3zunz51iU

## Qué enseña
Explica la estructura actualizada de campañas en Facebook Ads (Power Editor), detallando cómo organizar campañas, conjuntos de anuncios (ad sets) y anuncios. Muestra un ejemplo real con 13 ad sets para Page Likes, cada uno con presupuesto, audiencia y copia diferenciados.

## Lecciones accionables
- Al crear una campaña en Power Editor, haz clic en **Campaigns** > **Create Campaign** y nombra la campaña incluyendo el objetivo exacto, ej: `test clicks to website auction [nombre del producto]`.
- Deja el tipo de compra en **Auction** (recomendado para la mayoría de casos).
- No configures fecha de inicio/fin ni presupuesto a nivel de campaña; todo eso se define ahora en el **ad set**.
- Para crear un ad set: selecciona la campaña a la izquierda, haz clic en **Ad Sets** > **Create Adset**, y nómbralo de forma descriptiva para identificar la audiencia después, ej: `test clicks to website fans only`.
- Asigna un presupuesto diario distinto por ad set (ej: $10 para un ad set, $5 para otro).
- Para crear un anuncio: selecciona la campaña a la izquierda, haz clic en **Ads** > **Create Ad**, elige la campaña y el ad set específico. Nombra el anuncio con detalles como `test clicks to website image sizes infographic fans only`.
- Las variaciones de placement, copia o tipo de anuncio se gestionan dentro del ad set, no a nivel de anuncio.
- En el ejemplo real: creó 13 ad sets bajo una misma campaña con objetivo **Page Likes**, usando **Website Custom Audiences** basadas en duración (ej: visitantes de 1 día, 7 días, etc.), y ajustó el presupuesto según rendimiento: $15 para el mejor, $2 o $5 para otros.

## Reglas para agentes
- Usa **Power Editor** para crear y gestionar la estructura de campañas.
- Nombra cada campaña con el objetivo y el producto promocionado (ej: `Page Like Campaign - New Campaign Structure`).
- Nunca configures presupuesto ni fechas a nivel de campaña; hazlo siempre en cada **ad set**.
- Crea un ad set separado por cada segmento de audiencia diferente (intereses, duración de visita, etc.).
- Asigna presupuestos distintos por ad set según el rendimiento observado (más presupuesto al que mejor convierte).
- Usa **Website Custom Audiences** en lugar de interest targeting o lookalike audiences cuando sea posible.

## Errores comunes que evita o menciona
- No poner detalles descriptivos en el nombre de la campaña, ad set o anuncio, lo que dificulta identificarlos después.
- Configurar fecha o presupuesto a nivel de campaña (ahora se hace en ad set).
- Usar interest targeting o lookalike audiences cuando ya se tienen Website Custom Audiences disponibles.
- No ajustar presupuestos por ad set según rendimiento, perdiendo oportunidad de optimizar gasto.
