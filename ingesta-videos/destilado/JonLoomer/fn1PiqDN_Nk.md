# How to Structure Seasonal Ad Campaigns
**Fuente:** JonLoomer | https://youtu.be/fn1PiqDN_Nk

## Qué enseña
Cómo estructurar campañas publicitarias en Meta Ads para negocios estacionales que venden productos físicos para diferentes festividades. Explica por qué separar los anuncios estacionales en sus propios conjuntos de anuncios dentro de una campaña CBO es más limpio y controlable que mezclarlos con anuncios siempre activos (evergreen).

## Lecciones accionables
- Crea una campaña CBO (Campaña basada en presupuesto) con múltiples conjuntos de anuncios: un **ad set principal** para el producto evergreen que corre todo el año, y **ad sets separados** para cada temporada o festividad (ej. San Valentín, Navidad).
- Para lanzar un nuevo ad set estacional, inicia con un **creative test** que incluya de 2 a 5 anuncios dentro de ese ad set. Dedica un porcentaje específico del presupuesto de la campaña a esa prueba.
- Cuando llegue la siguiente temporada, crea un nuevo ad set para ese producto estacional. No apagues automáticamente el ad set anterior; evalúa si sigue generando resultados. Si el rendimiento cae, apágalo.
- Al año siguiente, al reiniciar el ciclo, comienza el ad set estacional usando los anuncios del año anterior como base. Si el tiempo lo permite, añade nuevos anuncios usando el **creative testing tool**.
- Para actualizar el ad set evergreen con nuevos anuncios durante el año, usa también el **creative testing tool** para introducir las variantes.
- **Resiste aplicar ad set spending limits** (límites de gasto por conjunto) si es posible. Confía en que Meta optimizará el presupuesto hacia lo que mejor funciona para maximizar resultados.
- Si Meta tiene problemas con la rotación constante de productos estacionales, considera aplicar un **ad set spending maximum** o **minimum** según el problema específico que necesites resolver (ya sea en el ad set evergreen o en los estacionales).

## Reglas para agentes
- Usa una campaña CBO con ad sets separados para productos estacionales y evergreen; nunca mezcles ambos tipos en un mismo ad set.
- Cuando inicies un ad set estacional nuevo, incluye siempre entre 2 y 5 anuncios en un creative test; nunca lances un solo anuncio sin prueba.
- No apagues un ad set estacional solo porque la festividad haya pasado; apágalo solo cuando el rendimiento decline.
- Al reiniciar un ciclo anual, reutiliza los anuncios del año anterior como punto de partida; nunca empieces desde cero si ya tienes activos probados.
- Evita aplicar spending limits a menos que el sistema de Meta muestre problemas de optimización con la rotación estacional.

## Errores comunes que evita o menciona
- Mezclar anuncios estacionales y evergreen en el mismo ad set, lo cual reduce el control organizativo y dificulta la gestión de presupuestos por temporada.
- Apagar automáticamente un ad set estacional al terminar la festividad, ignorando que podría seguir generando resultados rentables.
- Aplicar spending limits innecesariamente, lo que puede limitar la capacidad de Meta para optimizar el presupuesto hacia los mejores resultados.
- No usar el creative testing tool para introducir nuevos anuncios, perdiendo la oportunidad de probar variantes de forma estructurada.
