# sb-shot-size
**Fuente:** https://www.studiobinder.com/camera-shots/shot-size/

## Que ensena (2-3 lineas)
Guía completa de tamaños de plano en cine, desde el Extreme Close Up hasta el Extreme Wide Shot. Explica cómo cada tamaño afecta la narrativa visual, la proximidad emocional con el personaje y la información que recibe el espectador. Incluye combinaciones con ángulo, encuadre, movimiento y lentes para un storytelling visual preciso.

## Lecciones accionables (terminos en ingles para prompts, focales, esquemas de luz, reglas de composicion/color, pasos, valores)
- **Terminos para prompts:** ECU, CU, MCU, MS, Cowboy Shot, MFS, FS, WS/LS, EWS/ELS
- **Focales sugeridas:** ECU (50-100mm), CU (35-50mm), MS (35mm), WS (18-24mm), EWS (12-18mm)
- **Esquemas de luz:** ECU/CU: luz suave frontal o Rembrandt para intimidad; MS: luz lateral o 3/4 para volumen; WS: luz natural o ambiental para contexto
- **Reglas de composicion:** Regla de tercios, espacio de mirada (lead room), equilibrio visual, lineas guia
- **Pasos:** 1) Definir intención narrativa → 2) Elegir tamaño de plano → 3) Seleccionar lente y distancia → 4) Ajustar encuadre y composición → 5) Iluminar según el tono
- **Valores:** Proximidad emocional (CU=íntimo, WS=distante), información visual (WS=contexto, ECU=detalle), ritmo narrativo

## Reglas para el director de fotografia IA
- Usa ECU solo para emociones extremas o detalles críticos; no abuses.
- Combina tamaños de plano para crear variación rítmica y evitar monotonía visual.
- El Cowboy Shot (CS) es ideal para diálogos con acción física o armas.
- El MFS (Medium Full Shot) funciona bien para presentar personajes de cuerpo entero sin perder expresividad.
- En WS/EWS, prioriza la composición del entorno y la posición del sujeto en el cuadro.

## Errores comunes
- Usar CU sin justificación emocional o narrativa, saturando la escena de primeros planos.
- Ignorar el espacio de mirada en MS y WS, dejando al personaje pegado al borde del encuadre.
- No diferenciar entre MS y MCU, perdiendo la intención de cercanía o distancia.
- Elegir WS para diálogos íntimos, diluyendo la conexión emocional.
- Olvidar que el tamaño de plano debe cambiar con el tono de la escena, no ser estático.
