# tiktok-smartplus-best-practices
**Fuente:** https://ads.tiktok.com/help/article/best-practices-for-smart-plus-web-campaigns | canal temático: tiktok | destilado 2026-07-09
## Qué enseña (2-3 líneas)
Smart+ Web Campaigns funciona mejor cuando la campaña llega con buena señal de datos, suficiente material creativo, presupuesto proporcional al CPA histórico y una fase de aprendizaje sin interrupciones.  
El documento define umbrales concretos para lanzamiento, pujas objetivo, ROAS mínimo, edición de campañas y escalamiento de presupuesto.
## Lecciones accionables
- Configura conexiones de datos antes de lanzar: instala TikTok Pixel y Events API para alimentar la optimización de campañas web.
- Para anuncios web, adopta catálogo cuando aplique, porque permite aprovechar Smart+ Catalog Ads.
- Sube al menos 6 activos creativos durante la creación de la campaña; después deja que Smart+ gestione la optimización creativa.
- Programa la campaña Smart+ para correr al menos 7 días.
- Como punto de partida de presupuesto, usa al menos 10x tu CPA histórico para objetivos comparables.
- Usa split testing para validar el incremento real de rendimiento de Smart+ Campaigns, en vez de asumir mejora por observación simple.
- Si usas Target CPA o Minimum ROAS, define el CPA objetivo o ROAS mínimo tomando como referencia campañas similares que usen maximum delivery.
- Para campañas con Target CPA o Minimum ROAS, el presupuesto diario ideal es 30x el CPA histórico.
- Para campañas con Target CPA o Minimum ROAS, no bajes de 10x el CPA histórico como presupuesto diario.
- Durante los primeros 7 días de entrega, evita cambios significativos en puja, audiencia o creatividad, porque pueden afectar la fase de aprendizaje.
- Después de los primeros 7 días, prueba editar la puja en incrementos de hasta 15% cada 2 días hasta observar el rendimiento deseado.
- Si una campaña usa atribución view-through y no consume el presupuesto como se espera, prueba desactivar el seguimiento view-through para favorecer mayor entrega.
- Si la campaña consume regularmente 90% o más del presupuesto diario, prueba aumentar el presupuesto diario hasta 30% para ganar volumen mientras mantienes rendimiento.
## Reglas para agentes
- Usa TikTok Pixel y Events API antes de recomendar el lanzamiento de una Smart+ Web Campaign.
- Usa catálogo en campañas web cuando el anunciante tenga inventario o feed disponible.
- Incluye al menos 6 activos creativos al crear la campaña.
- Configura una duración mínima de 7 días para cualquier Smart+ Campaign.
- Usa un presupuesto inicial mínimo de 10x el CPA histórico para objetivos comparables.
- Usa 30x el CPA histórico como presupuesto diario ideal cuando la campaña tenga Target CPA o Minimum ROAS.
- Nunca recomiendes un presupuesto diario menor a 10x el CPA histórico en campañas con Target CPA o Minimum ROAS.
- Define Target CPA o Minimum ROAS usando campañas similares con maximum delivery como referencia.
- No edites puja, audiencia ni creatividad durante los primeros 7 días salvo que sea estrictamente necesario.
- Después del día 7, ajusta la puja como máximo 15% cada 2 días.
- Usa split testing para comprobar el lift de rendimiento de Smart+.
- Si la campaña no agota presupuesto y usa view-through attribution, prueba apagar view-through tracking.
- Si la campaña consume 90% o más del presupuesto diario de forma regular, prueba subir el presupuesto diario hasta 30%.
## Errores comunes que evita o menciona
- Lanzar campañas Smart+ Web sin TikTok Pixel ni Events API.
- Crear campañas con menos de 6 activos creativos.
- Interrumpir la fase de aprendizaje con cambios en puja, audiencia o creatividad durante los primeros 7 días.
- Empezar con presupuesto demasiado bajo respecto al CPA histórico.
- Configurar Target CPA o Minimum ROAS sin referencia de campañas similares con maximum delivery.
- Escalar presupuesto agresivamente cuando la campaña ya consume bien; la guía limita la prueba a aumentos de hasta 30%.
- Ajustar pujas demasiado rápido o con cambios grandes; la guía recomienda hasta 15% cada 2 días después del periodo inicial.
- Mantener view-through tracking activo cuando la campaña no está agotando presupuesto y se necesita más entrega.
