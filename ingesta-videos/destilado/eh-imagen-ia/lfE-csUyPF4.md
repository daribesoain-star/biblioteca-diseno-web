# Nano Banana AI Tutorial | How to Use Google’s Free Tool (Step-by-Step)
**Fuente:** eh-imagen-ia | https://youtu.be/lfE-csUyPF4

## Qué enseña (2-3 líneas)
Este tutorial explica cómo usar Nano Banana, la herramienta gratuita de generación y edición de imágenes de Google integrada en Gemini. Muestra cómo crear imágenes desde texto, editar fotos propias manteniendo la consistencia facial, combinar múltiples imágenes y refinar resultados iterativamente, además de conectar con Veo 3 para animar las imágenes generadas.

## Lecciones accionables
- Accede a Nano Banana desde el sitio web de Gemini o descargando la app de Google Gemini en iPhone o Android. Inicia sesión con tu cuenta de Google.
- En la página principal, haz clic en **Tools** y selecciona **Create Images** (opción con el ícono de banana).
- Para generar una imagen desde texto: escribe un prompt descriptivo en el campo de texto y haz clic en **Submit** (esquina inferior derecha).
- Para editar una imagen ya generada: escribe un nuevo prompt en el campo inferior (ejemplo: "name the cookie store Kevin Cookie Company") y presiona Submit. La herramienta mantiene la composición y solo modifica lo solicitado.
- Para cambiar condiciones de iluminación: usa prompts como "make it brighter and sunnier" — la herramienta ajusta la escena conservando la estructura y añadiendo sombras realistas.
- Para usar una foto propia: haz clic en el ícono **+**, elige **upload photo**, arrastra y suelta la imagen, o pégala directamente. Luego escribe el prompt de transformación.
- Prompt viral para figurita 3D: "Turn me into a 1/7 scale collectible figurine on a desk with toy packaging beside me". La herramienta mantiene la cara consistente con la foto original.
- Para combinar dos fotos: sube ambas imágenes (ejemplo: foto tuya + foto de una chaqueta). Escribe prompt como: "Blend these two images. Put me into the jacket so it looks like I'm actually wearing it. Match the colors, shadows, and textures so it looks realistic."
- Para cambiar el entorno manteniendo el outfit: después de combinar, escribe un nuevo prompt como "Make me sit in an executive office".
- Si el resultado no te gusta, haz clic en **Redo** para generar otra versión.
- Para descargar: pasa el cursor sobre la imagen y haz clic en el ícono de descarga (esquina superior derecha) para obtener la imagen a tamaño completo.
- Para convertir imagen en video: haz clic en **Tools** → **Create Videos with Veo** (opción paga). Sube la imagen como fotograma inicial, escribe un prompt de animación (ejemplo: "animate this photo into a realistic video") y haz clic en Submit.

## Reglas para agentes
- Usa "Create Images" desde el menú Tools cuando quieras activar Nano Banana en Gemini.
- Siempre sube una foto de referencia cuando necesites mantener la identidad facial consistente en múltiples ediciones.
- Nunca uses prompts vagos; especifica cambios concretos (color, iluminación, texto en letreros, entorno) para que la herramienta itere correctamente.
- Cuando combines dos imágenes, incluye en el prompt instrucciones explícitas sobre colores, sombras y texturas para lograr realismo.
- Si el resultado no es satisfactorio, usa Redo en lugar de empezar desde cero.
- Para descargar, pasa el cursor sobre la imagen y usa el ícono de descarga; no uses capturas de pantalla.

## Errores comunes que evita o menciona
- No usar prompts demasiado genéricos: el tutorial muestra que prompts específicos como "name the cookie store Kevin Cookie Company" o "make it brighter and sunnier" producen ediciones precisas.
- No asumir que la herramienta solo genera imágenes desde texto: el tutorial enfatiza que Nano Banana también edita fotos existentes y combina múltiples imágenes.
- No olvidar que la consistencia facial es una fortaleza clave: el tutorial lo demuestra al convertir una foto en figurita manteniendo el rostro idéntico.
- No pensar que solo funciona en web: la app de Gemini está disponible para iPhone y Android.
- No confundir Nano Banana (gratuito) con Veo 3 (pago): el tutorial aclara que la creación de video requiere la opción paga.
