# How to Make & Edit Images with Nano Banana for Beginners
**Fuente:** eh-imagen-ia | https://youtu.be/-Yq1Yi0MpDQ
## Qué enseña (2-3 líneas)
Este tutorial enseña a crear y editar imágenes con Nano Banana (accedido vía Gemini), el mejor modelo de edición de imágenes según el autor, sin necesidad de Photoshop. Muestra cómo generar imágenes desde texto, usar el modo Pro para mayor realismo, y editar fotos subidas (incluyendo rostros de personas) para crear composiciones personalizadas como tarjetas de cumpleaños en menos de 3 minutos.

## Lecciones accionables
- **Acceder a la herramienta:** ir a `gemini.google.com` e iniciar sesión con una cuenta de Google (gratuito, sin descargas).
- **Crear imagen básica:** en un chat nuevo, escribir un prompt como `"create an image of a stunning beach in the Maldives"`. El modelo genera la imagen en ~8 segundos.
- **Activar Nano Banana Pro (gratuito):** en la interfaz de Gemini, hacer clic en el botón "fast" (abajo) y cambiar el modelo a `"thinking with free Pro"`. Esto permite ediciones avanzadas, más realismo y mejor manejo de texto.
- **Subir imágenes propias para editar:** hacer clic en "add files" > "upload files" y seleccionar la imagen desde el ordenador. También se puede arrastrar o tomar una foto directa.
- **Crear composición con múltiples imágenes:** subir varias fotos (ej: dos personas) y luego escribir un prompt que incluya `"create an image of..."` seguido de la descripción deseada. El modelo usará todas las imágenes subidas automáticamente.
- **Prompt de ejemplo (tarjeta de cumpleaños):** `"create an image of a surfing competition trophy podium and confetti. I'll add, I'll say, 'Dream team, 30 years of winning' on a banner, plus fake career stats that are actually life highlights."`
- **Editar imagen generada:** después de obtener el resultado, escribir una instrucción como `"Make the sunset in the image more dramatic"` para modificar cualquier aspecto. El modelo entiende referencias al contexto de la imagen generada.

## Reglas para agentes
- Usa `"thinking with free Pro"` cuando necesites ediciones avanzadas, mayor realismo o incluir texto en la imagen.
- Sube siempre las imágenes de las personas (tuyas o de amigos) antes de escribir el prompt de creación; el modelo las usará automáticamente si el prompt comienza con `"create an image of..."`.
- Nunca intentes usar rostros de celebridades; el modelo rechaza trabajar con caras de famosos.
- Cuando edites una imagen ya generada, refiérete explícitamente a ella en la instrucción (ej: `"Make the [elemento] in the image more [cambio]"`).

## Errores comunes que evita o menciona
- **No cambiar al modo Pro:** si usas el modelo estándar (no "thinking with free Pro"), las ediciones serán menos realistas y no podrás manejar texto correctamente.
- **Olvidar subir imágenes antes del prompt:** si quieres incluir fotos de personas, debes subirlas primero; de lo contrario, el modelo no las usará.
- **Usar caras de celebridades:** el modelo bloquea explícitamente este uso; solo funciona con personas reales (tú o tus amigos).
- **Exceder el límite de generaciones gratuitas:** en cuenta gratuita, tras crear varias imágenes, hay que esperar un tiempo para seguir usando la herramienta.
