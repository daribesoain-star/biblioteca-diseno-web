# Create Product Ads in Minutes with Higgsfield AI (3-Step Tutorial)
**Fuente:** hf_product | https://youtu.be/IhEByygnUnM

## Que ensena sobre Higgsfield (2-3 lineas)
Flujo completo para crear anuncios de producto: escribir prompt cinematográfico, subir foto del producto como start frame, generar video con modelo Kling. Optimizado para redes sociales (Instagram, TikTok, YouTube Shorts). Cada segundo de video = 1 crédito.

## Tips y flujos accionables

**Configuracion inicial:**
- Template: **General** (no usar templates temáticos para evitar restricciones de estilo)
- Modelo: **Kling** (mejor motion y consistencia para producto)
- Versión modelo: **2.1**
- Duración: **10 segundos** (ideal para shorts)
- Resolución: **720p** (exporta más rápido, calidad suficiente para mobile; 1080p no es notablemente mejor en feeds)

**Prompt cinematográfico (copiar textual):**
```
The bottle stands upright next to dumbbells on a gym floor, spotlight catching its matte surface. Then, bam, quick cut. The bottle snaps into focus in a hiking scene, resting on a rock with mountains blurred in the background. Final cut. the bottle beside a laptop on a desk glowing softly in warm light.
```
- Estructura: 3 escenas con cortes rápidos (gym → montaña → escritorio)
- Cada escena ataca un segmento de audiencia diferente

**Start frame:**
- Subir foto real del producto (no generada)
- Higgsfield la usa como referencia visual exacta

**Flujo A/B testing:**
- Una idea por video
- Duplicar proyecto para variantes A/B
- Iterar sobre perfeccionar

**Exportación:**
- Click **Download** directo
- Subir a Facebook, TikTok, Instagram, YouTube

## Reglas para el erudito de Higgsfield

- Para anuncios de producto usa **modelo Kling 2.1** con **template General**
- Para duración en redes sociales usa **10 segundos** exactos
- Para exportación rápida usa **720p** (no 1080p)
- Para consistencia visual sube **foto real del producto** como start frame
- Para prompts efectivos estructura **3 escenas con cortes rápidos** y descripciones sensoriales (spotlight, matte surface, glowing softly)
- Para A/B testing crea **un video por idea** y duplica el proyecto

## Errores comunes / que evitar

- **No usar prompts vagos** como "Make a video of a bottle" – da resultados genéricos
- **No usar templates temáticos** – limitan la flexibilidad creativa
- **No usar 1080p** para redes sociales – no mejora calidad notablemente pero ralentiza exportación
- **No exceder 10 segundos** – la atención en social media es menor que un scroll
- **No subir imágenes generadas** como start frame – usar foto real del producto para mejor referencia
- **No mezclar múltiples ideas** en un solo video – una idea por video para claridad de mensaje
