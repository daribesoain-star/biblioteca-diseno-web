# How To Create $100k AI Ads with Higgsfield (Step-by-Step)
**Fuente:** hf_product | https://youtu.be/itv4Xljkbqw

## Que ensena sobre Higgsfield (2-3 lineas)
Higgsfield Marketing Studio permite crear anuncios hiperrealistas sin actores, equipo de filmación ni producción tradicional. Con solo un link de producto, un avatar (opcional) y un prompt, genera desde spots cinematográficos tipo TV hasta UGC, unboxing y tutoriales, manteniendo consistencia del personaje entre escenas.

## Tips y flujos accionables

### Flujo general para cualquier anuncio
1. **Entrar a Marketing Studio** desde el dashboard de Higgsfield.
2. **Seleccionar formato de anuncio**: TV spot, hyperfocus, UGC, tutorial, unboxing, product review, wildcard, hypermotion.
3. **Añadir producto**: pegar link de página de producto (Amazon, etc.) o subir imágenes directamente.
4. **Añadir avatar** (opcional): si quieres una persona consistente en todas las escenas. Si no, dejar en blanco.
5. **Escribir prompt**: usar Claude AI para generar borrador, luego ajustar manualmente.
6. **Ajustar parámetros**: aspect ratio (según plataforma destino), resolución, duración.
7. **Generar** y repetir si es necesario (regenerar o ajustar prompt hasta obtener resultado deseado).

### Prompts VERBATIM usados en el video
- **Para TV spot/cinematográfico**: "Write me a short TV spot style ad for this product. Make it feel high-end and cinematic." + pegar link del producto.
- **Para UGC**: "Write me a short UGC style ad for these markers, something that feels natural like a TikTok video." + pegar link de Amazon.
- **Para unboxing**: "Write a short unboxing style script for a milk frother focusing on packaging, what's included, and first impressions."
- **Para tutorial**: "Write a short tutorial style script for a blender showing how it works and why someone would use it."

### Trucos para look cinematográfico
- **Usar formato "TV spot"** para el look más premium.
- **Alternativas**: hypermotion o wildcard también funcionan para estilo cinematográfico.
- **Para anuncios sin avatar**: usar formato "hyperfocus" con prompt que describa el producto en acción de forma exagerada y emocionante.
- **Para UGC**: usar formato "UGC" (también funciona tutorial, unboxing, product review).

### Flujo avanzado: combinar múltiples formatos
1. Crear un clip de **unboxing** (primeros 5-10 segundos como hook).
2. Crear un segundo clip con el **mismo producto y mismo avatar** en formato **tutorial** (explicación detallada).
3. **Unir ambos clips** en cualquier editor de video (splice).
4. Resultado: hook visual + demostración de uso → mayor confianza y conversión.

### Estrategia de testing con avatares
- Usar el **mismo prompt** pero con **diferentes avatares** para generar múltiples variaciones de UGC.
- Ejemplo: un avatar probando el producto, otro dibujando, otro reaccionando a colores.
- Permite testear sin pagar a múltiples creadores ni esperar revisiones.

## Reglas para el erudito de Higgsfield

- **Para anuncio cinematográfico tipo TV spot**: usa el preset **"TV spot"** con avatar y link de producto.
- **Para anuncio sin personas, solo producto en acción**: usa el preset **"hyperfocus"** sin avatar.
- **Para UGC estilo TikTok**: usa el preset **"UGC"** con avatar y prompt natural.
- **Para unboxing**: usa el preset **"unboxing"** con avatar y prompt enfocado en packaging y primeras impresiones.
- **Para tutorial/demostración**: usa el preset **"tutorial"** con avatar y prompt que explique funcionamiento.
- **Para máxima consistencia del personaje entre escenas**: usa el **mismo avatar** en todos los clips que planeas combinar.
- **Para generar prompts base**: usa **Claude AI** con instrucción simple + link del producto, luego ajusta manualmente.
- **Para aspecto profesional**: ajusta **aspect ratio** según plataforma destino (vertical para TikTok/Reels, horizontal para TV/YouTube).
- **Para evitar resultados imperfectos**: regenera el anuncio varias veces o tweetea el prompt hasta lograr el look deseado.

## Errores comunes / que evitar

- **No esperar resultados perfectos al primer intento**: como en filmación real, a veces necesitas regenerar o ajustar el prompt varias veces.
- **Usar el mismo avatar para todos los anuncios**: limita la variedad; mejor crear múltiples versiones con diferentes avatares para testear.
- **Copiar el prompt de Claude sin ajustar**: siempre tweetea el prompt generado para que suene exactamente como quieres.
- **No aprovechar la combinación de formatos**: limitarse a un solo tipo de anuncio pierde la oportunidad de crear secuencias completas (hook + tutorial).
- **Ignorar los formatos alternativos**: hypermotion y wildcard también sirven para estilo cinematográfico, no solo TV spot.
- **No usar avatar cuando se necesita consistencia**: si quieres el mismo personaje en múltiples escenas, es obligatorio seleccionar un avatar.
- **Pensar que solo sirve para e-commerce**: también funciona para Etsy, print-on-demand, o como servicio para pequeñas empresas.
