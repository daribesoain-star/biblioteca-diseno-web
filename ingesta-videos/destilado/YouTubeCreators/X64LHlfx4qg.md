# Members Only Content: Videos, Shorts, Live Streams, & Posts for Channel Memberships
**Fuente:** YouTubeCreators | https://youtu.be/X64LHlfx4qg

## Qué enseña
Cómo crear y compartir contenido exclusivo para miembros del canal en YouTube, incluyendo videos, Shorts, transmisiones en vivo y publicaciones. Explica el proceso técnico de configuración, las opciones de visibilidad y programación, y cómo maximizar el valor de esta función para fidelizar miembros.

## Lecciones accionables
- Para crear un video exclusivo para miembros: durante la subida, ve a **Visibilidad** → selecciona **"Solo para miembros"**.
- Para convertir un video existente en exclusivo: edita su visibilidad en YouTube Studio y cambia a **"Solo para miembros"**.
- Para programar que un contenido pase de "solo para miembros" a público: en **Visibilidad** → **Programar** → elige **"De solo para miembros a público"** → selecciona día y hora.
- Para crear un Short exclusivo: durante la subida, ve a **Visibilidad** → **"Solo para miembros"**. Nota: no puede incluir música de terceros ni de la biblioteca de Shorts.
- Para notificar sobre un Short exclusivo: crea una **publicación solo para miembros** y adjunta el Short, así se enviará notificación a los miembros elegibles.
- Para transmisión en vivo exclusiva desde computadora: inicia sesión en **YouTube Studio** → **Crear** → **En vivo** → en visibilidad selecciona **"Solo para miembros"** y elige el nivel de membresía.
- Para transmisión en vivo exclusiva desde celular: toca el ícono **Crear** → desliza a **En vivo** → **Visibilidad** → **"Solo para miembros"** → selecciona nivel → **Siguiente** → **En vivo**.
- Para publicaciones exclusivas: al crear una publicación, despliega **Visibilidad** y selecciona **"Solo para miembros"** o un nivel específico.
- Para agregar una sección de contenido exclusivo en la página de inicio del canal: ve a **YouTube Studio** → **Personalización** → **Secciones destacadas** → **Agregar sección** → selecciona **"Contenido solo para miembros"**.
- Para promocionar contenido exclusivo a la audiencia general: compártelo mediante **publicaciones**, **tarjetas** o **listas de reproducción**. Los no miembros verán un aviso para registrarse.

## Reglas para agentes
- Usa **Visibilidad → "Solo para miembros"** cuando subas cualquier contenido que deba ser exclusivo para miembros.
- Nunca incluyas música de terceros ni de la biblioteca de Shorts en Shorts exclusivos para miembros; deben ser contenido 100% original.
- Cuando programes contenido, usa **"De solo para miembros a público"** si quieres dar acceso anticipado antes del lanzamiento público.
- Siempre selecciona el **nivel de membresía** específico al configurar visibilidad en transmisiones en vivo exclusivas.
- Para maximizar notificaciones en Shorts exclusivos, crea una **publicación solo para miembros** que lo contenga, ya que los Shorts no generan notificación por sí solos.

## Errores comunes que evita o menciona
- No usar música protegida por derechos de autor en Shorts exclusivos para miembros (ni siquiera de la biblioteca de Shorts).
- Olvidar que los Shorts exclusivos no envían notificaciones automáticas; se debe adjuntar a una publicación para notificar.
- No promocionar el contenido exclusivo a la audiencia general, perdiendo oportunidades de conversión a membresías.
- No configurar el nivel de membresía correcto en transmisiones en vivo, permitiendo acceso a niveles no deseados.
- No usar la opción de programación "De solo para miembros a público" para contenido planeado, desaprovechando el beneficio de acceso anticipado.
