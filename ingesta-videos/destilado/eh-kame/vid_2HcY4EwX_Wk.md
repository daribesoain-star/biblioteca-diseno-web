# Kame One ERP en programa de TV Sabores
**Fuente:** kame | https://youtu.be/2HcY4EwX_Wk

## Qué enseña
El video presenta Kame One ERP como un software de gestión empresarial diseñado específicamente para pymes chilenas, destacando su implementación rápida (2 horas), acceso desde celular y bajo costo. Se enfoca en la importancia del control financiero, facturación electrónica integrada con el Servicio de Impuestos Internos (SII) y la gestión del flujo de caja para evitar quiebras por falta de liquidez.

## Lecciones accionables
- **Facturación desde celular:** Usar la app de Kame One para emitir facturas electrónicas en línea con el SII sin necesidad de computador de escritorio.
- **Captura automática de documentos:** Configurar la integración con el SII para que Kame One importe automáticamente todas las facturas de compra recibidas, eliminando la digitación manual.
- **Informe de simulación F29:** Acceder al módulo de "Simulaciones F29" para conocer en cualquier momento el IVA a pagar antes de la fecha límite (día 20).
- **Control de cuentas por cobrar:** Revisar desde la app el estado de deuda de los clientes para anticipar problemas de flujo de caja.
- **Gestión multiusuario:** Asignar usuarios (dueño, contador, RRHH) sin cobrar por cantidad de empleados, solo por usuarios activos simultáneos.
- **Conciliación bancaria y tesorería:** Usar los módulos de conciliación bancaria, tesorería, compras y órdenes de compra para centralizar la administración.

## Reglas para agentes
- Si el usuario pregunta sobre implementación, indicar que Kame One se configura en 2 horas y es 100% online, sin instalación de servidores.
- Si el usuario menciona problemas de flujo de caja, recomendar usar el informe de simulación F29 y el módulo de cuentas por cobrar para anticipar pagos.
- Si el usuario es una pyme pequeña (1-5 empleados), destacar que el sistema parte desde 18 UF/año (~$52,000 CLP mensuales) y no requiere inversión en hardware.
- Si el usuario pregunta sobre soporte, mencionar que el tiempo promedio de respuesta es de 5 minutos.

## Errores comunes que menciona
- **Usar Excel o planillas manuales:** Muchas pymes gestionan con múltiples hojas de cálculo, lo que genera errores y falta de control en tiempo real.
- **No saber plazos de pago de clientes:** Las pymes ignoran a cuántos días les pagan, lo que causa quiebras por flujo de caja aunque el negocio sea rentable.
- **Digitar facturas manualmente:** En sistemas tradicionales se debe ingresar cada documento a mano, mientras que Kame One captura automáticamente del SII.
- **Depender de un computador fijo:** El dueño no puede tomar vacaciones porque debe estar en la oficina para controlar el negocio; Kame One permite operar desde cualquier dispositivo con internet.
- **Pagar precios elevados por ERP tradicionales:** Los sistemas para grandes corporaciones cuestan millones de pesos, mientras que Kame One ofrece solución para pymes a bajo costo.
