# Higgsfield tips tricks best settings workflow
**Fuente:** hf_tips | https://youtu.be/Higgsfield tips tricks best settings workflow

## Que ensena sobre Higgsfield
Higgsfield permite generar imágenes y videos con control profesional de cámara, consistencia de personaje y transiciones cinematográficas. Incluye 7 funciones clave: generación de imágenes con modelos específicos, apps de edición (cambio de outfit, ángulos, transiciones), creación de personajes consistentes, AI influencers personalizables, y Cinema Studio con ajustes de cámara reales.

## Tips y flujos accionables

### 1. Selección de modelo de imagen (presets exactos)
- **Ultra-realismo:** usa **Nano Banana 2** (ej: "young woman exploring an erupting volcano")
- **Artístico/estilizado:** usa **Sea Dream 5.0**
- **Personajes propios:** usa **Higgsfield Soul**
- **Resolución:** 16:9 + 4K para máxima calidad

### 2. Apps de edición (flujo paso a paso)
**Outfit Swap:**
1. Ve a **Apps > Outfit Swap**
2. Sube imagen del personaje + imagen de la prenda
3. Genera → el fondo y luces se mantienen, solo cambia ropa

**Shots (9 ángulos automáticos):**
1. **Apps > Shots**
2. Sube imagen base → genera en 30 segundos
3. Obtienes: close-ups, upward framing, overhead shots
4. Selecciona todo, upscale, descarga

**Angles (control manual de cámara):**
- Usa el **círculo en panel derecho** para posicionar cámara
- Arriba = drone shot, abajo = low angle

**Transitions (transiciones cinematográficas):**
1. **Apps > Transitions**
2. Sube start frame + end frame
3. Elige entre 20+ transiciones: **Raven** (recomendada), flying camera, morphing
4. Selecciona duración → genera

### 3. Consistencia de personaje (flujo completo)
1. Genera imagen base con **Nano Banana 2**
2. **Apps > Shots** → genera 9 ángulos → upscale → descarga
3. Repite hasta tener **20-30 imágenes** de referencia
4. Ve a **Character** → sube todas → nombra (ej: "Kim")
5. Espera entrenamiento del modelo
6. Usa el personaje en cualquier prompt: "Kim as a middle school teacher"

### 4. AI Influencer (sin prompts)
1. **AI Influencer** en menú superior
2. Configura:
   - **Character type:** human / elf / crocodile / octopus
   - **Gender:** male / female
   - **Ethnicity:** Asian, etc.
   - **Skin color:** selector exacto
   - **Eye color:** amber, etc.
   - **Skin condition:** freckles, birthmarks, scars
   - **Age:** adult
   - **Body type:** athletic
   - **Style:** hair, tattoos
   - **Render style:** hyperrealistic
3. Genera → obtienes personaje listo para redes

### 5. Cinema Studio (configuración profesional)
- **Preset:** Premium Large Format Digital
- **Lente:** Classic Anamorphic 35mm
- **Apertura:** F1.4 (fondo borroso, profundidad natural)
- **Aspect ratio:** 16:9
- **Resolución:** 1K a 4K
- **Grid:** 2x2 (previsualiza sin gastar créditos)
- **What's Next:** genera variaciones de historia sin prompt

### 6. Video (Director Panel)
- **Camera movements:** docenas de opciones
- **Speed ramping:** manual (acelerar/slow motion) o auto
- **Duration:** 5 segundos
- **Genre:** action, horror, suspense, **intimate** (recomendado)
- Sube start frame + end frame → genera

## Reglas para el erudito de Higgsfield

1. **Para ultra-realismo** usa **Nano Banana 2** con resolución **16:9 + 4K**
2. **Para estilo artístico** usa **Sea Dream 5.0**
3. **Para personajes propios** usa **Higgsfield Soul**
4. **Para consistencia de personaje** entrena con **20-30 imágenes** de diferentes ángulos usando **Shots**
5. **Para transiciones cinematográficas** usa **Apps > Transitions** con preset **Raven**
6. **Para control de cámara profesional** usa **Cinema Studio** con preset **Premium Large Format Digital**, lente **Classic Anamorphic 35mm**, apertura **F1.4**
7. **Para AI influencers sin prompts** usa **AI Influencer** con **hyperrealistic render style**
8. **Para variaciones de historia** usa **What's Next** después de generar imagen
9. **Para video** selecciona **genre: intimate** para look cinematográfico
10. **Para ahorrar créditos** usa **Grid 2x2** antes de generar individualmente

## Errores comunes / que evitar

- **No usar modelo incorrecto:** no uses Sea Dream 5.0 para ultra-realismo ni Nano Banana 2 para estilizado
- **No regenerar con prompt para cambios pequeños:** usa **Outfit Swap** o **Shots** en lugar de modificar prompt
- **No usar menos de 20 imágenes de referencia** para consistencia de personaje
- **No saltarse el upscale** antes de descargar imágenes para entrenar personaje
- **No usar transiciones sin elegir preset:** elige **Raven** para resultados suaves
- **No ignorar el género en video:** usar **intimate** mejora calidad cinematográfica
- **No generar sin grid primero:** usa **2x2 grid** para comparar antes de gastar créditos
- **No usar prompts largos para AI Influencer:** configura todos los parámetros visuales en lugar de texto
