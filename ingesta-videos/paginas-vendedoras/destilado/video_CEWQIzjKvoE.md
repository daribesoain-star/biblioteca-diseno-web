# The ONLY resources you need to design award winning websites
**Fuente:** https://youtu.be/CEWQIzjKvoE | destilado 2026-07-22

## Qué enseña (2-3 líneas)
El video compila los mejores recursos digitales y físicos para inspirarse en diseño web de alto nivel (sitios galardonados, dark mode, UI interactiva). Explica cómo usar mood boards y análisis de UX para transformar inspiración en proyectos únicos, y descarta plataformas como Behance o Dribbble por ser poco específicas.

## Lecciones accionables
- **Minimum Gallery** (https://minimum.gallery): fuente de inspiración exclusiva para sitios minimalistas. No permite filtrar por industria o UI, pero es ideal si buscas diseños limpios y simples.
- **Darkmode Design** (https://darkmodedesign.com): galería solo de sitios en modo oscuro, cubriendo desde moda hasta startups tech. Úsala si tu proyecto requiere dark mode.
- **Awwwards** (https://www.awwwards.com): recurso principal con diversidad de estilos. Permite filtrar por tipo de negocio y elementos UI (acordeones, menús, scroll interactions). Además, los blogs/case studies revelan procesos y tech stack reales de sitios premiados.
- **Referral.design** (https://referral.design): UI y UX de marcas reconocidas y startups. Ideal para proyectos business-oriented; tiene listas vastas de elementos con buena experiencia de usuario.
- **Stack Sorted** (https://stacksorted.com): similar a Referral pero enfocado en UI elements altamente interactivos. Se actualiza semanalmente. Úsalo para practicar desarrollo recreando componentes.
- **Recursos físicos – revistas**: ej. revista de Uniqlo (tiendas físicas o digitales) para inspiración de tipografía, layouts y paletas de color. No necesitas comprarlas; están en librerías, tiendas de ropa o centros comerciales.
- **Proceso de uso**: antes de diseñar, crea un mood board con las inspiraciones recolectadas para definir el estilo general. Luego, navega los sitios de inspiración evaluando facilidad de navegación y búsqueda de información para mejorar la UX de tu proyecto.
- **Cita textual del creador**: "Creativity is subtraction" (Austin Kleon). Recomienda tener solo 1-2 recursos de confianza y alternar entre ellos, en lugar de acumular muchos.

## Reglas para agentes
- Usa **Awwwards** cuando necesites inspiración diversa con filtros por industria o elemento UI específico (acordeón, menú, scroll).
- Usa **Referral.design** cuando diseñes para marcas establecidas o proyectos business-oriented que requieran UX probada.
- Usa **Stack Sorted** cuando quieras practicar desarrollo recreando componentes UI interactivos; actualiza semanalmente.
- Usa **Minimum Gallery** solo si tu proyecto es minimalista y no necesitas filtrar por industria.
- Usa **Darkmode Design** exclusivamente cuando el sitio final deba tener modo oscuro.
- Para inspiración de color y layout, busca revistas físicas en tiendas (ej. Uniqlo) en lugar de recursos digitales.
- Antes de diseñar, crea un mood board con 2-3 inspiraciones y navega los sitios elegidos para evaluar su UX (facilidad de navegación, claridad de información).
- Limítate a 1-2 recursos de inspiración por proyecto; alterna entre ellos, no acumules muchos.
- No uses Behance ni Dribbble como fuentes primarias: son demasiado genéricos y difíciles de filtrar para proyectos reales.

## Errores comunes
- Usar Behance o Dribbble como fuente principal de inspiración: el creador los considera "demasiado genéricos" y difíciles de filtrar para necesidades específicas.
- Acumular muchos recursos de inspiración sin enfocarse: el creador cita "creativity is subtraction" y recomienda solo 1-2 fuentes.
- Ignorar el análisis de UX en los sitios de inspiración: no basta con copiar estilos; hay que navegarlos para evaluar navegación y flujo de información.
- No crear un mood board antes de diseñar: saltarse este paso lleva a proyectos sin dirección visual clara.
- No aprovechar los blogs/case studies de Awwwards: contienen procesos y tech stack reales que pueden ahorrar horas de investigación.
