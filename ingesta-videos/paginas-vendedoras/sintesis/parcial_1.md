# ESTILOS ÚNICOS QUE VENDEN

**Anti-diseño para destacar en mercados saturados**
Rompe convenciones usando imágenes desproporcionadamente grandes o movimiento en elementos estáticos. Funciona en landing pages experimentales, no en sitios corporativos. (Fuente: Popular Web Design Trends — https://youtu.be/8ahnUt_A5eA)

**Navegación experimental que retiene usuarios**
Implementa sistemas de zoom-in/zoom-out tipo lienzo (Skyvia) o páginas que se despliegan como pliegues (Canal Street Market). Solo si el contenido es visual y el usuario espera exploración. (Fuente: Popular Web Design Trends — https://youtu.be/8ahnUt_A5eA)

**Macroanimaciones sutiles que responden al mouse y scroll**
Combina scroll jacking, animación por movimiento del ratón y elementos 3D con velocidad lenta. Limita a 1-2 elementos animados por sección para no abrumar. (Fuente: Popular Web Design Trends — https://youtu.be/8ahnUt_A5eA)

**Smart videos de ~10 segundos activados por scroll**
Muestran funcionalidad real del producto. El video solo se reproduce cuando el usuario scrollea sobre él (ej: Hulie). Nunca con autoplay al cargar. (Fuente: Popular Web Design Trends — https://youtu.be/8ahnUt_A5eA)

**Custom illustrations con toque humano y animación ligera**
Mezcla arte dibujado a mano con la web (ej: London Therapy). Añade animación de aparición, transición o superposición para diferenciarte de contenido generado por IA. (Fuente: Popular Web Design Trends — https://youtu.be/8ahnUt_A5eA)

**Bento Grid (mosaico UI) para múltiples piezas de información**
Estructura de cuadrícula tipo mosaico (ej: Walmart homepage). Cada celda debe tener propósito claro y tamaño proporcional a su importancia. No usar si el contenido es lineal o jerárquico. (Fuente: Popular Web Design Trends — https://youtu.be/8ahnUt_A5eA)

**Expressive typography como parte del estilo visual**
Usa tipografías fuera de lo común (no solo Roboto) con mayúsculas, contornos, emojis, secuencias animadas (ej: Ancient Drinks con fuente nórdica). No combines más de 2 estilos extremos por página. (Fuente: Popular Web Design Trends — https://youtu.be/8ahnUt_A5eA)

**Brutalist design solo para marcas disruptivas y jóvenes**
Diseño sin excusas con textos y botones abrumadoramente grandes. No recomendado para marcas que buscan confianza, seriedad o lujo. (Fuente: Popular Web Design Trends — https://youtu.be/8ahnUt_A5eA)

**Text-only design con jerarquía visual y white space**
El texto como elemento más prominente (ej: portfolio de Sam). Acompaña con white space generoso y animación sutil. Usa variaciones de tamaño, color y espaciado para guiar la lectura. (Fuente: Popular Web Design Trends — https://youtu.be/8ahnUt_A5eA)

**Diseño de lujo: lenguaje de adquisición, no de venta**
Usa verbos como "adquirir", "poseer", "acceder", "pertenecer". Nunca "comprar ahora" o "añadir al carrito". Diseña la web como una galería o showroom privado, no como catálogo. (Fuente: How to Use Psychology to Sell Luxury Items — https://youtu.be/4Z3ZkSDzc_Y)

**Marcadores de identidad visibles para productos de lujo**
Incluye insignias de calidad, certificaciones, logos de colaboradores de lujo, imágenes que asocien el producto con estatus. Crea umbral de exclusividad visible: "Solo 50 unidades disponibles este año". (Fuente: How to Use Psychology to Sell Luxury Items — https://youtu.be/4Z3ZkSDzc_Y)

**Scroll tracking con smooth scroll obligatorio**
Rastrea el progreso del scroll en una porción específica (valor 0 a 1) para animaciones. Implementa smooth scroll con Lenis o Locomotive para evitar animaciones entrecortadas. (Fuente: My Top 5 Techniques for Web Animation — https://youtu.be/9eHEOAn2FOA)

**Viewport detection en TODAS las animaciones**
Detecta cuándo un elemento entra en viewport para disparar animación. Usa ScrollTrigger de GSAP o `useInView` de Framer Motion, no Intersection Observer directamente. (Fuente: My Top 5 Techniques for Web Animation — https://youtu.be/9eHEOAn2FOA)

**Text splitting para animar tipografía por caracteres/palabras**
Divide párrafos en líneas, palabras y caracteres con SplitText de GSAP o SplitType. Implementa manejo de accesibilidad y eventos de resize obligatoriamente. (Fuente: My Top 5 Techniques for Web Animation — https://youtu.be/9eHEOAn2FOA)

# PATRONES DE CONVERSIÓN

**Lead generation como estrategia principal para crecimiento sostenido**
No vendas de inmediato: obtén datos de contacto (email, LINE) para vender después. Con 10,000 contactos vs 10 contactos, el impacto en ventas es radicalmente diferente. (Fuente: [2026 Edition] How to Create a Selling Landing Page — https://youtu.be/7eX5LG6q8Ys)

**Benchmark de lead generation: 500-1000 contactos en LINE**
Con esa lista puedes llenar reservas de último minuto con un mensaje. La venta directa solo cuando tienes producto de bajo precio y alta confianza (como libros). (Fuente: [2026 Edition] How to Create a Selling Landing Page — https://youtu.be/7eX5LG6q8Ys)

**Estructura de bloques: cada bloque comunica UNA sola cosa**
Los usuarios escanean bloque por bloque, no leen linealmente. Si mezclas beneficios, características y pruebas en un mismo bloque, el usuario no leerá nada. (Fuente: [2026 Edition] How to Create a Selling Landing Page — https://youtu.be/7eX5LG6q8Ys)

**Los 6 elementos obligatorios en toda landing page:**
1. Headline corto (1-2 líneas) que resume el beneficio principal
2. Key visual REAL (nunca stock photo)
3. Subhead que expande el headline
4. Benefit: resultados o cambios que obtiene el cliente
5. CTA múltiples veces (arriba, medio, final) como elemento más visible
6. Social proof en la primera pantalla visible, no al final
(Fuente: [2026 Edition] How to Create a Selling Landing Page — https://youtu.be/7eX5LG6q8Ys)

**Patrón de diseño probado: bloques de colores alternados**
Usa colores alternados (ej: azul y blanco) para separar visualmente cada bloque. Esto facilita el escaneo y mejora la comprensión del contenido. (Fuente: [2026 Edition] How to Create a Selling Landing Page — https://youtu.be/7eX5LG6q8Ys)

**Regla de copywriting: insinúa el beneficio, no lo declares**
Si dices "las ventas aumentarán" o "perderás peso", los revisores de anuncios rechazarán la página. En su lugar: "al dominar esta tendencia, podrás adquirir clientes abrumadoramente mejor que los demás". (Fuente: [2026 Edition] How to Create a Selling Landing Page — https://youtu.be/7eX5LG6q8Ys)

**Lanza con 60-70% de avance como hipótesis**
No busques la perfección antes de lanzar. Los resultados numéricos te dirán qué mejorar. Un mes sin lanzar es un mes perdido. (Fuente: [2026 Edition] How to Create a Selling Landing Page — https://youtu.be/7eX5LG6q8Ys)

**Haz 1 split test por semana, enfocado en above the fold**
El 90% de las pruebas deben ser sobre el headline o la imagen principal. Hacer esto 52 semanas al año durante 5 años genera mejoras compuestas. (Fuente: 3 Landing Page Tests To Skyrocket Conversions & Optins — https://youtu.be/9Dc2UQbQNTo)

**Above the fold = todo lo visible sin hacer scroll**
Ahí es donde vendes al usuario si se queda o se va. No pierdas tiempo probando elementos del footer hasta dominar esto. (Fuente: 3 Landing Page Tests To Skyrocket Conversions & Optins — https://youtu.be/9Dc2UQbQNTo)

**Proceso semanal concreto de split testing:**
1. Reunión de marketing cada lunes
2. Revisar ganador de la prueba de la semana pasada
3. Si superó al control, ese es el nuevo control
4. Diseñar la siguiente prueba (cambiar headline, o imagen, o ambos si el cambio de uno invalida al otro)
(Fuente: 3 Landing Page Tests To Skyrocket Conversions & Optins — https://youtu.be/9Dc2UQbQNTo)

**Jerarquía visual sobre patrón F de lectura**
No forces a los usuarios a leer en patrón F. Usa tamaños y pesos tipográficos para guiar la atención: el elemento más importante de cada sección debe ser el más grande y negrita. (Fuente: Give Me 7 Minutes & Your Web Design Skills Will Take Off — https://youtu.be/1NTKwpAVcHg)

**Contraste extremo en botones y CTAs**
Los botones deben tener alto contraste contra todo lo que los rodea. Prohibido usar "ghost buttons" (botones solo con borde, sin fondo sólido) — no se ven y no se cliquean. (Fuente: Give Me 7 Minutes & Your Web Design Skills Will Take Off — https://youtu.be/1NTKwpAVcHg)

**Paleta de color con la regla 60-30-10:**
- 60% colores dominantes (grises, negros, blancos)
- 30% colores de marca
- 10% color de acento (reservado para CTAs y botones)
(Fuente: Give Me 7 Minutes & Your Web Design Skills Will Take Off — https://youtu.be/1NTKwpAVcHg)

**Verifica contraste con coolers.co antes de aprobar combinaciones**
Ingresa los colores de marca, activa el "color contrast teller" y te dirá qué combinaciones pasan o fallan. Si no pasa, cámbiala. (Fuente: Give Me 7 Minutes & Your Web Design Skills Will Take Off — https://youtu.be/1NTKwpAVcHg)

**Conversión = reducción de fricción**
Haz que sea "estúpidamente fácil" para el usuario encontrar lo que busca y completar una acción. Esto se logra con: claridad (quién eres, qué debe hacer, por qué), escaneabilidad (organiza para que puedan saltar y decidir rápido), motivación (habla al corazón del usuario). (Fuente: Give Me 7 Minutes & Your Web Design Skills Will Take Off — https://youtu.be/1NTKwpAVcHg)

**Captura de emails desde el día 1 con pop-up e incentivo**
Usa pop-ups con descuento 10%, ebook gratis o challenge gratuito. Herramientas: JustUno, Sumo. El email representa 30-40% del revenue de marcas de ecommerce. (Fuente: Top 10 Ecommerce Marketing Tips — https://youtu.be/3DdJ1JynG8M)

**Email blasts semanales + automatizaciones base**
Envía al menos 1 email/semana para mantener list health y deliverability. Secuencias base: welcome sequence (open rate ~50%), abandoned cart emails, product sequence educativa sobre top sellers. (Fuente: Top 10 Ecommerce Marketing Tips — https://youtu.be/3DdJ1JynG8M)

**Aumentar AOV con upsells, cross-sells y bundles**
Upsell = vender más del mismo producto (ej: de 1 a 2 unidades con descuento). Cross-sell = producto complementario. Bundles = paquetes con precio dinámico e interactivo (el precio sube/baja al seleccionar). (Fuente: Top 10 Ecommerce Marketing Tips — https://youtu.be/3DdJ1JynG8M)

**Micro influencers (5k-50k seguidores) con las 3 R:**
Reach (micro, no macro), Relevancy (busca en marcas similares no competidoras), Relationship (fuerte engagement, comentarios, diálogo bidireccional). Gifting a cambio de contenido gratuito. (Fuente: Top 10 Ecommerce Marketing Tips — https://youtu.be/3DdJ1JynG8M)

**Customer reviews con fotos (UGC) usando app Loox**
Las reseñas con foto son más poderosas que solo texto porque muestran el producto en uso. Activa secuencia post-purchase para pedir review (ej: 5 semanas después para un detox de 28 días). (Fuente: Top 10 Ecommerce Marketing Tips — https://youtu.be/3DdJ1JynG8M)

**Viral giveaways con plataformas Gleam o VIPRE**
Diseña para que cada entrada incentive compartir (viral coefficient > 1). Ejemplo: sorteo de 14 relojes en 14 días generó 80,000 nuevos emails. (Fuente: Top 10 Ecommerce Marketing Tips — https://youtu.be/3DdJ1JynG8M)

**Text message abandoned cart con app Tobu**
Los SMS tienen 80-90% de apertura en los primeros 15 minutos (vs 20-30% de email en toda su vida). Misma lógica que email abandon cart pero por SMS. (Fuente: Top 10 Ecommerce Marketing Tips — https://youtu.be/3DdJ1JynG8M)

**Facebook e Instagram video ads con captions obligatorias**
85-90% ven sin sonido. Fórmula: hook convincente → social proof o stat del problema → producto como solución (beneficios, no características) → más social proof → oferta (descuento, envío gratis). (Fuente: Top 10 Ecommerce Marketing Tips — https://youtu.be/3DdJ1JynG8M)

# POR SECCIÓN

## HERO / ABOVE THE FOLD

**Headline corto (1-2 líneas) que resume el beneficio principal**
Los usuarios deciden en segundos si leer o salir. Headline corto + subhead que expande = fórmula ganadora. Ejemplo probado: "Vender es 90%". (Fuente: [2026 Edition] How to Create a Selling Landing Page — https://youtu.be/7eX5LG6q8Ys)

**Key visual REAL, nunca stock photo**
Usa foto del dueño con cliente real, foto del seminario en vivo, portada del reporte. Las fotos de aficionado generan más realismo y confianza que imágenes profesionales genéricas. (Fuente: [2026 Edition] How to Create a Selling Landing Page — https://youtu.be/7eX5LG6q8Ys)

**Coloca social proof en la primera pantalla visible**
Los visitantes sospechan inmediatamente; necesitas generar confianza antes de que se vayan. Ejemplo: reporte de embarazo, cobertura en medios, premios en la primera pantalla. (Fuente: [2026 Edition] How to Create a Selling Landing Page — https://youtu.be/7eX5LG6q8Ys)

**Full-screen headers/hero sections sin scroll vertical**
La página permanece en pantalla completa mientras los elementos interactúan. Usa sliders para mostrar múltiples contenidos en una vista (ej: Off Menu mantiene hero fijo mientras case studies scrollean a la derecha). (Fuente: Popular Web Design Trends — https://youtu.be/8ahnUt_A5eA)

**Tipografía en tres niveles en el hero:**
- H1: texto más grande y prominente; describe de qué trata toda la página
- H2 (subheadings): dividen el resto de la página y guían la atención
- P (párrafos): texto simple, legible. Nunca uses tipografía decorativa en párrafos
(Fuente: Give Me 7 Minutes & Your Web Design Skills Will Take Off — https://youtu.be/1NTKwpAVcHg)

**CTA en el hero: el elemento visualmente más destacado**
Debe aparecer arriba (después del headline), en medio y al final. Para CTAs con múltiples opciones de precio, diseña visualmente cuál es la opción recomendada. (Fuente: [2026 Edition] How to Create a Selling Landing Page — https://youtu.be/7eX5LG6q8Ys)

## PRODUCTO / DESCRIPCIÓN

**Separa producto genérico de variante (SKU concreto)**
Tabla `product` (nombre, descripción, imagen) para ítems genéricos. Tabla `product_item` (SKU, precio, stock, imagen) para cada combinación de atributos (talla, color). (Fuente: eCommerce Database Design — https://youtu.be/1HamqOuv2Cw)

**Atributos flexibles con patrón EAV para variantes**
Tabla `variation` (nombre: "Size", "Color") vinculada a categoría. Tabla `variation_option` (valores: "S", "M", "L"). Tabla puente `product_configuration` conecta `product_item` con múltiples `variation_option`. (Fuente: eCommerce Database Design — https://youtu.be/1HamqOuv2Cw)

**Congela precio en el momento de la compra**
Almacena el precio en `order_line` en el momento del pedido, no consultes desde `product_item` que puede cambiar. Si el precio cambia después, el histórico del pedido mostrará el precio incorrecto. (Fuente: eCommerce Database Design — https://youtu.be/1HamqOuv2Cw)

**Beneficios presentados como "tres razones para comprar" o tabla de contenidos**
Muestra resultados o cambios que obtiene el cliente. Puede presentarse como problemas que resolvemos o lista de lo que incluye. (Fuente: [2026 Edition] How to Create a Selling Landing Page — https://youtu.be/7eX5LG6q8Ys)

**Narrativa emocional a través de historia y artesanía**
Conecta emocionalmente al cliente con la herencia de la marca. Marcas "ruidosas" (Versace/Gucci) usan logos grandes y colores vibrantes. Marcas "silenciosas" (Brunello Cucinelli) priorizan minimalismo y tipografía serif elegante. (Fuente: How to Use Psychology to Sell Luxury Items — https://youtu.be/4Z3ZkSDzc_Y)

**Smart videos que muestran funcionalidad real del producto**
Videos de ~10 segundos que se activan solo cuando el usuario scrollea sobre ellos (ej: Hulie). Muestran la funcionalidad real, no solo estética. (Fuente: Popular Web Design Trends — https://youtu.be/8ahnUt_A5eA)

## CHECKOUT / CARRITO

**Carrito separado del pedido**
Tabla `shopping_cart` vinculada a usuario (no invitados). `shopping_cart_item` almacena `product_item_id` y `quantity`. Al convertir a pedido, se crea `shop_order` (evita usar "order" por ser palabra reservada SQL). (Fuente: eCommerce Database Design — https://youtu.be/1HamqOuv2Cw)

**Usa `shop_order` en lugar de `order` como nombre de tabla**
Evita conflictos con palabra reservada SQL. (Fuente: eCommerce Database Design — https://youtu.be/1HamqOuv2Cw)

**Implementa `user_payment_method` con tipo de pago referenciado**
FK a `payment_type` (valores: "credit card", "PayPal"), campo `provider` (Mastercard, Visa), `account_number`, `expiry_date` y `is_default`. Cumple regulaciones PCI-DSS si almacenas datos de tarjeta; alternativa: usar procesador externo. (Fuente: eCommerce Database Design — https://youtu.be/1HamqOuv2Cw)

**Text message abandoned cart además de email**
Los SMS tienen 80-90% de apertura en los primeros 15 minutos (vs 20-30% de email en toda su vida). Usa app Tobu Abandoned Cart. (Fuente: Top 10 Ecommerce Marketing Tips — https://youtu.be/3DdJ1JynG8M)

**Micro-interactions en cada punto de conversión del checkout**
Efectos hover con glow en botones, animaciones sutiles al hacer login, transiciones de página, expansión de componentes toast. Cada interacción pequeña suma a la sensación de calidad. Mantenlas sutiles (<300ms). (Fuente: Popular Web Design Trends — https://youtu.be/8ahnUt_A5eA)

## SOCIAL PROOF / TESTIMONIOS

**Reviews vinculadas a `order_line` para autenticidad**
Tabla `user_review` con FK a `order_line` (no a `product` directamente), asegurando que solo usuarios que compraron puedan reseñar. Incluye `rating` (1-5) y `comment`. (Fuente: eCommerce Database Design — https://youtu.be/1HamqOuv2Cw)

**Customer reviews con fotos (UGC) usando app Loox**
Las reseñas con foto son más poderosas que solo texto porque muestran el producto en uso. Activa secuencia post-purchase para pedir review (tiempo de espera = ciclo de uso del producto + 1 semana). (Fuente: Top 10 Ecommerce Marketing Tips — https://youtu.be/3DdJ1JynG8M)

**Social proof en la primera pantalla visible, no al final**
Ejemplos: reseñas (447 calificaciones, promedio 4.5 estrellas), cobertura en medios, premios, testimonios con fotos reales. Los visitantes sospechan inmediatamente; necesitas generar confianza antes de que se vayan. (Fuente: [2026 Edition] How to Create a Selling Landing Page — https://youtu.be/7eX5LG6q8Ys)

**Ejemplo concreto de farmacia:**
Headline: "Tenemos una clienta que dio a luz a los 48 años". Social proof en primera pantalla: reporte de embarazo, cobertura en medios, premios. La foto del bebé fue tomada por el dueño, no es stock. (Fuente: [2026 Edition] How to Create a Selling Landing Page — https://youtu.be/7eX5LG6q8Ys)

**Ejemplo de clínica quiropráctica:**
Key visual muestra al doctor con un cliente (evidencia de que tiene clientes reales). Incluye "acompañó a los Hanshin Tigers en su campo de entrenamiento" como social proof. (Fuente: [2026 Edition] How to Create a Selling Landing Page — https://youtu.be/7eX5LG6q8Ys)
