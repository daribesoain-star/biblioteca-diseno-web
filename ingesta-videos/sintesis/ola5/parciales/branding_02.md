- **Técnica de wordmark con espacio negativo**: Trabajar con lápiz y papel primero. Para incorporar un icono "smash" entre la M y la E, colocar la E debajo de la M y usar Shape Builder para integrar el icono. (Fuente: ola5-branding — https://youtu.be/y_2P5o_msuo)
- **Corrección de grosor en wordmark**: Seleccionar el wordmark, ir a Appearance Panel → FX button → Add New Effect → Path → Offset Path. Presionar flecha abajo hasta 0, luego mantener Command y presionar flecha abajo hasta ~0.3 para reducir los contornos y que todas las variantes se vean igual. (Fuente: ola5-branding — https://youtu.be/y_2P5o_msuo)
- **Práctica para diseño tipo estudio**: Elegir un estudio de diseño favorito (ej. Cotto Studios). Recrear un asset digital copiando la fuente, midiendo proporciones: calcular line height dividiendo el tamaño de la fuente (ej. 53/47 ≈ 1.1 line height). Multiplicar el font size por 1.1 para obtener line height suficiente. (Fuente: ola5-branding — https://youtu.be/y_2P5o_msuo)
