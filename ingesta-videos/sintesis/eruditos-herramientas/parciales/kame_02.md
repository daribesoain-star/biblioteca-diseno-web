- **Crear cuenta corriente:** Ruta `Maestros --> Cuentas Corrientes`, botón verde, campos obligatorios: Banco, Cuenta Contable, Token Bancario (dejar en blanco si no hay), Tipo Numero Mov. Predeterminado ("por periodo" o "un número en específico"), Cuenta corriente (número). (Fuente: bancos_conciliacion — https://help.kameone.cl/es/articles/9448765-manual-de-conciliacion-bancaria)
- **Crear cartola bancaria:** Módulo de conciliación → botón verde. Seleccionar cuenta corriente. Campos: Número de cartola, Periodo, Saldo inicial, Saldo final. Métodos de ingreso: Manual ("agregar" → Fecha, Comentario, Tipo de movimiento, Número de movimiento, Monto de movimiento → "Agregar línea"), Por Importador ("importar" → descargar formato → cargar archivo), Sincronizar con token bancario ("Sincronizar con Banco"). Finalizar con "Registrar cartola". (Fuente: bancos_conciliacion — https://help.kameone.cl/es/articles/9448765-manual-de-conciliacion-bancaria)
- **Conciliar cartola:** Desde menú de opciones de la cartola → "Conciliar". Dos listas: movimientos de cartola no conciliados y movimientos contables no conciliados. Conciliar Manual: seleccionar un movimiento de cada lista → botón azul "Conciliar Manual". Conciliar Automático: botón verde "Conciliar Automáticamente". Condición: la información entre cartola y contabilidad debe coincidir exactamente. (Fuente: bancos_conciliacion — https://help.kameone.cl/es/articles/9448765-manual-de-conciliacion-bancaria)
- **Crear formato de cobranza:** Ruta `Maestros -> Formatos de Cobranza`, botón verde, asignar nombre, usar editor tipo Word para redactar contenido. (Fuente: cobranza_manual — https://help.kameone.cl/es/articles/9345665-manual-de-cobranza)
- **Registrar cobranza masiva:** Módulo `Cobranza Masiva` → "Crear nueva cobranza". Campos: Fecha, Comentario, Cobrador, Cliente, Fecha de vencimiento. Hacer clic en "Buscar" para ver documentos pendientes, seleccionar y registrar. (Fuente: cobranza_manual — https://help.kameone.cl/es/articles/9345665-manual-de-cobranza)
- **Enviar cobranza:** Desde panel principal, "Enviar Cobranza", seleccionar formato previamente creado. El cliente recibe correo con cobranza detallada. (Fuente: cobranza_manual — https://help.kameone.cl/es/articles/9345665-manual-de-cobranza)
- **Activar cobranza automática:** Usuario administrador → `Perfil -> Automatizaciones` → sección `Automatización Integración de Documentos de Cobranza` → activar. Configurar: Formato de cobranza, Frecuencia (`Semanalmente` o `Mensualmente`). Requisito: clientes deben tener "envío de cobranza" activado en su ficha. (Fuente: cobranza_manual — https://help.kameone.cl/es/articles/9345665-manual-de-cobranza)
- **Cobro individual:** `Registro de Cobros → Ingreso de Cobros` → botón verde. Campos: Fecha, Comentario, Cobrador, Ficha del cliente, Documentos a cobrar, Forma de pago. Presionar "Registrar Cobro". (Fuente: cobros_manual — https://help.kameone.cl/es/articles/10504288-manual-de-cobros)
- **Cobro masivo:** `Registro de Cobros → Documentos por Cobrar` → filtrar (ej. por cliente) → "Buscar" → marcar facturas → `Acciones Masivas → Generar Cobro Masivo` → escoger forma de pago para cada documento → hacer clic en cuadrito gris de la derecha para ingresar análisis de cuenta de banco (tipo de movimiento + número) → "Registrar Cobro". (Fuente: cobros_manual — https://help.kameone.cl/es/articles/10504288-manual-de-cobros)
- **Configurar artículo para Factura de Compra Electrónica:** `Maestros → Artículos` o `Maestros → Servicios` → seleccionar ítem → habilitar "inventariable" → indicar "código de producto cambio de sujeto" → guardar. (Fuente: compras_factura_electronica — https://help.kameone.cl/es/articles/10922563-manual-ingreso-de-facturas-de-compra-electronica-propia)
- **Emitir Factura de Compra Electrónica:** Módulo de compras → nuevo documento → seleccionar: Ficha del proveedor, Tipo de compra, Comprador, Lista de precio, Glosa → habilitar "inventariable" en el documento → seleccionar artículo/servicio configurado → ingresar Cantidades, Unidad de negocio, Precio unitario → visualizar desglose de IVA → emitir. (Fuente: compras_factura_electronica — https://help.kameone.cl/es/articles/10922563-manual-ingreso-de-facturas-de-compra-electronica-propia)
- **Registrar Orden de Compra:** Módulo "Registro de OC" → completar: proveedor, fecha, glosa, tipo de compra, lista de precios, marcar "inventariable" si afecta stock, seleccionar bodega, ingresar tipo de cambio manualmente si moneda distinta a local, seleccionar artículos con cantidades y precios, forma de pago → "Registrar documento". (Fuente: compras_oc_manual — https://help.kameone.cl/es/articles/9352667-manual-de-ordenes-de-compra)
- **Aprobar y editar OC:** Desde menú de opciones → "Aprobar OC" (con observación opcional). Para editar, primero desaprobar la OC. (Fuente: compras_oc_manual — https://help.kameone.cl/es/articles/9352667-manual-de-ordenes-de-compra)
- **Generar recepción desde OC:** Desde OC aprobada → "Generar recepción" → movimiento de entrada de inventario → confirmar desde botón verde. OC queda con estado "recepción completa". (Fuente: compras_oc_manual — https://help.kameone.cl/es/articles/9352667-manual-de-ordenes-de-compra)
- **Facturar OC (total o parcial):** Desde menú de OC → "Ingresar factura" → sistema genera borrador con proveedor, productos y valores → ingresar folio de la factura → registrar. Para facturación parcial, modificar cantidades antes de registrar. OC queda con estado "facturación completa" o "parcial". Si factura ya existe, sistema pregunta si sobrescribir. (Fuente: compras_oc_manual — https://help.kameone.cl/es/articles/9352667-manual-de-ordenes-de-compra)
- **Importar OC masivamente:** Usar opción "Importar documentos" → descargar archivo de ejemplo en formato CSV → completar respetando estructura → cargar. (Fuente: compras_oc_manual — https://help.kameone.cl/es/articles/9352667-manual-de-ordenes-de-compra)
- **Registrar factura de compra manual:** Módulo "Registro de Compras" → botón verde → seleccionar tipo (ej. "Factura Electrónica") → completar: proveedor, fecha, folio, periodo libro, glosa, tipo de compra, comprador, lista de precios → elegir "inventariable" (seleccionar bodega) o "no inventariable" (seleccionar concepto de compras) → indicar cantidades, unidad de negocio, precio unitario, descuento → seleccionar impuestos adicionales, marcar "IVA no recuperable" si aplica → seleccionar condición de venta → registrar. (Fuente: compras_registro — https://help.kameone.cl/es/articles/9324521-manual-de-registro-de-compras)
- **Sincronizar documentos de compra:** Desde "Otras acciones" → "Sincronización de documentos". Opciones: "Desde el registro compra venta del SII", "Desde el SII (Portal Mi Pyme)" (recomendado si empresa emite desde SII), "Desde los documentos electrónicos recibidos" (recomendado si empresa emite con Kame). (Fuente: compras_registro — https://help.kameone.cl/es/articles/9324521-manual-de-registro-de-compras)
- **Crear cuenta contable:** `Maestros → Plan de Cuentas` → localizar "cuenta madre" (terminación en **00**) → hacer clic en icono verde a la derecha → asignar nombre → registrar. (Fuente: contabilidad_maestro — https://help.kameone.cl/es/articles/9761104-manual-maestro-de-contabilidad)
- **Configurar Impuesto Cuenta:** `Maestros → Impuesto Cuenta` → botón verde → seleccionar impuesto, cuenta contable para Compras (solo activo/pasivo), cuenta contable para Ventas (solo activo/pasivo) → registrar. (Fuente: contabilidad_maestro — https://help.kameone.cl/es/articles/9761104-manual-maestro-de-contabilidad)
- **Crear comprobante contable manual:** Módulo Contabilidad → "Nuevo Comprobante" → completar: Fecha, Tipo (Ingreso/Egreso/Traspaso), Folio, Comentario, Detalle (cuentas contables con saldo Debe o Haber) → usar botón azul para agregar líneas → "Registrar comprobante". (Fuente: contabilidad_manual — https://help.kameone.cl/es/articles/9297989-manual-de-contabilidad)
- **Cierre contable:** Hacer clic en "Cierres" → seleccionar meses individuales o tildar todos los meses de un año → confirmar. Solo usuario administrador puede ejecutar. Mes cerrado bloquea creación de nuevos movimientos. (Fuente: contabilidad_manual — https://help.kameone.cl/es/articles/9297989-manual-de-contabilidad)
- **Crear cotización:** Módulo Cotizaciones → botón verde "Nuevo" → completar: ficha del cliente, fecha de vigencia, glosa, vendedor, lista de precio, observaciones técnicas → marcar "inventariable" o no (si inventariable, seleccionar bodega) → agregar artículos, cantidades, Unidad de Negocio, Precio Unitario, Descuentos → seleccionar condición de venta ("Contado" o "Crédito") → presionar botón verde para guardar. (Fuente: cotizaciones_manual — https://help.kameone.cl/es/articles/10517925-manual-de-cotizaciones)
- **Acciones tras aprobar cotización:** "Generar Documento de Pedido" (crea pedido), "Generar Documento de Venta" (emite Factura/Boleta al SII), "Generar Guía de Despacho" (mueve inventario y genera salida de productos). Para corregir datos, usar "Desaprobar Cotización" primero. (Fuente: cotizaciones_manual — https://help.kameone.cl/es/articles/10517925-manual-de-cotizaciones)
- **Rechazar cotización:** Seleccionar tipo de rechazo (creable desde `Maestros > Tipo de rechazo`) y añadir observación. Rechazo se puede reversar desde menú de opciones. (Fuente: cotizaciones_manual — https://help.kameone.cl/es/articles/10517925-manual-de-cotizaciones)
