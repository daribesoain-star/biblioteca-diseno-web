Por qué esta ola

De "tener la herramienta" a "saber operarla como los mejores"

Teníamos 10 MCP de diseño conectados y ~17 apps con saldo, pero el conocimiento de CÓMO usar cada una vivía disperso. Esta ola busca los mejores tutoriales por herramienta (192 candidatos → triage → 55 elegidos), los transcribe y los destila en manuales operativos por herramienta que cualquier agente consulta antes de gastar un crédito.

Regla nueva del arsenal: ANTES de operar una herramienta conectada, el agente lee su entrada herramienta-* en la KB (MCP kb-diseno, base videos). Los créditos se gastan con el manual leído, no probando a ciegas.

Las 8 fichas

Qué aprendimos de cada herramienta (muestra)

herramienta-higgsfieldHiggsfield — Soul, presets y créditos

Soul SOLO para personajes personalizados; consistencia = 20-30 fotos de referencia multi-ángulo con fondo sólido; Nano Banana Pro para fotorrealismo vs Sea Dream para estilizado. 28 reglas + 22 errores comunes.

herramienta-fal-flux-apifal.ai + FLUX — la pasarela API

Video con VO3 Fast: la duración se ajusta al diálogo o rellena con "gibberish"; 3D: primero imagen de referencia con "side profile view"; Video Understanding con pedidos específicos (timestamps).

herramienta-elevenlabs-vozElevenLabs — TTS v3 + Eleven Music

Con v3 usar SOLO voces "best voices for V3"; prompts ≥250 caracteres o sale inconsistente; estabilidad en Creative/Natural para tags emocionales (nunca Robust).

herramienta-suno-musicaSuno — música completa

Meta tags entre corchetes dentro de la letra: [chorus], [soft synth pads], [explosive chorus, soaring vocals]; style description con género+instrumentos+ambiente exactos.

herramienta-recraftRecraft — diseño con marca

Nombre de marca ENTRE COMILLAS para que aparezca bien escrito; style diversity solo para explorar (apagarlo en series); 3:2 para logos.

herramienta-ideogram-textoIdeogram — texto legible

Efecto "Typography" para texto en imagen; pósters en 9:16 con Magic Prompt en Auto; fórmula [nombre] movie poster [título].

herramienta-gpt-image-openaigpt-image (OpenAI)

Fórmula de 6 componentes: Sujeto+Acción+Entorno+Estilo+Iluminación+Detalles; aspect ratio en la primera línea; inpainting con máscara PNG de canal alfa.

herramienta-replicate-apiReplicate — catálogo por API

Publicar modelos propios con Cog: setup() y predict() obligatorios en predict.py; cog login antes de push.
