# ✅ ¿CÓMO VENDER por TikTok y Cómo usar Tik Tok para VENTAS? (Mis 2 MÉTODOS 2023)
**Fuente:** retail_social | https://youtu.be/orksvd-nlS0

## Qué enseña
Explica los dos métodos principales para vender en TikTok: generar conversaciones directas con clientes potenciales (DM, WhatsApp, Telegram) o usar páginas de venta (landing pages). Introduce el concepto del "Marketing Tetrahedron" como marco para entender los cuatro fundamentos necesarios para cerrar una venta: oferta/producto, visibilidad, autoridad y conversión.

## Lecciones accionables
- Aplica el **Marketing Tetrahedron** con estos 4 fundamentos: (1) oferta y producto consistentes con el cliente objetivo, (2) visibilidad masiva (solo un pequeño porcentaje comprará, así que necesitas que muchos te vean), (3) autoridad (trabájala en tu estrategia de contenido para generar confianza), (4) conversión (cierra la venta inmediatamente, no dejes que el cliente lo posponga).
- **Método 1 – Conversación:** Lleva al cliente al chat (DM de TikTok, WhatsApp o Telegram) para iniciar el proceso de venta. Si vendes un producto simple (ej. taza de 10€ donde el cliente ya sabe lo que quiere), solo pide datos de envío y comparte tu enlace de pago (PayPal, Visum, etc.). Si vendes un servicio de 500-1000€, haz un proceso consultivo: haz preguntas para entender necesidades, verifica si tu servicio ayuda, y explica cómo lo harás; puede requerir varios días y una llamada o videollamada.
- **Método 2 – Página de venta (landing page):** Crea una página web diseñada exclusivamente para convencer al cliente de comprar, estructurada para reemplazar toda la conversación del método 1. Publica el enlace en tu biografía de TikTok y en algunos videos di "ve al link para saber más". Es automática (cierra ventas mientras duermes), pero las conversaciones cierran un porcentaje más alto de ventas.
- Para maximizar ventas, combina ambos métodos según el precio y complejidad del producto: productos baratos y simples → landing page; productos caros y complejos → conversación consultiva.

## Reglas para agentes
- Usa el Marketing Tetrahedron como checklist antes de cualquier estrategia de venta en TikTok: verifica que tienes un buen producto, visibilidad suficiente, autoridad construida y un mecanismo de cierre inmediato.
- Cuando vendas productos de bajo precio (<50€) y decisión simple, usa siempre landing page en bio para automatizar; cuando vendas servicios o productos >500€, prioriza llevar al cliente a chat para conversación consultiva.
- Nunca dejes que el cliente posponga la compra: si no cierras la venta en el momento, "mañana nunca llega".
- Siempre mide la madurez del cliente potencial antes de elegir método: cliente indeciso o producto complejo = conversación; cliente decidido o producto simple = landing page.

## Errores comunes que evita o menciona
- Tener una comunidad grande pero no cerrar ventas porque falta autoridad (la gente no confía en ti aunque te vean millones).
- Usar solo visibilidad sin trabajar la confianza: puedes tener millones de vistas pero si no eres visto como experto, casi nadie comprará.
- No tener un buen producto empaquetado en una oferta consistente: sin esto, tendrás muchas devoluciones o simplemente no habrá venta.
- Dejar la venta para después: si no conviertes en el momento, el cliente lo pospone indefinidamente.
