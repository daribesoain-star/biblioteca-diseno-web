Manual de Órdenes de Compra

El módulo de Órdenes de Compra permite gestionar de forma ordenada todo el proceso de compras a proveedores, desde la creación de la orden hasta su recepción y facturación.

A través de este módulo, podrás mantener un control claro sobre los productos adquiridos, sus cantidades, valores y estados, facilitando así la trazabilidad y la toma de decisiones dentro de la empresa.

Entre sus principales funcionalidades se encuentran:

  • Registro de órdenes de compra

  • Seguimiento del estado de cada documento

  • Aprobación de órdenes

  • Generación de recepciones

  • Facturación total o parcial

  • Importación y exportación de información

Registro de Órdenes de Compra

Para comenzar, debes ingresar al módulo “Registro de OC” desde el menú principal del sistema.

Una vez dentro, podrás crear una nueva orden de compra seleccionando la opción correspondiente.

Al hacerlo, el sistema mostrará una pantalla donde deberás completar la información del documento.

Primero, deberás ingresar los datos generales, como el proveedor, la fecha y la glosa. El folio del documento se asigna automáticamente, y los datos de dirección se completan en base al proveedor seleccionado.

Luego, podrás definir configuraciones adicionales como el tipo de compra, la lista de precios, y si la orden será inventariable, lo que permitirá que afecte directamente el stock. En este punto también deberás seleccionar la bodega correspondiente.

Importante: El sistema permite trabajar con distintas monedas, pero el tipo de cambio debe ingresarse manualmente.

A continuación, deberás ingresar el detalle de los productos, seleccionando los artículos, indicando sus cantidades y precios. Si necesitas agregar más productos, puedes hacerlo fácilmente utilizando el botón “+”.

Finalmente, completa la forma de pago y haz clic en “Registrar documento” para guardar la orden de compra.

Gestión y acciones disponibles

Una vez registrada la orden de compra, el sistema permite realizar diversas acciones desde el menú de opciones:

  • Consultar el seguimiento del documento

  • Agregar comentarios o estados

  • Aprobar la orden de compra

  • Visualizar el documento en PDF

  • Adjuntar archivos

  • Copiar o eliminar el documento

Estas opciones facilitan la gestión y el control de cada orden dentro del proceso de compra.

Aprobación de la Orden de Compra

Luego de registrar la orden, el siguiente paso es su aprobación. Para ello, debes seleccionar la opción “Aprobar OC” desde el menú del documento, ingresar una observación si es necesario y confirmar la acción.

Importante: Si necesitas editar una orden de compra ya registrada, primero deberás desaprobarla.

Generación de Recepción

Una vez aprobada la orden, se debe registrar la recepción de los productos. Para esto, selecciona la opción “Generar recepción”.

El sistema solicitará confirmación y luego te llevará al movimiento de entrada de inventario, donde podrás visualizar los productos asociados a la orden. Finalmente se registra desde el boton verde.

Después de confirmar la recepción, la orden de compra quedará con estado de recepción completa.

Facturación de la Orden de Compra

El siguiente paso en el flujo es la facturación. Desde el menú de la orden de compra, selecciona “Ingresar factura”.

El sistema generará automáticamente un borrador con la información de la orden, incluyendo proveedor, productos y valores. Solo deberás ingresar el folio de la factura y registrar el documento para completar el proceso.

Una vez realizado, la orden de compra quedará con estado de facturación completa.

Nota: Si la factura ya esta integrada en el sistema antes de ingresarla desde la OC, al momento de ingresar el folio y registrarla el sistema preguntará si desean sobre-escribir la informacion (osea que se unificaram ambos registros para dejar solo uno).

Facturación Parcial

El sistema también permite realizar facturación parcial de una orden de compra. Esto significa que puedes facturar solo una parte de los productos y dejar pendiente el resto.

Por ejemplo, si la orden es por 10 unidades, puedes facturar 5 y posteriormente completar la facturación restante.

Para hacerlo, al momento de ingresar la factura, solo debes modificar las cantidades de los productos antes de registrar el documento.

El sistema dejará la orden con estado parcial, permitiendo continuar el proceso más adelante.

Exportar Órdenes de Compra

El módulo permite exportar la información de órdenes de compra a un archivo Excel.

Solo debes seleccionar el periodo que deseas consultar y utilizar la opción de exportación.

El sistema descargará automáticamente el archivo con el detalle correspondiente.

Importación Masiva de Documentos

Si necesitas ingresar múltiples órdenes de compra, puedes utilizar la opción “Importar documentos”.

Al acceder, el sistema te mostrará las instrucciones necesarias y te permitirá descargar un archivo de ejemplo en formato CSV.

Deberás completar este archivo respetando la estructura indicada y luego cargarlo en el sistema para realizar la importación masiva.

El módulo de Órdenes de Compra es una herramienta clave para mantener un control eficiente y ordenado del proceso de adquisiciones dentro de la empresa.

A través de sus distintas funcionalidades, permite gestionar cada etapa, desde la creación de la orden hasta su recepción y facturación, asegurando una correcta trazabilidad de la información.

Además, opciones como la facturación parcial, la importación masiva y la exportación de datos facilitan la gestión operativa, adaptándose a diferentes necesidades y volúmenes de trabajo.

Se recomienda utilizar este módulo de forma ordenada, respetando el flujo del proceso (registro, aprobación, recepción y facturación), para asegurar la consistencia de la información y un mejor control de las compras.
