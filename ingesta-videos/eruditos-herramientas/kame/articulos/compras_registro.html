Manual de Registro de Compras

El módulo de Registro de Compras permite gestionar de forma eficiente el ingreso, control y seguimiento de los documentos asociados a las adquisiciones de la empresa.

A través de este manual, conocerás cómo registrar facturas de compra, identificar si estas afectan o no el inventario, y utilizar las distintas funcionalidades disponibles para optimizar la administración de los documentos.

Además, se explicarán las acciones masivas y otras herramientas complementarias, como la importación, exportación y sincronización de documentos con plataformas externas.

El correcto uso de estas opciones facilitará la organización de la información, el control contable y la gestión oportuna de pagos a proveedores.

Como ingresar una factura de Compra

Para registrar el documento de una compra debemos hacer ingresar en el módulo de “Registro de Compras” ubicado del lado izquierdo de nuestra pantalla.

Una vez ingresado en el módulo podremos visualizar varios botones para realizar diferentes funciones.

Desde el boton verde, podemos registrar un nuevo documento manualmente en el sistema

Desde aqui, puedes seleccionar el documento a registrar, por ejemplo 'Factura Electrónica', con lo cual el sistema desplegara una  nueva ventana en donde se deben completar los datos de la factura recibida con la información del proveedor, fecha, folio, periodo libro, glosa, tipo de compra, comprador, lista de precios.

Continuando con el registro, podemos seleccionar si una compra será inventariable o no, recordando que las compras inventariables mueven el inventario, y posterior a ello debemos seleccionar la bodega correspondiente.

En el caso de que la compra no sea inventariable, en el detalle del documento debemos seleccionar un concepto de compras segun lo que se haya comprado.

Luego, indicamos las cantidades, unidad de negocio, precio unitario y algun descuento si corresponde.

Luego podemos seleccionar algun impuesto adicional si corresponde, al igual que seleccionar si es IVA no recuperable.

Finalmente, seleccionamos la condicion de venta, y registramos el documento.

Una vez ingresada nuestra compra figurará dentro del modulo y podremos realizar las siguientes funciones desde el menu de opciones:

  • Adjuntar Archivo: Para adjuntar aldun documento PDF que este relacionado con la compra.

  • Nota de Crédito: Para genera una nota de crédito al documento

  • Copiar documento: Para copiar el documento en un nuevo registro

  • Nota de Débito: Para genera una nota de débito al documento

  • Ver comprobante contable: Esta opción es para ver el registro contable directamente desde contabilidad

  • Generar pago: Esta opción es para pagar la factura dese el registro de pagos

  • Eliminar: Para eliminar la factura.

Acciones Masivas

Las acciones masivas nos permiten realizar ciertas acciones masivamente para más de un documento.

En compras, contamos con dos acciones masivas, una para registrar pagos, y otra para eliminar los documentos.

Para utilizar cualqueira de estas, primero se debenseleccionar los documentos dentro del modulo

Y luego seleccionamos la accion masiva que deseemosutilizar para los documentos.

Otras Acciones

Desde el boton de otras acciones, contamos con varias funcionalidades que complementan el uso del modulo.

Dentro de estas acciones tenemos:

  • Importacion de documentos:

    • Desde un archivo CSV

    • Desde un archivo XML

  • Exportar documentos:

    • Exportar a excel

    • Exportar facturas hacia el portal del SII

  • Sincronizacion de documentos:

    • Desde el registro compra venta del SII

    • Desde el SII (Portal Mi Pyme)

    • Desde los documentos electronicos recibidos

1) Importacion de documentos

1.1) Desde un archivo CSV: Esta opcion nos permite cargar los documentos de compra masivamente desde un archivo excel conformato CSV.

Al hacer clic en esta opcion el sistema nos redirigira a una ventana en donde podremos descargar un archivo de ejemplo y visualizar las instrucciones de llenado del archivo.

1.2) Desde un archivo XML: Con esta opcion, podremos cargar el XML de cada documento en el sistema

Al realizar la carga, el documento quedará registrado en el modulo de compras.

2) Exportar documentos

2.1) Exportar a excel: Con esta opcion podremos exportar a excel los documentos del periodo que estemos filtrando en el modulo.

2.2) Exportar facturas hacia el portal del SII: Desde aqui, podremos exportar los doumentos de compra a excel, pero por cada periodo.

3) Sincronizacion de documentos

3.1) Desde el registro compra venta del SII: Esta opcion nos permite integrar los documentos de compra que figuren en el Registro de Compra y Venta del SII.

3.2) Desde el SII (Portal Mi Pyme): Nos permite integrar los documentos de compra que figuren en el Portal Mi Pyme del SII, por lo que si la empresa emite documentos de venta desde el SII, se recomienda utilizar esta integracion en compras.

3.3) Desde los documentos electronicos recibidos: Nos permite integrar los documentos de compra que nuestros proveedores nos envien a la casilla de intercambio. Por lo que si la empresa emite documentos con Kame, se recomienda utilizar esta integracion de compras.

Una adecuada gestión del registro de compras es fundamental para mantener un control ordenado de los documentos, asegurar la correcta imputación contable y facilitar los procesos de pago y análisis financiero.

Gracias a las funcionalidades descritas en este manual, es posible registrar documentos de manera manual o masiva, integrarlos desde distintas fuentes y realizar acciones que optimizan el trabajo operativo.

Aplicar estos procedimientos permitirá mejorar la eficiencia administrativa, fortalecer el control del inventario y mantener la información actualizada para la toma de decisiones.
