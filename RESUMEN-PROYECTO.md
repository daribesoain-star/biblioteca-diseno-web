# Resumen del proyecto · Biblioteca de Diseño Web
Fecha: julio 2026

## Qué construimos en esta conversación

Partiste queriendo aprender diseño web desde cero y terminamos construyendo una
biblioteca interactiva completa para humanos y agentes de IA.

### 1. Lo que aprendiste (ruta recorrida)
- Los 3 lenguajes base: HTML (estructura), CSS (estilo), JavaScript (interacción)
- Principios de diseño: jerarquía visual (5 palancas: tamaño, peso/grises,
  acento, espacio, posición), squint test, espacio en blanco, tipografía, color
- Diseño = comunicación: patrones de lectura F y Z, pirámide invertida,
  estructura de landing (gancho → beneficio → prueba → acción), microcopy
- Producción: 3D pre-renderizado (Blender: HDRI, PBR, oclusión) vs tiempo real
  (Spline, Three.js); optimización WebP/AVIF <300KB
- Carreras: Diseño Gráfico, UX/UI, Desarrollo Web, Publicidad/Marketing,
  ruta autodidacta, y las mejores escuelas del mundo (RCA, UAL, Parsons, RISD)
- Stack de herramientas con APIs y automatización vía agentes (MCP, orquestadores)

2. Arrastrar esta carpeta completa
