# Practical Typography — Summary of Key Rules
**Materia:** tipografia-jerarquia-visual | **Fuente:** https://practicaltypography.com/summary-of-key-rules.html (consultada 2026-07-07)

## Qué aporta (2-3 líneas)
Proporciona las reglas tipográficas fundamentales para texto de cuerpo, con valores numéricos precisos para tamaño, interlineado y longitud de línea. Establece criterios verificables para mejorar la legibilidad y profesionalismo en documentos impresos y web.

## Reglas candidatas para agentes
- Usar tamaño de punto de 10–12 pt en documentos impresos.
- Usar tamaño de 15–25 px en web.
- Ajustar el interlineado al 120–145% del tamaño de punto.
- Mantener longitud de línea promedio de 45–90 caracteres (incluyendo espacios).
- Usar sangría de primera línea de 1 a 4 veces el tamaño de punto, O espacio entre párrafos de 4–10 pt (nunca ambos).
- Aplicar 5–12% de espaciado extra entre letras en mayúsculas sostenidas y versalitas.
- Usar solo un espacio entre oraciones.
- No usar más de un espacio entre palabras ni otros caracteres de espacio en blanco consecutivos.
- No usar subrayado excepto para enlaces web.
- Limitar el uso de negrita o cursiva al mínimo, y nunca combinarlas.
- Limitar el texto centrado a usos esporádicos.
- Limitar las mayúsculas sostenidas a menos de una línea de texto.
- Usar guiones con texto justificado siempre.
- Usar comillas curvas, no rectas.
- Usar apóstrofos que apunten hacia abajo.
- Usar comillas de pie y pulgada rectas, no curvas.
- Usar el carácter de elipsis correcto, no puntos y espacios.
- Poner un espacio de no separación después de marcas de párrafo y sección.
- En documentos de más de tres páginas, usar solo un signo de exclamación.
- Usar ampersands solo esporádicamente, salvo en nombres propios.
- Usar símbolos de marca registrada y copyright correctos, no aproximaciones alfabéticas.
- No confundir guiones con rayas, ni usar múltiples guiones como raya.
- No usar versalitas falsas; si no se tienen reales, no usarlas.
- Activar el kerning siempre.
- Evitar fuentes monospaciadas, gratuitas, del sistema, Times New Roman y Arial.

## Errores comunes que documenta
- Usar comillas rectas en lugar de curvas.
- Poner dos espacios entre oraciones.
- Usar múltiples espacios en blanco consecutivos.
- Subrayar texto que no sea un enlace web.
- Usar mayúsculas sostenidas en bloques largos de texto.
- Centrar texto excesivamente.
- Usar sangría y espacio entre párrafos simultáneamente.
- No usar guiones con texto justificado.
- Confundir guiones con rayas.
- Usar puntos y espacios para simular una elipsis.
- Usar fuentes Times New Roman y Arial.

## Datos/citas clave textuales
- "Point size should be 10–12 points in printed documents, 15-25 pixels on the web."
- "Line spacing should be 120–145% of the point size."
- "The average line length should be 45–90 characters (including spaces)."
- "Use 5–12% extra letterspacing with all caps and small caps."
- "Use first-line indents that are one to four times the point size of the text, or use 4–10 points of space between paragraphs. Don’t use both."
- "In a document longer than three pages, one exclamation point is plenty."
- "Put only one space between sentences."
- "Don’t use multiple word spaces or other white-space characters in a row."
